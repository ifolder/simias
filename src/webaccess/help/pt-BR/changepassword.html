Mudando senhas

Para mudar a senha da sua conta do iFolder, siga as etapas descritas abaixo:

  1. Verifique se a caixa de seleção Mudar Senha está marcada. Essa caixa de seleção é marcada por padrão e os campos de mudança de senha só podem ser editados se essa caixa estiver marcada.
  2. Especifique sua senha atual no campo Senha Atual.
  3. Especifique a senha que deseja configurar para a conta do iFolder atual no campo Nova Senha.
  4. Especifique a nova senha mais uma vez para confirmação no campo Confirmar Nova Senha.
  5. Clique em Gravar para gravar as suas mudanças.

Nota: O iFolder muda a senha diretamente no servidor LDAP. Se o servidor LDAP for a origem de gerenciamento de usuários de outros aplicativos, a nova senha
também será usada nesses aplicativos. Depois que a senha for mudada, a sessão do iFolder expirará e você será desconectado. Efetue login novamente para acessar os serviços do iFolder.
