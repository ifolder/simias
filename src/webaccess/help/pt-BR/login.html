Efetuando login no Web Access

Efetue login no Novell iFolder® 3.x Web Access para obter acesso a pastas e arquivos de qualquer pasta do iFolder disponível em sua conta do iFolder 3.x. Isso inclui pastas do iFolder das quais você participa, mas que não estão configuradas em um computador local. (Não é possível acessar arquivos de sua conta do Novell iFolder 2.1x com o Web Access.) Seu nome de usuário e sua senha serão autenticados nos serviços de diretório LDAP.

Você também pode especificar qual dos idiomas suportados usará. Normalmente é o mesmo que o código de idioma do servidor empresarial iFolder e do computador local. Verifique se definiu as configurações de idioma do browser para aceitar o idioma desejado.

  1. Especifique o nome de usuário e a senha da conta do iFolder que deseja acessar.

    Nomes de usuários e senhas diferenciam maiúsculas de minúsculas.

  2. Use a lista suspensa Idiomas para especificar qual idioma deseja usar para acessar os arquivos.

  3. Clique em Efetuar Login.

 

Um símbolo de marca registrada (®, TM etc.) indica uma marca registrada da Novell. Um asterisco (*) indica uma marca registrada de terceiros. Para obter mais informações, consulte Informações legais.
