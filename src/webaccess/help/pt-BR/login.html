Efetuando login no Web Access

Efetue login no Web Access do Novell iFolder® 3.7 ou posterior para obter acesso a pastas e arquivos de qualquer pasta do iFolder disponível para você ou para o Grupo LDAP do qual você faz parte na conta do iFolder 3.7 ou posterior. Isso inclui pastas do iFolder das quais você participa, mas que não estão configuradas em um computador local. (Não é possível acessar arquivos em sua conta do Novell iFolder 2.x com o Web Access.) A sua senha e o seu nome de usuário serão autenticados em relação aos serviços de diretório LDAP.

Você também pode especificar quais dos idiomas suportados serão usados. Geralmente são os mesmos idiomas que o código de idioma do servidor empresarial do iFolder e do seu computador local. Defina a configuração Idiomas do browser para que o idioma desejado seja suportado.

  1. Especifique o nome de usuário e a senha da conta do iFolder que deseja acessar.

    Nomes de usuário e senhas diferenciam maiúsculas de minúsculas.

  2. Use a lista suspensa Idiomas para especificar qual idioma deseja usar para acessar os arquivos.

  3. Clique em Efetuar Login.
