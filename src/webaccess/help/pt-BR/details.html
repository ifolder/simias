Vendo propriedades do iFolder

Use esta página para ver as propriedades do iFolder selecionado. Você também pode editar a descrição que foi fornecida quando o iFolder foi criado.

É possível ver as seguintes propriedades do iFolder.

Parâmetro

Descrição

Nome

(ver apenas)

O nome designado ao iFolder.

Descrição

Uma breve descrição do iFolder.

Para editar a descrição do iFolder, clique em Editar. Na nova página aberta, digite a nova descrição no campo Descrição e clique em Gravar.

Para cancelar as mudanças feitas, clique em Cancelar.

Last modified

(ver apenas)

Exibe a hora em que o iFolder foi alterado pela última vez por um usuário.

Criado

(ver apenas)

Exibe a data em que o iFolder foi criado.

Direitos

(ver apenas)

Exibe os direitos do proprietário do iFolder.

Proprietário

(ver apenas)

Exibe o nome de usuário do proprietário do iFolder.

Esse é o nome exclusivo do proprietário do iFolder. O proprietário do iFolder pode ser um usuário individual ou um Grupo LDAP.

Se um Grupo LDAP for proprietário do iFolder selecionado, todos os membros do grupo terão os mesmos direitos nesse iFolder.

Tamanho

(ver apenas)

Exibe o tamanho do iFolder.

Membros

(ver apenas)

Relata o número total de usuários pertencentes ao iFolder. Inclui usuários individuais e Grupos LDAP.

Arquivos

(ver apenas)

Exibe o número total de arquivos neste iFolder.

Pastas

(ver apenas)

Exibe o número total de pastas neste iFolder.

Bloqueado

(ver apenas)

Mostra se a sincronização do iFolder está habilitada ou não. Se o Administrador do iFolder definir a sincronização do iFolder como habilitada, o valor para Bloqueado será Não. Se estiver definida como desabilitada, o valor para Bloqueado será Sim.
