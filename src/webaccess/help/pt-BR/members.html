Gerenciando usuários de um iFolder

Em uma pasta do iFolder, é possível executar as seguintes tarefas:

Adicionando novo usuário compartilhado

Clique em Adicionar para adicionar novos membros compartilhados à pasta do iFolder selecionada. É possível adicionar um usuário individual ou um Grupo LDAP como membro compartilhado. A nova página aberta permite a adição de usuários à lista de usuários compartilhados.

Designando direitos a um usuário compartilhado

Na página Membros, selecione os membros compartilhados e clique nos direitos que você deseja designar a eles.

Compreendendo direitos de acesso do iFolder

O cliente iFolder suporta o proprietário e três níveis de acesso para membros de um iFolder:

IMPORTANTE:Todos os membros de um Grupo LDAP herdam os direitos de acesso definidos para esse grupo.

Somente leitura
Um membro da pasta iFolder compartilhada, com direito de acesso Apenas leitura aos diretórios e arquivos do iFolder. Esse membro pode copiar um arquivo do iFolder para outro local e modificá-lo fora do iFolder.

Leitura/Gravação
Um membro da pasta do iFolder compartilhada, com direito de acesso Leitura/Gravação aos diretórios e arquivos do iFolder.

Controle Total
Um membro da pasta iFolder compartilhada, com direito de acesso Controle Total. O usuário com o direito Controle Total possui o acesso de leitura/gravação para o iFolder e gerencia os direitos de associação e acesso para todos os usuários exceto o proprietário. Porém, somente o proprietário tem o direito para apagar um iFolder.

Proprietário
Apenas um usuário pode ser o proprietário de um iFolder. Geralmente é o usuário que criou o iFolder e somente ele pode apagar esse iFolder O usuário proprietário pode usar o cliente iFolder para transferir a propriedade para outro membro do iFolder. O usuário Admin do iFolder pode transferir a propriedade para qualquer usuário no servidor iFolder.

O proprietário de um iFolder tem direito Controle Total. Esse usuário tem acesso de leitura/gravação ao iFolder, gerencia a participação e os direitos de acesso de usuários membros e pode remover o direito Controle Total de qualquer membro.

Se um usuário for apagado como um usuário do sistema do iFolder, os iFolders de propriedade do usuário ficarão órfãos. Os iFolders órfãos são designados temporariamente ao usuário Admin do iFolder como proprietário. A associação e a sincronização continua para outros usuários enquanto que o usuário Admin do iFolder determina se um iFolder órfão deve ser excluído ou designado a um novo proprietário.
