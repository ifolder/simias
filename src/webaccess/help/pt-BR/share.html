Adicionando novo usuário compartilhado

Para adicionar um novo usuário ou Grupo LDAP à lista de usuários compartilhados,

  1. Use a funcionalidade de pesquisa para localizar o usuário ou Grupo LDAP com o qual você deseja compartilhar o iFolder.

    2. Selecione um critério de filtro (Nome de Usuário, Nome, Sobrenome) e digite uma ou mais letras no campo Pesquisar.

  2. Clique no (o sinal de mais em azul) para adicionar os usuários à lista de usuários compartilhados.

    Os membros adicionados aparecem no lado direito da página em Compartilhar com. Depois que um usuário for adicionado, o sinal de mais ao lado do nome do iFolder fica preto.

    Quando você compartilha uma pasta do iFolder com um Grupo LDAP, ele é compartilhado com todos os membros do grupo.

  3. Clique em Compartilhar para confirmar o compartilhamento.

    Clique em Cancelar para cancelar todas as operações nesta página.
