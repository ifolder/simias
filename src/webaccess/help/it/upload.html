Caricamento di un file

  1. Fare clic su Sfoglia nel campo Carica file per individuare il file locale che si desidera caricare.

  2. Selezionare il file, quindi fare clic su Apri > Carica.

    3. Quando la pagina viene aggiornata, il file verrà visualizzato in ordine alfabetico nell'elenco. Per default, se si carica un file con lo stesso nome file, questo verrà sovrascritto. Se non si desidera sovrascrivere i file esistenti, deselezionare la casella di controllo Sovrascrivi il file esistente.
