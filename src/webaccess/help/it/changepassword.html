Modifica della password

Per cambiare la password del conto iFolder, seguire la procedura indicata di seguito.

  1. Verificare che la casella di controllo Cambia password sia selezionata. Per impostazione di default questa casella di controllo è selezionata ed è possibile modificare i campi relativi alla modifica della password solo se questa casella di controllo è selezionata.
  2. Specificare la password attuale nel campo Password attuale.
  3. Specificare la password che si desidera impostare per il conto iFolder attuale nel campo Nuova password.

    Consentendo l'utilizzo di caratteri speciali, l'amministratore può migliorare l'invulnerabilità delle password.

  4. Specificare di nuovo la nuova password per conferma nel campo Conferma nuova password.
  5. Fare clic su Salva per salvare le modifiche.

Nota: la password viene cambiata direttamente nel server LDAP. Se lo stesso server LDAP è un'origine di gestione utenti per altre applicazioni, la nuova password sarà applicabile anche a tali applicazioni. Una volta modificata la password, la sessione di iFolder scadrà e si verrà scollegati. È necessario eseguire di nuovo il login per connettersi ai servizi iFolder.
