Ricerca di cartelle e file

La pagina Cerca consente di individuare i file o le cartelle desiderate. A questo scopo utilizzare lo strumento di ricerca situato nella parte superiore destra della pagina. Digitare il nome del file o della cartella nel campo di immissione e premere Invio per ottenere il risultato nella pagina Cerca con il testo della ricerca evidenziato in giallo.

Utilizzare uno o più dei seguenti metodi di ricerca e premere Invio.

  • Digitare una o più lettere del nome del file o delle cartelle nel campo di immissione Cerca.

  • Digitare un asterisco (*) nel campo di ricerca di cartelle iFolder per ottenere un elenco di tutte le cartelle iFolder presenti nel sistema.

  • Lasciare vuoto il campo di ricerca di cartelle iFolder per ottenere un elenco di tutte le cartelle iFolder presenti nel sistema.

Non fare clic sulla pagina finché non si è completamente aggiornata; a questo punto sarà possibile cercare il percorso dei file o delle cartelle iFolder elencate nei risultati della ricerca oppure gestirli. Scorrere la pagina verso l'alto e verso il basso per cercare il percorso dei risultati della ricerca e individuare la cartella o il file desiderato. La combinazione di nome del file o della cartella e data di creazione consentono di identificare il file o la cartella ricercata.
