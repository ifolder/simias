Creazione di una nuova cartella iFolder

Per creare una nuova cartella iFolder, eseguire le seguenti operazioni:

  1. Fare clic sul collegamento Nuova cartella iFolder nella home page.

  2. Specificare nella nuova pagina le seguenti informazioni:

    3. Nome
    Assegnare un nome alla cartella iFolder, ad esempio iFolder.

    Nota: non utilizzare i caratteri non validi indicati di seguito nei nomi delle cartelle iFolder né nei nomi delle cartelle e dei file in esse contenuti:

    \/:*?"<>|;

    IMPORTANTE: quando si tenta di creare una cartella iFolder il cui nome supera il limite supportato dal file system del server iFolder, viene visualizzato un messaggio di errore indicante che il nome della cartella è troppo lungo.


    Descrizione
    Una breve descrizione della cartella iFolder.

    Tipo
    Selezionare il tipo della cartella iFolder che si desidera creare.
    Cifrata
    Selezionare questa opzione per cifrare la cartella iFolder e i file in essa contenuti.

    IMPORTANTE: l'opzione non è abilitata finché l'amministratore non attiverà le norme di cifratura.

    Normale
    Selezionare l'opzione Condivisibile per condividere la cartella iFolder con altri conti iFolder.

    IMPORTANTE: non è possibile cifrare le cartelle iFolder condivise, né condividere quelle cifrate.

  3. Selezionare l'agente di recupero dall'elenco a discesa.

    Nota: questa opzione non sarà disponibile se l'amministratore iFolder non avrà selezionato l'opzione di abilitazione del recupero in fase di configurazione del server iFolder.

    L'agente di recupero può essere un sistema, un'organizzazione o una persona indipendente dal sistema iFolder in cui è possibile memorizzare la chiave per recuperare la stringa di cifratura segreta.

    IMPORTANTE: a meno che non sia stato impostato l'agente di recupero, è impossibile recuperare i dati cifrati in caso di perdita della stringa di cifratura.

  4. Digitare la stringa di cifratura nel campo corrispondente.

  5. Immettere di nuovo la stringa di cifratura per conferma.

  6. Fare clic su Crea per completare la creazione della nuova cartella iFolder.

    Fare clic su Annulla per annullare l'operazione.
