Login di accesso Web

Eseguire il login ad Accesso Web di Novell iFolder® 3.7 per accedere alle cartelle e ai file delle cartelle iFolder disponibili per l'utente o per il gruppo LDAP di cui questi è membro nel conto iFolder 3.7. Sono incluse le cartelle iFolder di cui si è membri per le quali tuttavia non si è configurata una cartella corrispondente nel computer locale. Non è possibile accedere a file del conto Novell iFolder 2.x con Accesso Web. Il nome utente e la password vengono autenticati in base ai servizi di directory LDAP.

È inoltre possibile specificare le lingue supportate da utilizzare. Si tratta in genere della stessa lingua del server aziendale iFolder e del computer locale dell'utente. Verificare che l'impostazione delle lingue nel browser sia configurata per supportare la lingua desiderata.

  1. Digitare il nome utente e la password del conto iFolder con cui si desidera accedere.

    Per nomi utenti e password viene fatta distinzione tra maiuscole e minuscole.

  2. Impostare la lingua che si desidera utilizzare per l'accesso ai file nell'apposito elenco a discesa.

  3. Fare clic su Login.
