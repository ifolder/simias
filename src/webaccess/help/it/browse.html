Accesso alle cartelle e ai file nella cartella iFolder

Per passare alla pagina Sfoglia, fare clic su uno dei collegamenti iFolder elencati nella home page.

Utilizzare questa pagina per accedere alle cartelle e ai file all'interno della cartella iFolder selezionata nel conto attuale. Per ogni cartella iFolder selezionata è possibile visualizzare un elenco delle cartelle e dei file disponibili, caricare e scaricare file, creare cartelle e cancellare cartelle e file.

NOTA: non è possibile accedere a cartelle iFolder disabilitate né selezionarle.

Gestione di cartelle e file in una cartella iFolder

Nelle cartelle iFolder è possibile eseguire i task seguenti:

Visualizzare un elenco di cartelle e file
Nella pagina Sfoglia vengono elencati file e cartelle in ordine alfabetico. Scorrere l'elenco per individuare la directory o il file desiderato.

Accedere a una directory
Fare clic sul nome della cartella per aprire la cartella e visualizzare un elenco dei suoi contenuti. Per tornare a una directory di livello superiore, utilizzare la barra di navigazione in alto. Per tornare alla directory desiderata, è possibile fare clic sull'icona o sul nome.

Aprire o salvare un file
Fare clic sul nome del file che si desidera aprire o salvare.

  • Fare clic su Apri per visualizzare il contenuto del file. Per aprire un file è necessario che nel computer in uso sia installata un'applicazione compatibile. Se si intende utilizzare il file, scaricarlo, apportare le modifiche desiderate e quindi caricarlo.

  • Fare clic su Salva per salvare il file sul disco rigido locale o su un'unità di rete mappata. Selezionare l'ubicazione in cui si desidera collocare il file, quindi fare clic su Salva. Il file verrà scaricato nel computer in uso.

Creare una nuova cartella
Fare clic sul collegamento Nuova cartella per creare una nuova cartella.

Cancellare cartelle o file
Selezionare la casella di controllo vicino a una o più cartelle o file che si desidera cancellare, quindi fare clic su Cancella per confermare la cancellazione.

Caricare un file
Fare clic su Carica e, nella pagina corrispondente, fare clic su Sfoglia e individuare il file locale che si desidera caricare, quindi fare clic su Apri > Carica. Quando la pagina viene aggiornata, il file verrà visualizzato nel corretto ordine alfabetico nell'elenco.

Visualizzare la cronologia
Fare clic su per visualizzare la cronologia.

Cancellare una cartella iFolder

Per cancellare una cartella iFolder:

  1. Fare clic sul collegamento Cancella in Azioni nel lato sinistro della pagina.

    Viene visualizzata una finestra di dialogo in cui si richiede se si desidera cancellare la cartella iFolder attuale.

  2. Fare clic su OK nella finestra di dialogo per confermare la cancellazione della cartella iFolder.

  3. Fare clic su Annulla per annullare l'operazione.
