Visualizzazione delle proprietà della cartella iFolder

Utilizzare questa pagina per visualizzare le proprietà iFolder della cartella iFolder selezionata. È inoltre possibile modificare la descrizione fornita una volta creata la cartella iFolder.

È possibile visualizzare le seguenti proprietà iFolder.

Parametro

Descrizione

Nome

(sola lettura)

Il nome assegnato alla cartella iFolder.

Descrizione

Una breve descrizione della cartella iFolder.

Fare clic su Modifica per modicare la descrizione della cartella iFolder. Digitare nella nuova pagina la nuova descrizione nell'apposito campo, quindi fare clic su Salva.

Fare clic su Annulla per annullare le modifiche apportate.

Ultima modifica

(sola lettura)

Consente di visualizzare l'ora in cui è stata apportata l'ultima modifica alla cartella iFolder da parte di un utente.

Data di creazione

(sola lettura)

Consente di visualizzare la data di creazione della cartella iFolder.

Diritti

(sola lettura)

Consente di visualizzare i diritti di cui dispone il proprietario di iFolder sulla cartella iFolder.

Proprietario

(sola lettura)

Consente di visualizzare il nome utente del proprietario iFolder.

Si tratta del nome distinto completo del proprietario iFolder. Il proprietario della cartella iFolder può essere un singolo utente o un gruppo LDAP.

Se la cartella iFolder è di proprietà di un gruppo LDAP, tutti i membri appartenenti al gruppo dispongono dei medesimi diritti sulla cartella.

Dimensioni

(sola lettura)

Consente di visualizzare le dimensioni della cartella iFolder.

Membri

(sola lettura)

Indica il numero totale di utenti appartenenti alla cartella iFolder. Include singoli utenti e gruppi LDAP.

File

(sola lettura)

Consente di visualizzare il numero totale di file nella cartella iFolder.

Cartelle

(sola lettura)

Consente di visualizzare il numero totale di cartelle nella cartella iFolder.

Bloccato

(sola lettura)

Consente di visualizzare se la sincronizzazione delle cartelle iFolder è attivata. Se l'amministratore iFolder ha attivato la sincronizzazione delle cartelle iFolder, il valore Bloccato è impostato su No. Se invece la sincronizzazione è attivata, il valore è impostato su Sì.
