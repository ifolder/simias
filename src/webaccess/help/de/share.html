Hinzufügen neuer freigegebener Benutzer

So fügen Sie der Liste der freigegebenen Benutzer einen neuen Benutzer bzw. eine neue LDAP-Gruppe hinzu:

  1. Verwenden Sie das Suchwerkzeug, um den Benutzer bzw. die LDAP-Gruppe zu suchen, für den bzw. die Sie einen iFolder-Ordner freigeben möchten.

    2. Wählen Sie ein Filterkriterium (Benutzername, Vorname, Nachname) und geben Sie dann mindestens einen Buchstaben in das Suchfeld ein.

  2. Klicken Sie auf (das blaue Pluszeichen), um den Benutzer zur Liste der freigegebenen Benutzer hinzuzufügen.

    Das hinzugefügte Mitglied wird auf der rechten Seite unter "Gemeinsam verwenden mit" angezeigt. Nachdem ein Benutzer hinzugefügt wurde, wird das Pluszeichen neben dem Namen des iFolder-Ordners schwarz angezeigt.

    Wenn Sie einen Ordner mit einer LDAP-Gruppe freigeben, wird er für alle Mitglieder der LDAP-Gruppe freigegeben.

  3. Klicken Sie auf "Freigeben", um die Operation zu bestätigen.

    Klicken Sie auf "Abbrechen", um alle Operationen auf dieser Seite rückgängig zu machen.
