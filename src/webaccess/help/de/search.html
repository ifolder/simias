Suchen von Ordnern und Dateien

Die Seite "Suche" hilft Ihnen beim Suchen der gewünschten Dateien und Ordner. Verwenden Sie dazu das Suchwerkzeug oben rechts auf der Seite. Geben Sie den Namen der Datei bzw. des Ordners in das Suchfeld ein und drücken Sie die Eingabetaste, um das Suchergebnis auf der Seite "Suche" anzuzeigen. Der gesuchte Text wird gelb hervorgehoben.

Wenden Sie eine oder mehrere der folgenden Suchmethoden an und drücken Sie die Eingabetaste.

  • Geben Sie mindestens einen Buchstaben des Namens der gesuchten Datei bzw. des Ordners in das Suchfeld ein.

  • Geben Sie ein Sternchen (*) in das Feld "iFolder-Ordner suchen" ein, damit eine Liste aller im System vorhandenen iFolder-Ordner wiedergegeben wird.

  • Lassen Sie das Feld "iFolder-Ordner suchen" leer, damit eine Liste aller im System vorhandenen iFolder-Ordner wiedergegeben wird.

Klicken Sie nicht auf die Seite, bis sie vollständig neu geladen ist. Anschließend können Sie die im Suchergebnis angezeigte Datei- bzw. Ordnerliste durchsuchen oder verwalten. Blättern Sie im Suchergebnis, bis Sie die gewünschte Datei bzw. den gewünschten Ordner gefunden haben. Zusammen mit dem Namen der Datei bzw. des Ordners kann Ihnen auch das Erstellungsdatum dabei helfen, die richtige Datei bzw. den richtigen Ordner zu identifizieren.
