Neuen iFolder-Ordner erstellen

Führen Sie zum Erstellen eines neuen iFolder-Ordners die folgenden Schritte aus:

  1. Klicken Sie auf den auf der Startseite angezeigten Link für einen neuen iFolder-Ordner.

  2. Geben Sie auf der daraufhin geöffneten Seite die folgenden Informationen an:

    3. Name
    Weisen Sie dem iFolder-Ordner einen Namen zu, beispielsweise "MeinOrdner".

    Hinweis: Verwenden Sie die folgenden ungültigen Zeichen nicht für die Namen von iFolder-Ordnern oder für die Namen von darin enthaltenen Ordnern und Dateien:

    \/:*?"<>|;

    WICHTIG: Wenn Sie versuchen, einen iFolder-Ordner zu erstellen, dessen Name die vom Dateisystem des iFolder-Servers festgelegte Obergrenze für Dateien überschreitet, wird eine Fehlermeldung ausgegeben, die besagt, dass der Name des iFolder-Ordners so lang ist, dass er nicht erstellt werden kann.


    Beschreibung
    Eine kurze Beschreibung des iFolder-Ordners.

    Typ
    Wählen Sie den gewünschten Typ des zu erstellenden iFolder-Ordners aus.
    Encrypted
    Wählen Sie diese Option aus, wenn der iFolder-Ordner und die darin enthaltenen Dateien verschlüsselt werden sollen.

    WICHTIG:Diese Option ist nur verfügbar, wenn der Administrator für Ihre Verschlüsselungsrichtlinie "EIN" festgelegt hat.

    Regulär
    Wählen Sie "Gemeinsam benutzbar" aus, wenn der iFolder-Ordner für andere iFolder-Konten freigegeben werden soll.

    WICHTIG:Freigegebene iFolder-Ordner können nicht verschlüsselt werden bzw. verschlüsselte iFolder-Ordner können nicht freigegeben werden.

  3. Wählen Sie den Wiederherstellungsagenten aus der Dropdown-Liste aus.

    Hinweis: Diese Option ist nur verfügbar, wenn Ihr iFolder-Administrator beim Konfigurieren des iFolder-Servers die Option "Wiederherstellungsagent" ausgewählt hat.

    Der Wiederherstellungsagent kann ein System, eine Organisation oder eine vom iFolder-System unabhängige Person sein, wo Sie den Schlüssel zur Wiederherstellung Ihres geheimen Passwortsatzes speichern.

    WICHTIG: Wenn Sie keinen Wiederherstellungsagenten festlegen, können die Daten nicht wiederhergestellt werden, falls Sie Ihren Passwortsatz verlieren.

  4. Geben Sie den Passwortsatz in das Feld "Passwortsatz eingeben" ein.

  5. Geben Sie den Passwortsatz zur Bestätigung erneut ein.

  6. Klicken Sie auf "Erstellen", um den Erstellungsprozess des neuen iFolder-Ordners abzuschließen.

    Klicken Sie auf "Abbrechen", um den Prozess abzubrechen.

Verwandte Themen

Zugreifen auf Ordner und Dateien in Ihrem iFolder-Ordner
Anzeigen von iFolder-Ordnereigenschaften
Verwalten von iFolder-Ordnermitgliedern
