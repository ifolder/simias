Ändern des Passworts

Führen Sie die nachfolgenden Schritte aus, um das Passwort Ihres iFolder-Kontos zu ändern:

  1. Vergewissern Sie sich, dass das Kontrollkästchen Passwort ändern aktiviert ist. Dieses Kontrollkästchen ist standardmäßig aktiviert und die Felder zum Ändern des Passworts können nur bearbeitet werden, wenn dieses Kontrollkästchen aktiviert ist.
  2. Geben Sie Ihr aktuelles Passwort im Feld Aktuelles Passwort an.
  3. Geben Sie das Passwort, das Sie für das aktuelle iFolder-Konto festlegen möchten, im Feld Neues Passwort an.
  4. Geben Sie das neue Passwort zur Bestätigung im Feld Neues Passwort bestätigen erneut an.
  5. Klicken Sie auf Speichern, um Ihre Änderungen zu speichern.

Hinweis: iFolder ändert das Passwort direkt im LDAP-Server. Wenn derselbe LDAP-Server als Benutzerverwaltungsressource für eine andere Anwendung fungiert, ist das neue Passwort
auch in diesen Anwendungen verfügbar. Nach der Änderung Ihres Passworts läuft die iFolder-Sitzung aus und Sie werden abgemeldet. Sie müssen sich anschließend neu anmelden, um eine Verbindung zu den iFolder-Services herzustellen.
