Zugreifen auf iFolder-Ordner

Verwenden Sie diese Seite, um auf alle für Sie in diesem Konto verfügbaren iFolder-Ordner zuzugreifen. Die verfügbaren iFolder-Ordner umfassen diejenigen, deren Eigentümer Sie sind, und diejenigen, die andere Personen für Sie oder die LDAP-Gruppe, der Sie angehören, freigegeben haben. Sie können über Web Access auf freigegebene iFolder-Ordner zugreifen, selbst wenn Sie den iFolder-Ordner nicht auf einem lokalen Computer eingerichtet haben.

Das Symbol zeigt an, dass der iFolder-Ordner deaktiviert ist. Sie können nicht auf einen deaktivierten iFolder-Ordner zugreifen oder ihn für iFolder-Operationen auswählen.

Sie können für jeden iFolder-Ordner das Erstellungsdatum und den Eigentümer anzeigen. Klicken Sie auf den iFolder-Link, um die Liste der Ordner und Dateien, die der iFolder-Ordner enthält, auf einer neuen Seite anzuzeigen.

Anmeldung und Abmeldung

Falls für die Systemverbindung eine Zeitüberschreitung erfolgt, während Sie Ihre iFolder-Ordner bearbeiten, werden Sie durch Web Access auf die Anmeldeseite zurückgeleitet. Sie können sich erneut anmelden, um mit der Bearbeitung fortzufahren, ohne Ihren Browser zu schließen.

Sie können sich abmelden, indem Sie im Header-Bereich oben rechts auf "Abmelden" klicken und anschließend Ihren Browser schließen. Durch die Abmeldung werden Sie zu der URL weitergeleitet, die Ihr Administrator während der Einrichtung festgelegt hat. Wenn Sie sich nicht abmelden, bleibt die Sitzung bis zur Zeitüberschreitung der Sitzung geöffnet. Dies kann ein Sicherheitsrisiko darstellen.

Suchen nach iFolder-Ordnern

Das Suchwerkzeug oben rechts auf der Seite unterstützt Sie dabei, den gewünschten iFolder-Ordner zu finden. Das Suchergebnis wird auf derselben Seite geöffnet. Der gesuchte Text wird hervorgehoben dargestellt.

Wenden Sie eine oder mehrere der folgenden Suchmethoden an:

  • Wählen Sie ein Filterkriterium (Alle, Neuere, Eigene, Freigegebene) für den iFolder-Ordner und geben Sie dann mindestens einen Buchstaben in das Feld "iFolder-Ordner suchen" ein.

  • Geben Sie ein Sternchen (*) in das Feld "iFolder-Ordner suchen" ein, damit eine Liste aller im System vorhandenen iFolder-Ordner wiedergegeben wird.

  • Lassen Sie das Feld "iFolder-Ordner suchen" leer, damit eine Liste aller im System vorhandenen iFolder-Ordner wiedergegeben wird.

Klicken Sie nicht auf die Seite, bis sie vollständig neu geladen ist. Anschließend können Sie die im Suchergebnis angezeigte iFolder-Ordnerliste durchsuchen. Blättern Sie im Suchergebnis, bis Sie den zu verwaltenden iFolder-Ordner gefunden haben. Zusammen mit dem Namen des iFolder-Ordners kann Ihnen das Erstellungsdatum dabei helfen, den richtigen Ordner zu identifizieren.

Klicken Sie auf den iFolder-Ordnernamen, um eine Liste der enthaltenen Ordner und Dateien anzuzeigen, Dateien hoch- oder herunterzuladen, Ordner zu erstellen oder Dateien und Ordner zu löschen.

Neuen iFolder-Ordner erstellen

Führen Sie zum Erstellen eines neuen iFolder-Ordners die folgenden Schritte aus:

  1. Klicken Sie auf den auf der Startseite angezeigten Link für einen neuen iFolder-Ordner.

  2. Geben Sie auf der daraufhin geöffneten Seite die gewünschten Informationen ein und klicken Sie auf "Erstellen".

iFolder-Ordner löschen

So löschen Sie einen iFolder-Ordner:

  1. Klicken Sie auf den zu löschenden iFolder-Ordner.

  2. Klicken Sie auf der sich öffnenden iFolder-Seite auf den angezeigten Link zum Löschen.

    Ein Dialogfeld wird angezeigt, in dem Sie gefragt werden, ob Sie den aktuellen iFolder-Ordner löschen möchten.

  3. Klicken Sie in diesem Dialogfeld auf "OK", um das Löschen des iFolder-Ordners zu bestätigen.

  4. Klicken Sie auf "Abbrechen", um den Löschvorgang abzubrechen.

iFolder-Quoten

Sie können den dem aktuellen Konto insgesamt zugeordneten Speicherplatz anzeigen. Links werden der benutzte Speicherplatz sowie der insgesamt verfügbare Speicherplatz angezeigt.

WICHTIG: Wenn Sie Mitglied einer LDAP-Gruppe sind, wird die Speicherquote für die LDAP-Gruppe angezeigt, es sei denn, sie wurde auf Benutzerebene festgelegt.

Zum Anzeigen der iFolder-Quote eines bestimmten iFolder-Ordners klicken Sie auf den iFolder-Link, um die Seite zum Durchsuchen des iFolder-Ordners zu öffnen. Der benutzte sowie der verfügbare Speicherplatz des aktuellen iFolder-Ordners wird unter "iFolder-Quote" links auf der Seite zum Durchsuchen angezeigt.
