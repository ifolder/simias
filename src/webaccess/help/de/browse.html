Zugreifen auf Ordner und Dateien in Ihrem iFolder

Sie können die Seite zum Durchsuchen anzeigen, indem Sie auf einen der iFolder-Links auf der Startseite klicken.

Verwenden Sie diese Seite, um auf Ordner und Dateien im ausgewählten iFolder-Ordner des aktuellen Kontos zuzugreifen. Für jeden ausgewählten iFolder-Ordner können Sie eine Liste der enthaltenen Ordner und Dateien anzeigen, Dateien hoch- oder herunterladen, Ordner erstellen oder Dateien und Ordner löschen.

HINWEIS: Es ist nicht möglich, auf iFolder-Ordner mit deaktiviertem zuzugreifen oder diese zu öffnen.

Verwalten von Ordnern und Dateien in einem iFolder-Ordner

In einem iFolder-Ordner können Sie die folgenden Aufgaben ausführen:

Eine Liste von Ordnern und Dateien anzeigen
Auf der Seite "Durchsuchen" werden Ordner und Dateien alphabetisch aufgelistet. Blättern Sie in der Liste, um das gewünschte Verzeichnis bzw. die gewünschte Datei zu finden.

In einem Verzeichnis navigieren
Klicken Sie auf den Namen des Ordners , um den Ordner zu öffnen und eine Liste des Ordnerinhalts anzuzeigen. Verwenden Sie die Navigationsleiste im oberen Bereich des Fensters, um zu einem Verzeichnis einer höheren Ebene zurückzukehren. Sie können entweder auf sein Symbol oder seinen Namen klicken, um zu dem gewünschten Verzeichnis zu gelangen.

Eine Datei öffnen oder speichern
Klicken Sie auf den Namen der Datei , die Sie öffnen oder speichern möchten.

  • Klicken Sie auf "Öffnen", um den Inhalt der Datei anzuzeigen. Auf Ihrem Computer muss eine kompatible Anwendung installiert sein, um die Datei zu öffnen. Wenn Sie in der Datei arbeiten möchten, laden Sie sie herunter, ändern Sie die Datei und laden Sie sie anschließen erneut hoch.

  • Klicken Sie auf "Speichern", um die Datei auf Ihrem lokalen Festplattenlaufwerk oder auf einem zugeordneten Netzlaufwerk zu speichern. Navigieren Sie zu der Position, an der Sie die Datei speichern möchten, und klicken Sie auf "Speichern". Die Datei wird auf Ihren Computer geladen.

Einen neuen Ordner erstellen
Klicken Sie auf den Link "Neuer Ordner", um den neuen Ordner zu erstellen.

Ordner oder Dateien löschen
Aktivieren Sie das Kontrollkästchen neben einem oder mehreren zu löschenden Ordnern oder Dateien und klicken Sie anschließend auf "Löschen", um die Aktion zu bestätigen.

Datei heraufladen
Klicken Sie auf "Heraufladen" und klicken Sie dann auf der Seite "Heraufladen" auf "Durchsuchen", um die lokale Datei aufzufinden, und klicken Sie dann auf "Öffnen" > "Heraufladen". Wenn die Seite aktualisiert wird, wird die Datei entsprechend ihrem Dateinamen in einer alphabetisch sortierten Liste angezeigt.

Anzeigen des Verlaufs
Klicken Sie auf , um den Verlauf anzuzeigen.

iFolder-Ordner löschen

So löschen Sie einen iFolder-Ordner:

  1. Klicken Sie auf den Link "Löschen" unter "Aktionen" links auf der Seite.

    Ein Dialogfeld wird angezeigt, in dem Sie gefragt werden, ob Sie den aktuellen iFolder-Ordner löschen möchten.

  2. Klicken Sie in diesem Dialogfeld auf "OK", um das Löschen des iFolder-Ordners zu bestätigen.

  3. Klicken Sie auf "Abbrechen", um den Prozess abzubrechen.
