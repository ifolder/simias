Creación de una carpeta iFolder nueva

Realice los pasos siguientes para crear una carpeta iFolder nueva:

  1. Haga clic en el enlace Nueva carpeta iFolder que aparece en la página principal.

  2. En la página nueva que aparece, especifique la información siguiente:

    3. Nombre
    Asigne un nombre a la carpeta iFolder como, por ejemplo, mi_carpeta.

    Nota: No utilice los siguientes caracteres no válidos en los nombres de carpetas iFolder ni en los nombres de carpetas y archivos que estas contienen:

    \/:*?"<>|;

    IMPORTANTE: cuando intente crear una carpeta iFolder cuyo nombre supere el límite de archivo admitido por el sistema de archivos del servidor iFolder, el sistema le enviará un mensaje de error para informarle de que el nombre de la carpeta iFolder es demasiado largo y no puede crearse.


    Descripción
    Descripción breve de la carpeta iFolder.

    Tipo
    Seleccione el tipo de carpeta iFolder que desea crear.
    Cifrado
    Seleccione esta opción para cifrar la carpeta iFolder, así como los archivos que ésta contiene.

    IMPORTANTE: esta opción no estará habilitada hasta que el administrador establezca la directiva de cifrado en Activo.

    Normal
    Seleccione Compartible para compartir la carpeta iFolder con otras cuentas de iFolder.

    IMPORTANTE: las carpetas iFolder compartidas no pueden cifrarse y las carpetas cifradas no pueden compartirse.

  3. Seleccione el agente de recuperación en la lista desplegable.

    Nota: esta opción no estará disponible si el administrador de iFolder no selecciona la opción Habilitar agente de recuperación durante la configuración del servidor iFolder.

    El agente de recuperación puede ser un sistema, una organización, o una persona independiente del sistema de iFolder en el que podrá almacenar la clave para recuperar la contraseña codificada secreta.

    IMPORTANTE: a menos que haya definido el agente de recuperación, los datos cifrados no podrán recuperarse si, por desgracia, olvida la contraseña codificada.

  4. Escriba la contraseña codificada en el campo Introduzca la contraseña codificada.

  5. Introduzca de nuevo la contraseña codificada para confirmarla.

  6. Haga clic en Crear para finalizar el proceso de creación de la nueva carpeta iFolder.

    Haga clic en Cancelar para cancelar la operación.
