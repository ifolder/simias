Criar um novo iFolder

Realize as seguintes etapas para criar um novo iFolder:

  1. Clique no link Novo iFolder fornecido na Home page.

  2. Na nova página aberta, especifique as seguintes informações:

    3. Nome
    Designe um nome para o iFolder, como minhapasta.

    Nota: Não use os seguintes caracteres inválidos nos nomes dos iFolders ou nos nomes das pastas e arquivos contidos nela:

    \/:*?"<>|;

    Para obter mais informações sobre convenções de nomeação, consulte a Seção 7.3 do iFolder Cross Platform User Guide (Guia do Usuário para Várias Plataformas do iFolder).

    IMPORTANTE: Se você tentar criar uma pasta do iFolder cujo nome exceda o limite suportado pelo sistema de arquivos do servidor iFolder, uma mensagem de erro será exibida dizendo que o nome da pasta iFolder não pode ser criado por ser extenso demais.


    Descrição
    Uma breve descrição do iFolder.

    Tipo
    selecione o tipo de iFolder que você deseja criar.
    Criptografado
    Selecione essa opção para criptografar seu iFolder e os arquivos dentro dele.

    IMPORTANTE:Essa opção só é habilitada quando o administrador define a política de criptografia como Ativar.

    Regular
    Selecione Compartilhável para compartilhar seu iFolder com outras contas do iFolder.

    IMPORTANTE:Os iFolders compartilhados não podem ser criptografados e os que estão criptografados não podem ser compartilhados.

  3. Selecione o agente de Recuperação na lista suspensa.

    Nota: Esta opção só estará disponível se o Administrador do iFolder selecionar a opção Habilitar Agente de Recuperação durante a configuração do servidor iFolder.

    O agente de Recuperação pode ser um sistema, uma organização ou uma pessoa independente do sistema do iFolder no qual você pode armazenar a chave para recuperar a sua frase secreta.

    IMPORTANTE: A menos que você tenha definido o seu agente de Recuperação, os dados criptografados não poderão ser recuperados caso você tenha perdido sua frase secreta.

  4. Digite a frase secreta no campo Digite a Frase Secreta.

  5. Digite a frase secreta novamente para sua confirmação.

  6. Clique em Criar para concluir a nova criação do iFolder.

    Clique em Cancelar para cancelar a operação.
