Доступ к папкам iFolder

Используйте эту страницу для доступа ко всем папкам iFolder, назначенным данной учетной записи. К доступным папкам iFolder относятся папки, владельцем которых являетесь Вы, и папки, которые используют другие пользователи совместно с Вами или с группой LDAPGroup, в которую Вы входите. С помощью веб-клиента можно получить доступ к совместно используемым папкам iFolder даже в том случае, если клиент iFolder не установлен на локальном компьютере.

Для каждой папки iFolder можно просмотреть дату создания и владельца. Щелкните ссылку папки iFolder для открытия нового окна со списком содержащихся в ней файлов и папок.

Вход и выход

Если во время работы с папками iFolder происходит тайм-аут системного соединения, веб-клиент возвращает пользователя на страницу входа. Можно повторно войти и продолжить работу, не закрывая веб-навигатор.

Для выхода щелкните "Выход" в верхней правой области заголовка и закройте веб-навигатор. Во время выхода выполняется переход по URL-адресу, заданному администратором при настройке конфигурации. Если выход не выполнить, сеанс остается открытым до тех пор, пока не наступит тайм-аут сеанса, что может представлять угрозу системы защиты.

Поиск папок iFolder

Для поиска папок iFolder используется средство поиска, расположенное в верхней правой части страницы. Результаты поиска отображаются на той же странице, при этом искомый текст в них выделяется.

Доступные методы поиска:

  • Выберите критерий фильтрации имени папки iFolder ("Все", "Последний", "Владелец", "Совместное использование"), а затем введите одну или несколько букв в поле "Поиск папок iFolder".

  • Введите символ звездочки (*) в поле "Поиск папок iFolder" для возврата к списку всех папок iFolder в системе.

  • Оставьте поле "Поиск папок iFolder" пустым для возврата к списку всех папок iFolder в системе.

Не щелкайте какие-либо участки страницы до ее полного обновления; после этого Вы сможете просмотреть папки iFolder, попавшие в результаты поиска. Выполняйте прокрутку вверх и вниз для просмотра результатов поиска и выбора нужной папки iFolder. Определить нужную папку iFolder можно по имени и дате создания.

Щелкните имя папки iFolder для просмотра списка содержащихся в ней папок и файлов, отправки и загрузки файлов, создания папок, а также удаления файлов и папок.

Создание новой папки iFolder

Для создания новой папки iFolder выполните следующие действия:

  1. Щелкните ссылку "Новая папка iFolder" на основной странице.

  2. На открывшейся странице укажите необходимую информацию и щелкните "Создать".

Удаление папки iFolder

Для удаления папки iFolder выполните следующие действия:

  1. Щелкните папку iFolder, которую необходимо удалить.

  2. На открывшейся странице iFolder щелкните ссылку "Удалить".

    Отображается диалоговое окно с запросом на удаление текущей папки iFolder.

  3. Щелкните в этом диалоговом окне кнопку "OK" для подтверждения удаления iFolder.

  4. Щелкните кнопку "Отмена" для отмены операции.

Квота iFolder

Можно просмотреть общий объем выделенного пространства для текущей учетной записи. В левой части страницы отображается информация о занятом объеме, а также об общем доступном объеме.

ВАЖНО. Если Вы являетесь членом группы LDAPGroup, отображается дисковая квота для группы LDAPGroup, если она не установлена на уровне пользователя.

Чтобы узнать квоту iFolder для любой из папок iFolder, щелкните ссылку iFolder, чтобы открыть страницу "Просмотр" для этой папки iFolder. Информация об используемом и доступном пространстве для текущей папки iFolder отображается в поле "Квота iFolder", расположенном в левой части страницы "Просмотр".
