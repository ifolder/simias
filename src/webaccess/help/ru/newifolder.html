Создание новой папки iFolder

Для создания новой папки iFolder выполните следующие действия:

  1. Щелкните ссылку "Новая папка iFolder" на основной странице.

  2. На открывшейся странице укажите следующую информацию:

    3. Имя
    Назначьте папке iFolder имя, например "myfolder".

    Примечание. Не используйте следующие недопустимые символы для именования папок iFolder или вложенных папок и файлов:

    \/:*?"<>|;

    ВАЖНО. При попытке создания папки iFolder с именем, длина которого превышает ограничение файловой системы iFolder, выдается сообщение об ошибке, предупреждающее о невозможности создания папки iFolder с таким длинным именем.


    Описание
    Краткое описание папки iFolder.

    Тип
    Выберите тип создаваемой папки iFolder.
    Шифрованная
    Выберите данную опцию, чтобы зашифровать папку iFolder и содержащиеся в ней файлы.

    ВАЖНО. Данная опция недоступна до тех пор, пока администратор не включит политику шифрования.

    Обычная
    Выберите параметр "Совместно используется" для разрешения совместного использования Вашей папки iFolder с другими учетными записями iFolder.

    ВАЖНО. Совместно используемые папки iFolder нельзя зашифровать, а зашифрованные - совместно использовать.

  3. Выберите в раскрывающемся списке агент восстановления.

    Примечание. Данная опция будет недоступна, если администратор при настройке сервера iFolder не выбрал параметр "Включить агент восстановления".

    В качестве агента восстановления может выступать система, организация или лицо, не зависимое от системы iFolder, у которого может храниться ключ для восстановления секретной контрольной фразы.

    ВАЖНО. Если агент восстановления не задан, то в случае утери контрольной фразы восстановление зашифрованных данных невозможно.

  4. Введите контрольную фразу в поле "Контрольная фраза".

  5. Введите ключевую фразу еще раз для подтверждения.

  6. Щелкните "Создать" для завершения создания новой папки iFolder.

    Щелкните кнопку "Отмена" для отмены операции.
