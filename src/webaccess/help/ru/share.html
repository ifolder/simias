Добавление нового пользователя с правами на совместное использование

Для добавления нового пользователя или группы LDAPGroup в список пользователей с правами на совместное использование выполните следующие действия:

  1. С помощью функции поиска найдите пользователя или группу LDAPGroup, которым необходимо предоставить права на совместное использование папки iFolder.

    2. Выберите критерий фильтрации ("Имя пользователя", "Имя", "Фамилия"), а затем введите одну или несколько букв в поле "Поиск".

  2. Щелкните (синий знак "плюс"), чтобы добавить пользователя в список пользователей с правами на совместное использование.

    Добавленные члены отображаются в правой части страницы в поле "Совместное использование". После добавления пользователя цвет значка "плюс" рядом с именем папки iFolder изменяется на черный.

    В случае совместного использования папки iFolder с группой LDAPGroup права на совместное использование предоставляются всем членам группы LDAPGroup.

  3. Щелкните "Совм. исп." для подтверждения операции.

    Щелкните "Отмена" для отмены всех операций, выполненных на данной странице.
