Просмотр свойств iFolder

Используйте эту страницу для просмотра свойств iFolder выбранной папки iFolder. Здесь можно также изменить описание, введенное при создании папки iFolder.

Можно просматривать следующие свойства iFolder.

Параметр

Описание

Имя

(только просмотр)

Имя, назначенное папке iFolder.

Описание

Краткое описание папки iFolder.

Для изменения описания iFolder щелкните "Изменить". В поле "Описание" новой открывшейся страницы введите описание и щелкните "Сохранить".

Чтобы отменить внесенные изменения, щелкните "Отмена".

Последнее изменение

(только просмотр)

Отображение времени последнего изменения папки iFolder любым пользователем.

Создан.

(только просмотр)

Отображение даты создания папки iFolder.

Права

(только просмотр)

Права владельца папки iFolder по отношению к этой папке.

Владелец

(только просмотр)

Отображение имени пользователя владельца папки iFolder.

Полное характерное имя владельца папки iFolder. Владельцем папки iFolder может быть индивидуальный пользователь или группа LDAPGroup.

Если выбранной папкой iFolder владеет группа LDAPGroup, у всех членов этой группы одинаковые права на эту папку iFolder.

Размер

(только просмотр)

Отображение размера папки iFolder.

Члены

(только просмотр)

Отображение общего количества пользователей, принадлежащих этой папке iFolder. Это могут быть индивидуальные пользователи или группы LDAPGroup.

Файлы

(только просмотр)

Отображение общего количества файлов в данной папке iFolder.

Папки

(только просмотр)

Отображение общего количества папок в данной папке iFolder.

Заблокирована

(только просмотр)

Отображение состояния синхронизации iFolder. Если администратор iFolder включил синхронизацию, то для параметра "Заблокировано" установлено значение "Нет". Если синхронизация выключена, то для параметра "Заблокировано" установлено значение "Да".
