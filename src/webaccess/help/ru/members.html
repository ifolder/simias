Управление членами iFolder

Используйте данный интерфейс для добавления или удаления членов, имеющих права на совместное использование, назначения прав членам и изменения владения выбранной папки iFolder.

В данной справке рассматривается следующая информация:

Управление пользователями папки iFolder

В папке iFolder можно выполнять следующие задачи:

Добавление нового пользователя с правами на совместное использование

Щелкните "Добавить" для добавления новых членов с правами на совместное использование в выбранную папку iFolder. В качестве члена можно добавить индивидуального пользователя iFolder или группу LDAPGroup. Открывается новая страница, на которой можно добавить пользователей в список пользователей с правами на совместное использование.

Назначение прав пользователю с правами на совместное использование

На странице "Члены" выберите членов с правами на совместное использование и щелкните назначаемые им права.

Общие сведения о правах доступа пользователей iFolder

Клиент iFolder поддерживает владельца и три уровня доступа для членов папки iFolder:

ВАЖНО. Все члены группы LDAPGroup наследуют права доступа, установленные для данной группы LDAPGroup.

Только для чтения
Уровень доступа члена совместно используемой папки iFolder с правом только для чтения в каталогах и файлах папки iFolder. Этот член может скопировать файл из папки iFolder в другое место и изменить его вне папки iFolder.

Для чтения/записи
Уровень доступа члена совместно используемой папки iFolder с правом на чтение/запись в каталогах и файлах папки iFolder.

Полный контроль
Уровень доступа члена совместно используемой папки iFolder с правом на полный контроль. Пользователь с правами "Полный контроль" имеет доступ на чтение/запись в папке iFolder, управляет членством и правами доступа всех пользователей, за исключением владельца. Однако правом на удаление папки iFolder обладает только владелец.

Владелец
Только один пользователь может быть владельцем папки iFolder. Как правило, это пользователь, который создал данную папку iFolder, и только он может ее удалить. Владелец может использовать клиент iFolder для передачи права на владение другому члену папки iFolder. Администратор iFolder может передавать право на владение любому пользователю сервера iFolder.

Владелец папки iFolder имеет право на полный контроль. Этот пользователь имеет доступ для чтения/записи к папке iFolder, управляет членством и правами доступа пользователей, а также может удалять право "Полный контроль" для любого члена.

Если пользователь удаляется из системы iFolder, принадлежащие ему папки iFolder становятся папками с нарушенными ассоциациями. Владельцем папок iFolder с нарушенными ассоциациями временно назначается администратор iFolder Поддержка членства и синхронизация продолжаются, пока администратор iFolder не определит, нужно ли удалить папку iFolder с нарушенными ассоциациями или назначить другого владельца.
