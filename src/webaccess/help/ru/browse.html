Доступ к файлам и папкам в iFolder

Для перехода к странице "Просмотр" можно щелкнуть любую ссылку iFolder на основной странице.

Используйте эту страницу для доступа к папкам и файлам, расположенным внутри выбранной папки iFolder в текущей учетной записи. Для каждой выбранной папки iFolder можно просматривать список файлов и папок, отправлять и загружать файлы, создавать папки, а также удалять файлы и папки.

ПРИМЕЧАНИЕ. Нельзя выбрать запрещенные папки iFolder или получить к ним доступ.

Управление папками и файлами в iFolder

В папке iFolder можно выполнять следующие задачи:

Просмотр списка файлов и папок
На странице "Просмотр" файлы и папки отображаются в алфавитном порядке. Перейдите к необходимой папке или к необходимому файлу.

Навигация по каталогу
Щелкните имя папки для открытия папки и просмотра списка ее содержимого. Чтобы вернуться к папке более высокого уровня, воспользуйтесь расположенной в верхней части окна панелью навигации. Для возврата к нужному каталогу можно щелкнуть соответствующее имя или значок.

Открыть или сохранить файл
Щелкните имя файла , который необходимо открыть или сохранить.

  • Щелкните "Открыть" для просмотра содержимого файлов. Для открытия файла на компьютере должно быть установлено соответствующее приложение. Если Вы планируете работать с файлом, загрузите файл, измените его, а затем отправьте файл.

  • Щелкните "Сохранить" для сохранения файла на локальном жестком диске или назначенном сетевом диске. Перейдите к расположению, куда необходимо поместить файл, а затем щелкните "Сохранить". Файл загружается на компьютер.

Создание Новой Папки
Щелкните ссылку "Новая папка", чтобы создать новую папку.

Удаление папок и файлов
Установите флажок рядом с файлами и папками, которые необходимо удалить, и щелкните "Удалить" для подтверждения удаления.

Отправка файла
Щелкните "Загрузить", на странице "Загрузить" щелкните "Просмотр", выберите локальные файлы, которые необходимо отправить, и щелкните "Открыть" > "Загрузить". После обновления страницы файл отобразится в алфавитном порядке.

Просмотр протокола
Щелкните для просмотра протокола.

Удаление папки iFolder

Для удаления папки iFolder выполните следующие действия:

  1. Щелкните ссылку "Удалить", расположенную под элементом "Действия" в левой части страницы.

    Отображается диалоговое окно с запросом на удаление текущей папки iFolder.

  2. Щелкните в этом диалоговом окне кнопку "OK" для подтверждения удаления папки iFolder.

  3. Щелкните кнопку "Отмена" для отмены операции.
