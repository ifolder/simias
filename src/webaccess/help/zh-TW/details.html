檢視 iFolder 內容

使用此頁面檢視選定 iFolder 的 iFolder 內容。您也可以在建立 iFolder 後編輯先前提供的描述內容。

您可以檢視下列 iFolder 內容。

參數

描述

名稱

(僅能檢視)

指定給 iFolder 的名稱。

描述

iFolder 的簡短描述。

若要編輯 iFolder 的描述，請按一下「編輯」。在開啟的新頁面中，於「描述」欄位內輸入新的描述，然後按一下「儲存」。

若要取消所做的變更，請按一下「取消」。

上次修改

(僅能檢視)

若上次修改是由任何使用者進行的，請顯示上次修改 iFolder 的時間。

已建立

(僅能檢視)

建立 iFolder 後顯示日期。

權限

(僅能檢視)

顯示 iFolder 擁有者對於 iFolder 所擁有的權限。

擁有者

(僅能檢視)

顯示 iFolder 擁有者的使用者名稱。

這是 iFolder 擁有者的完整可辨識名稱。iFolder 擁有者可為個別使用者或 LDAPGroup。

如果 LdapGroup 擁有選定的 iFolder，則所有屬於此群組的成員都擁有相同的權限。

大小

(僅能檢視)

顯示 iFolder 的大小。

成員

(僅能檢視)

報告屬於 iFolder 的使用者總數。包括個別使用者與 LDAPGroup。

檔案

(僅能檢視)

顯示此 iFolder 內的檔案總數。

資料夾

(僅能檢視)

顯示此 iFolder 內的資料夾總數。

鎖定

(僅能檢視)

顯示是否已啟用 iFolder 同步。如果「iFolder 管理員」將 iFolder 同步設為啟用，則「鎖定」的值為「否」。如果設為停用，則「鎖定」的值為「是」。
