存取 iFolder

使用此頁存取可供此帳戶存取的所有 iFolder。可用的 iFolder，包括您所擁有及其他人與您或您所屬 LDAPGroup 共享的 iFolder。即使本機電腦上沒有設定 iFolder，您也可以透過 Web 存取來存取共用的 iFolder。

指出 iFolder 為「一般」的 iFolder。
指出 iFolder 為「加密」的 iFolder。
指出該 iFolder 已停用。您無法存取停用的 iFolder，或選取該 iFolder 進行任何 iFolder 操作。

對於每個 iFolder，您可以檢視建立日期及其個別擁有者。按一下 iFolder 連結，在新頁面內檢視其資料夾和檔案清單。

登入與登出

當您正在使用 iFolder 時，如果系統連線逾時，Web 存取會讓您返回登入頁面。您可以再次登入，以繼續使用 iFolder 而不需要關閉瀏覽器。

可以藉由在右上方標題區域中按一下「登出」來登出，然後關閉瀏覽器。登出會重改方向為您的「管理員」在安裝設定時設定的 URL。如果您未登出，則工作階段會保持開啟狀態直到其逾時，這可能會有安全性風險。

搜尋 iFolder

搜尋工具位於頁面的右上方，可協助您找出所需的 iFolder。搜尋結果將會開啟至相同頁面，在搜尋結果中會以反白顯示搜尋文字。

使用下列一或多種搜尋方法：

  • 選取 iFolder 名稱的過濾器標準 (「全部」、「最近」、「擁有」、「共用」)，然後在「搜尋 iFolder」欄位內輸入一或多個字母。

  • 在「搜尋 iFolder」欄位內輸入星號 (*)，以傳回系統上所有 iFolder 的清單。

  • 讓「搜尋 iFolder」欄位保留為空白，以傳回系統上所有 iFolder 的清單。

在頁面完全重新整理之前，請勿按下頁面的任何地方，接著您便可以瀏覽在「搜尋結果」中列出的 iFolder。向上及向下捲動以瀏覽搜尋結果並找出您想要管理的 iFolder。iFolder 名稱的組合和建立日期可協助您識別所尋找的 iFolder。

按一下 iFolder 的名稱，您即可檢視其資料夾和檔案的清單、上載及下載檔案、建立資料夾，以及刪除資料夾和檔案。

建立新的 iFolder

請執行下列步驟建立新的 iFolder：

  1. 按一下首頁中新 iFolder 連結。

  2. 在開啟的新頁面上指定資訊，然後按一下「建立」。

刪除 iFolder

若要刪除 iFolder：

  1. 按一下您要刪除的 iFolder。

  2. 在開啟的 iFolder 頁上，按一下頁面上的「刪除」連結。

    此時會出現一個對話方塊，詢問您是否要刪除目前的 iFolder。

  3. 按一下對話方塊內的「確定」，確認刪除 iFolder。

  4. 按一下「取消」取消操作。

iFolder 配額

您可以檢視為目前帳戶配置的總空間。頁面的左端會顯示已使用的空間，以及可供使用的總空間。

重要：如果您是 LDAPGroup 成員，除非已在使用者層級設定，否則將顯示 LDAPGroup 的磁碟配額。

若要檢視任何指定 iFolder 的 iFolder 配額，請按一下 iFolder 連結開啟「iFolder 瀏覽」頁。您可以在「瀏覽」頁左端的「iFolder 配額」下方檢視目前 iFolder 列出的已使用及可用的空間。
