建立新的 iFolder

請執行下列步驟建立新的 iFolder：

  1. 按一下首頁中新 iFolder 連結。

  2. 在開啟的新頁面中，指定下列資訊：

    3. 名稱
    指定 iFolder 的名稱，例如 myfolder。

    注意：請勿在 iFolder 名稱或 iFolder 內的資料夾和檔案名稱中使用下列無效字元：

    \/:*?"<>|;

    如需有關命名慣例的詳細資訊，請參閱《iFolder 跨平台使用者指南》的 7.3 節。

    重要：當您嘗試建立的 iFolder 名稱超過 iFolder 伺服器檔案系統所支援的檔案限制時，將出現 iFolder 名稱過長而無法建立的錯誤訊息。


    描述
    iFolder 的簡短描述。

    類型
    選取您要建立的 iFolder 類型。
    Encrypted
    選取這個選項將其中的 iFolder 和檔案加密。

    重要：您的「管理員」必須先將您的加密規則設為「啟用」，這個選項才會啟用。

    一般
    選取「可共用」，與其他 iFolder 帳戶共用您的 iFolder。

    重要：「共用的 iFolder」不能加密，而已加密的 iFolder 則無法共用。

  3. 從下拉式清單中選取「修復」代理程式。

    注意：如果您的「iFolder 管理員」未在設定 iFolder 時選取「啟用修復」代理程式選項，這個選項將無法使用。

    「修復」代理程式可以是一個系統、組織或與 iFolder 系統無關的人員，您可以在其中儲存修復秘密通關密語的金鑰。

    重要：除非已設定「修復」代理程式，否則便無法在發生遺失通關密語的不幸事件時，將加密資料復原。

  4. 請在「輸入通關密語」欄位中輸入通關密語。

  5. 再次輸入通關密語以確認。

  6. 按一下「建立」完成建立新增 iFolder 的程序。

    按一下「取消」取消操作。
