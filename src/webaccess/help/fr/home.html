Accès à vos dossiers iFolder

Cette page permet d'accéder à tous les dossiers iFolder qui vous sont accessibles sur ce compte. Les dossiers iFolders accessibles englobent ceux que vous possédez et ceux que d'autres ont partagés avec vous ou avec le groupe LDAP auquel vous appartenez. Vous pouvez accéder à des dossiers iFolder partagés via Web Access, même si vous n'avez pas configuré le dossier iFolder sur un ordinateur local.

L'icône indique que le dossier iFolder est ordinaire.
L'icône indique que le dossier iFolder est codé.
L'icône indique que le dossier iFolder est désactivé. Vous ne pouvez pas consulter, ni sélectionner un dossier iFolder désactivé pour quelque opération iFolder que ce soit.

Pour chaque dossier iFolder, vous pouvez afficher la date de création et le propriétaire respectif. Cliquez sur le lien du dossier iFolder pour afficher la liste de ses dossiers et fichiers dans une nouvelle page.

Login et logout

Si la connexion système est interrompue pendant que vous travaillez dans vos dossiers iFolder, Web Access vous renvoie à la page de login. Vous pouvez vous loguer de nouveau pour poursuivre votre travail sans fermer votre navigateur.

Déloguez-vous en cliquant sur Logout dans la zone d'en-tête supérieure droite, puis fermez votre navigateur. Logout vous redirigera vers l'URL que votre Admin a défini en configurant l'installation. Si vous ne vous déloguez pas, votre session demeure ouverte jusqu'à ce que le timeout y mette fin, ce qui peut présenter un risque pour la sécurité.

Recherche de dossiers iFolder

L'outil de recherche situé en haut à droite de la page vous aide à localiser le dossier iFolder souhaité. Les résultats de la recherche s'afficheront sur la même page, avec le texte de recherche surligné dans les résultats.

Utilisez une ou plusieurs des méthodes de recherche suivantes :

  • Sélectionnez un critère de filtre (Tout, Récent, Avec propriétaire, Partagé) pour le nom du dossier iFolder, puis entrez une ou plusieurs lettres dans le champ Rechercher dossiers iFolder.

  • Tapez un astérisque (*) dans le champ Rechercher dossiers iFolder pour obtenir une liste de tous les dossiers iFolder se trouvant sur le système.

  • Laissez le champ Rechercher dossiers iFolder vide pour obtenir une liste de tous les dossiers iFolder se trouvant sur le système.

Ne cliquez pas sur la page avant que celle-ci ne soit complètement rafraîchie. Une fois l'actualisation terminée, vous pouvez parcourir les dossiers iFolder listés dans les résultats de la recherche. Faites défiler la liste pour parcourir les résultats de la recherche et localiser le dossier iFolder à gérer. La combinaison du nom et de la date de création du dossier iFolder peut vous aider à identifier le dossier iFolder que vous recherchez.

Pour chaque iFolder, vous pouvez afficher la liste de ses dossiers et fichiers, télécharger des fichiers, créer des dossiers, et supprimer des dossiers et des fichiers.

Création d'un nouveau dossier iFolder

Pour créer un nouveau dossier iFolder, exécutez la procédure suivante :

  1. Cliquez sur le lien Nouveau dossier iFolder sur la page d'accueil.

  2. Spécifiez les informations sur la page qui s'ouvre, puis cliquez sur Créer.

Suppression d'un dossier iFolder

Pour supprimer un dossier iFolder :

  1. Cliquez sur le dossier iFolder à supprimer.

  2. Sur la page iFolder qui s'ouvre, cliquez sur le lien Supprimer.

    Une boîte de dialogue vous demande si vous souhaitez supprimer le dossier iFolder actuel.

  3. Cliquez sur OK pour confirmer la suppression du dossier iFolder.

  4. Cliquez sur Annuler pour annuler l'opération.

Quota iFolder

Vous pouvez afficher l'espace total occupé par le compte actuel. L'espace utilisé ainsi que l'espace total disponible sont indiqués sur le côté gauche de cette page.

IMPORTANT : si vous êtes membre d'un groupe LDAP, le quota de disque du groupe est affiché sauf s'il est défini au niveau utilisateur.

Pour afficher le quota iFolder d'un dossier iFolder donné, cliquez sur le lien iFolder pour ouvrir la page Parcourir dossiers iFolder. Vous pouvez afficher l'espace utilisé et l'espace disponible pour le dossier iFolder sélectionné sous Quota iFolder sur le côté gauche de la page Parcourir.
