Téléchargement d'un fichier

  1. Dans le champ Télécharger fichier, cliquez sur Parcourir pour localiser le fichier local à télécharger.

  2. Sélectionnez le fichier, puis cliquez sur Ouvrir > Télécharger.

    3. Lorsque la page est rafraîchie, le dossier apparaît dans la liste en ordre alphabétique. Par défaut, si vous téléchargez un fichier portant le même nom, il est écrasé. Si vous ne souhaitez pas écraser des fichiers existants, désélectionnez la case Écraser les fichiers existants.
