Création d'un nouveau dossier iFolder

Pour créer un nouveau dossier iFolder, exécutez la procédure suivante :

  1. Cliquez sur le lien Nouveau dossier iFolder sur la page d'accueil.

  2. Dans la nouvelle page qui s'ouvre, spécifiez les informations suivantes :

    3. Nom
    Assignez un nom au dossier iFolder, p. ex. mondossier.

    Remarque : n'utilisez pas les caractères incorrects suivants dans les noms des dossiers iFolder ou des dossiers et fichiers qu'ils contiennent :

    \/:*?"<>|;

    IMPORTANT : lorsque vous tentez de créer un dossier iFolder dont le nom dépasse la limite prise en charge par le système de fichiers du serveur iFolder, un message d'erreur indique que le nom du dossier iFolder est trop long pour qu'il soit créé.


    Description
    Affiche une brève description du dossier iFolder.

    Type
    Sélectionnez le type du dossier iFolder que vous souhaitez créer.
    Codée
    Sélectionnez cette option pour coder votre dossier iFolder et les fichiers qu'il contient.

    IMPORTANT :cette option n'est pas activée tant que votre administrateur n'active pas votre stratégie de codage.

    Standard
    Sélectionnez Partageable pour partager votre dossier iFolder avec d'autres comptes iFolder.

    IMPORTANT :les dossiers iFolder partagés ne peuvent être codés et ceux codés ne peuvent pas être partagés.

  3. Choisissez l'agent de récupération dans la liste déroulante.

    Remarque : cette option ne sera pas disponible si votre administrateur iFolder ne sélectionne pas l'option Activer l'agent de récupération lors de la configuration du serveur iFolder.

    L'agent de récupération peut être une organisation, une personne ou un système indépendant du système iFolder où vous pouvez enregistrer la clé servant à récupérer votre phrase secrète.

    IMPORTANT : si, par malchance, vous perdez votre phrase secrète, les données codées ne peuvent pas être récupérées, sauf si vous avez configuré votre agent de récupération à cet effet.

  4. Saisissez votre phrase secrète dans le champ Entrer la phrase secrète.

  5. Saisissez de nouveau la phrase secrète pour confirmer.

  6. Cliquez sur Créer pour terminer la création du nouveau dossier iFolder.

    Cliquez sur Annuler pour annuler l'opération.
