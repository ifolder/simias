Login à Web Access

Loguez-vous à Novell iFolder® 3.7 Web Access ou à une version ultérieure pour accéder aux dossiers et fichiers des dossiers iFolder disponibles pour vous ou pour le groupe LDAP dont vous êtes membre dans le compte iFolder 3.7 ou version ultérieure. Cela inclut les iFolders auxquels vous participez, mais que vous n'avez pas configurés sur un ordinateur local. (Vous ne pouvez pas accéder aux fichiers de votre compte Novell iFolder 2.x avec Web Access.) Votre nom d'utilisateur et votre mot de passe sont authentifiés auprès des services d'annuaire LDAP.

Vous pouvez également spécifier la langue à utiliser parmi celles prises en charge. Généralement, il s'agit de la même langue que le code de langue utilisé pour le serveur d'entreprise iFolder et votre ordinateur local. N'oubliez pas de configurer le paramètre de langues de votre navigateur pour qu'il prenne en charge la langue souhaitée.

  1. Spécifiez le nom d'utilisateur et le mot de passe pour le compte iFolder auquel vous voulez accéder.

    Les noms d'utilisateur et les mots de passe respectent la casse.

  2. Utilisez la liste déroulante Langues pour spécifier la langue à utiliser pour accéder à vos fichiers.

  3. Cliquez sur Se loguer.
