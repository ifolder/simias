Affichage des propriétés iFolder

Cette page permet d'afficher les propriétés iFolder du dossier iFolder sélectionné. Vous pouvez également éditer la description fournie lors de la création du dossier iFolder.

Vous pouvez afficher les propriétés iFolder suivantes :

Paramètre

Description

Nom

(affichage uniquement) 

Affiche le nom assigné au dossier iFolder

Description

Affiche une brève description du dossier iFolder.

Pour éditer la description du dossier iFolder, cliquez sur Éditer. Dans la nouvelle page qui s'affiche, entrez la nouvelle description dans le champ Description et cliquez sur Enregistrer.

Pour annuler les modifications effectuées, cliquez sur Annuler.

Dernière modification

(affichage uniquement) 

Affiche la dernière date de modification du dossier iFolder par un utilisateur quelconque.

Créé

(affichage uniquement) 

Affiche la date de création du dossier iFolder.

Droits

(affichage uniquement) 

Affiche les droits dont dispose le propriétaire du dossier iFolder.

Propriétaire

(affichage uniquement) 

Affiche le nom d'utilisateur du propriétaire du dossier iFolder.

Il s'agit du nom distinctif complet du propriétaire du dossier iFolder. Le propriétaire d'un dossier iFolder peut être un utilisateur individuel ou un groupe LDAP.

Si un groupe LDAP possède le dossier iFolder sélectionné, tous les membres appartenant à ce groupe bénéficient des mêmes droits sur ce dossier iFolder.

Taille

(affichage uniquement) 

Affiche la taille du dossier iFolder.

Membres

(affichage uniquement) 

Indique le nombre total d'utilisateurs auxquels le iFolder appartient. Cela englobe les utilisateurs individuels et les groupes LDAP.

Fichiers

(affichage uniquement) 

Affiche le nombre total de fichiers contenus dans ce dossier iFolder.

Dossiers

(affichage uniquement) 

Affiche le nombre total de dossiers contenus dans ce dossier iFolder.

Verrouillée

(affichage uniquement) 

Indique si la synchronisation iFolder est activée ou non. Si l'Admin iFolder définit la synchronisation iFolder comme activée, la valeur de Verrouillée est Non. Si elle est définie comme désactivée, la valeur de Verrouillée est Oui.
