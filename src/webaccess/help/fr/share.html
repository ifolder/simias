Ajout d'un nouvel utilisateur partagé

Pour ajouter un nouvel utilisateur ou groupe LDAP à la liste des utilisateurs partagés,

  1. Utilisez la fonctionnalité de recherche pour localiser l'utilisateur ou le groupe LDAP avec lequel vous souhaitez partager le dossier iFolder.

    2. Sélectionnez un critère de filtre (Nom d'utilisateur, Prénom, Nom), puis entrez une ou plusieurs lettres dans le champ Rechercher.

  2. Cliquez sur (le signe Plus de couleur bleue) pour ajouter les utilisateurs à la liste des utilisateurs partagés.

    Les membres ajoutés apparaissent sur la droite de la page sous Partager avec. Après l'ajout d'un utilisateur, le signe Plus en regard du nom du dossier iFolder devient noir.

    Lorsque vous partagez un dossier iFolder avec un groupe LDAP, il est partagé avec tous les membres du groupe LDAP.

  3. Cliquez sur Partager pour confirmer le partage.

    Cliquez sur Annuler pour annuler toutes vos opérations sur cette page.
