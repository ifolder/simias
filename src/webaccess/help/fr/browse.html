Accès aux dossiers et fichiers de votre dossier iFolder

Pour accéder à la page Parcourir, vous pouvez cliquer sur n'importe quel lien iFolder listé sur la Page d'accueil.

Cette page permet d'accéder aux dossiers et fichiers du dossier iFolder sélectionné dans le compte actuel. Pour chaque dossier iFolder sélectionné, vous pouvez afficher une liste de ses dossiers et fichiers, télécharger des fichiers, créer des dossiers et supprimer des dossiers et des fichiers.

REMARQUE : vous ne pouvez pas consulter, ni sélectionner de dossiers iFolder désactivés.

Gestion des dossiers et des fichiers d'un dossier iFolder

Vous pouvez exécuter les tâches suivantes à l'intérieur d'un iFolder :

Affichage d'une liste de dossiers et de fichiers
La page Parcourir établit une liste alphabétique des dossiers et fichiers. Faites défiler la liste pour localiser le répertoire ou le fichier qui vous intéresse.

Naviguer à l'intérieur d'un répertoire
Cliquez sur le nom du dossier pour ouvrir celui-ci et afficher la liste de son contenu. Pour revenir à un répertoire supérieur, utilisez la barre de navigation en haut de l'écran. Vous pouvez cliquer sur l'icône ou sur le nom pour revenir au répertoire souhaité.

Ouvrir un fichier ou l'enregistrer
Cliquez sur le nom du fichier à ouvrir ou à enregistrer.

  • Cliquez sur Ouvrir pour afficher le contenu du fichier. Vous devez posséder une application compatible sur votre ordinateur pour ouvrir le fichier. Si vous envisagez de travailler dans le fichier, téléchargez celui-ci, modifiez-le, puis téléchargez-le de nouveau.

  • Cliquez sur Enregistrer pour enregistrer le fichier sur votre disque dur local ou sur une unité réseau assignée. Accédez à l'emplacement où vous souhaitez placer le fichier, puis cliquez sur Enregistrer. Le fichier est téléchargé dans votre ordinateur.

Création d'un dossier
Cliquez sur le lien Nouveau Dossier pour créer un nouveau dossier.

Suppression de dossiers ou de fichiers
Cochez la case en regard d'un ou de plusieurs dossiers ou fichiers à supprimer, puis cliquez sur Supprimer pour confirmer la suppression.

Téléchargement d'un fichier
Cliquez sur Télécharger, sélectionnez Parcourir sur la page Télécharger pour localiser le fichier local à télécharger, puis cliquez sur Ouvrir > Télécharger. Lorsque la page est rafraîchie, le dossier apparaît dans la liste en ordre alphabétique.

Affichage de l'historique
Cliquez sur pour afficher l'historique.

Suppression d'un dossier iFolder

Pour supprimer un dossier iFolder :

  1. Cliquez sur le lien Supprimer listé sous Opérations sur le côté gauche de la page.

    Une boîte de dialogue vous demande si vous souhaitez supprimer le dossier iFolder actuel.

  2. Cliquez sur OK pour confirmer la suppression du dossier iFolder.

  3. Cliquez sur Annuler pour annuler l'opération.
