登录到万维网访问

使用 iFolder 3.7 或更高版本帐户登录到 Novell iFolder® 3.7 或更高版本万维网访问以获取对任何您或您所属的 LDAP 组可用的 iFolder 中的文件夹和文件的访问权限。这包括您加入但未在本地计算机上设置的 iFolder。（您不能使用万维网访问功能访问 Novell iFolder 2.x 帐户中的文件。）系统会根据 LDAP 目录服务对您的用户名和口令进行鉴定。

您还可以指定要使用哪种受支持的语言。该语言通常与 iFolder Enterprise Server 和本地计算机的语言代码相同。请务必对浏览器的语言设置进行配置，使其支持所需的语言。

  1. 指定要访问的 iFolder 帐户的用户名和口令。

    用户名和口令区分大小写。

  2. 使用“语言”下拉列表指定用于访问文件的语言。

  3. 单击“登录”。
