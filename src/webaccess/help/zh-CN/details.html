查看 iFolder 属性

使用此页面可查看选定 iFolder 的 iFolder 属性。也可编辑创建 iFolder 时提供的说明。

您可以查看以下 iFolder 属性。

参数

说明

名称

（仅供查看）

指派给 iFolder 的名称。

说明

iFolder 的简短说明。

要编辑 iFolder 说明，请单击“编辑”。在打开的新页面的“说明”字段中键入新说明，并单击“保存”。

若要取消所作的更改，请单击“取消”。

上次修改时间

（仅供查看）

显示上次由任意用户修改 iFolder 的时间。

创建时间

（仅供查看）

显示创建 iFolder 的日期

权限

（仅供查看）

显示 iFolder 拥有者对 iFolder 具有的权限。

拥有者

（仅供查看）

显示 iFolder 拥有者的用户名。

这是 iFolder 所有者的完整判别名。iFolder 所有者可以是单个用户，也可以是 LDAP 组。

如果某一 LDAP 组拥有选定的 iFolder，则属于该组的所有成员都对此 iFolder 具有相同权限。

大小

（仅供查看）

显示 iFolder 的大小。

成员数

（仅供查看）

报告属于该 iFolder 的用户总数。它包括单个用户和 LDAP 组。

文件数

（仅供查看）

显示该 iFolder 中的文件总数。

文件夹数

（仅供查看）

显示该 iFolder 中的文件夹总数。

已锁定

（仅供查看）

显示是否启用 iFolder 同步。如果 iFolder Admin 将 iFolder 同步设为“启用”，则“已锁定”的值为“否”。如果设为“禁用”，则“已锁定”的值为“是”。
