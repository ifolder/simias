访问 iFolder

使用此页面可访问在此帐户中您可用的所有 iFolder。可用 iFolder 包括您拥有的以及其他用户与您或与您所属的 LDAP 组共享的 iFolder。即使您未在本地计算机上设置 iFolder，也可以通过万维网访问来访问共享 iFolder。

图标指示该 iFolder 是常规的。
图标指示该 iFolder 是加密的。
图标指示该 iFolder 是禁用的。您不能访问被禁用的 iFolder，也不能选择它进行任何 iFolder 操作。

对于每个 iFolder，您可以查看已创建的数据和各自的拥有者。点击 iFolder 链接以查看它在新页面中的文件夹和文件列表。

登录和注销

当您在 iFolder 中工作时，如果系统连接超时，万维网访问会返回至登录页面。您可以再次登录以继续工作，而无需关闭浏览器。

您可以单击右上角标题区域的“注销”进行注销，然后关闭浏览器。注销后，您将被重定向至配置设置时 Admin 已设置的 URL。如果不注销，您的会话将在超时前一直保持打开状态，这可能会带来安全隐患。

搜索 iFolder

使用位于页面右上方的搜索工具可以查找需要的 iFolder。搜索结果将打开至相同的页面，且搜索结果中的搜索文本高亮显示。

使用下面一个或多个搜索方法：

  • 为 iFolder 的名称选择一个过滤器标准（“所有”、“最近”、“拥有”或“共享”），然后在“搜索 iFolder”字段中键入一个或多个字母。

  • 在“搜索 iFolder”字段中键入一个星号 (*) 可返回系统中所有 iFolder 的列表。

  • “搜索 iFolder”字段保留为空时可返回系统中所有 iFolder 的列表。

仅当页面完全刷新后才可单击页面，然后可浏览“搜索结果”中列出的 iFolder。上下滚动可浏览搜索结果并查找要管理的 iFolder。创建的 iFolder 名称和日期的组合可帮助您标识要寻找的 iFolder。

单击 iFolder 的名称可查看它的文件夹和文件列表，上载和下载文件，创建文件夹，以及删除文件夹和文件。

创建新 iFolder

执行下列步骤以创建新 iFolder：

  1. 单击主页中的“新建 iFolder”链接。

  2. 在打开的新页面上指定信息，然后单击“创建”。

删除 iFolder

删除 iFolder：

  1. 单击要删除的 iFolder。

  2. 在打开的 iFolder 页上，单击“删除”链接。

    显示一个对话框，询问是否要删除当前的 iFolder。

  3. 单击对话框中的“确定”可确认删除 iFolder。

  4. 单击“取消”可取消此操作。

iFolder 定额

您可以查看分配给当前帐户的总空间。该页左侧显示已用空间以及可用的总空间。

重要说明：如果您是 LDAP 组成员，除非是在用户级别设置的，否则将显示该组的磁盘定额。

要查看任何给定 iFolder 的 iFolder 定额，请点击 iFolder 链接以打开“iFolder 浏览”页。您可以查看“浏览”页左侧“iFolder 定额”下面列出的当前 iFolder 的已用和可用空间。
