创建新 iFolder

执行下列步骤以创建新 iFolder：

  1. 单击主页中的“新建 iFolder”链接。

  2. 在打开的新页面中，指定下列信息：

    3. 名称
    为 iFolder 指定一个名称，如 myfolder。

    注意：不要在 iFolder 的名称或其中的文件夹和文件的名称中使用以下无效字符：

    \/:*?"<>|;

    重要说明：当您尝试创建其名称超过 iFolder 服务器文件系统支持的文件限制的 iFolder 时，将引发错误讯息，指明该 iFolder 的名称太长，无法创建。


    说明
    iFolder 的简短说明。

    类型
    选择要创建的 iFolder 类型。
    Encrypted
    选择此选项可加密 iFolder 及其包含的文件。

    重要说明：仅当 Admin 将加密策略设为“启用”时才可启用此选项。

    常规
    选择“共享”可将 iFolder 与其他 iFolder 帐户共享。

    重要说明：共享的 iFolder 不能加密，加密的 iFolder 也不能共享。

  3. 从下拉列表中选择“恢复代理”。

    注意：如果 iFolder 管理员配置 iFolder 服务器时不选择“启用恢复代理”选项，则此选项将不可用。

    恢复代理可以是系统、组织或独立于 iFolder 系统（可储存恢复机密通行口令的密钥）的个人。

    重要说明：只有设置了恢复代理，才可在丢失通行口令的情况下恢复加密数据。

  4. 在“输入通行口令”字段中键入通行口令。

  5. 再次输入通行口令以进行确认。

  6. 单击“创建”以完成创建新 iFolder。

    单击“取消”可取消此操作。
