登录到万维网 Admin

登录到 Novell iFolder� 3.7 万维网 Admin 可管理 iFolder 和用户。系统会根据 LDAP 目录服务对 iFolder Admin 用户名和口令进行鉴定。

您还可以指定要使用哪种受支持的语言。该语言通常与 iFolder Enterprise Server 和本地计算机的语言代码相同。请务必对浏览器的语言设置进行配置，使其支持所需的语言。

  1. 指定要访问的 iFolder 帐户的用户名和口令。

  2. 使用“语言”下拉列表指定用于访问文件的语言。

  3. 单击“登录”。
