管理 iFolder

使用此界面可管理给定 iFolder 域上某一 iFolder 用户帐户的选定 iFolder 的所有权、策略和成员资格。搜索工具可帮助您查找要管理的帐户。

注意：当您以二级管理员身份登录时，只能查看您被指定为其二级管理员的用户所拥有的 iFolder。此外，您可以管理 iFolder，但仅取决于提供给您的用于管理 iFolder 的权限。

搜索 iFolder

iFolder 类型

指示该 iFolder 是常规的。
指示该 iFolder 是共享的。
指示该 iFolder 是加密的。

单击“全部”可查看当前 iFolder 域中的所有 iFolder。单击“孤立”可查看所有孤立的 iFolder。

检索服务器上的 iFolder 列表

使用下面的一种或多种搜索方法，然后单击“搜索”：

  • 选择“等于”作为过滤器标准，然后在“搜索 iFolder”字段中键入要查找的 iFolder 的名称。

  • 为 iFolder 的名称选择一个过滤器标准（“开始于”、“结束于”、“包含”、“等于”），然后在“搜索 iFolder”字段中键入一个或多个字母。

  • 在“搜索 iFolder”字段中键入一个星号 (*) 可返回系统中所有 iFolder 的列表。

  • “搜索 iFolder”字段保留为空时可返回系统中所有 iFolder 的列表。

请勿单击该页上的任何地方，直至该页完全刷新完毕，然后您可以浏览、排序或管理“搜索结果”报告中所列的 iFolder。

浏览搜索结果中的 iFolder

上下滚动可浏览搜索结果并查找要管理的 iFolder。iFolder 的名称和拥有者的组合可帮助您标识所需的 iFolder。

启用、禁用和删除 iFolder


  1. 查找要管理的 iFolder，然后选中该 iFolder 旁边的复选框。

  2. 选择一个要在该 iFolder 上执行的操作：

    • 单击“删除”可删除该 iFolder。

    • 单击“启用”可启用该 iFolder。

      此操作可使用户登录并同步 iFolder。

    • 单击“禁用”可禁用该 iFolder。

      如果用户在您进行此更改时已经登录，则在该用户注销之前，该用户的会话将继续。此策略将在用户下次试图登录该帐户时生效。要使锁定立即生效，则必须重启动 iFolder 服务器的 Apache 服务，该服务器将中断所有的活动会话，包括用户的会话。

注意： 默认情况下，已启用所有 iFolder。

创建新的 iFolder


  1. 在“iFolder 用户”页上，单击“新建”。

    iFolder 创建页打开，显示“iFolder”页。

  2. 指定以下信息，然后单击“创建”。

名称
为 iFolder 指派一个名称，如 ifolder 1。

说明
有关 iFolder 的简短说明。

访问“iFolder 细节”页


  1. 使用搜索工具查找要管理的 iFolder。

  2. 点击 iFolder 的链接以打开“iFolder 细节”页，显示“iFolder”选项卡。

    “iFolder 细节”页将显示当前 iFolder 的 iFolder 细节、拥有或共享的成员列表以及策略设置。
