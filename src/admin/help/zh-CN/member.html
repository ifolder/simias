管理万维网 Admin 控制台

使用此万维网 Admin 界面，可管理 iFolder 用户、LDAP 组、iFolder 系统、服务器、iFolder 以及 iFolder 统计信息报告。

“用户”选项卡
“iFolder”选项卡
“系统”选项卡
“服务器”选项卡
“报告”选项卡

“用户”选项卡

  1. 使用万维网 Admin 身份凭证登录到万维网 Admin。

    注意：术语 iFolder 用户既指单个用户，也指 LDAP 组。

    “用户”页显示该用户的类型（Admin 用户或用户）、用户名、用户全名（如果可用）、为其配置用户的服务器以及用户状态（启用或禁用）。

  2. 使用搜索功能可找到您要管理其 iFolder 帐户的用户。

  3. 单击该用户的名称链接以打开“用户细节”页。

    “用户细节”页打开，显示“用户”选项卡，该选项卡显示该特定用户帐户的用户细节、所拥有的 iFolder 以及共享和策略设置。

“iFolder”选项卡


  1. 在万维网 Admin 控制台上，单击“iFolder”选项卡。

    “iFolder”页显示 iFolder 类型（Admin 用户或一般用户）、iFolder 名称、iFolder 拥有者、成员、iFolder 的上次修改日期以及 iFolder 大小。

  2. 使用搜索工具通过搜索标准查找要管理的 iFolder。

  3. 点击 iFolder 的链接以打开“iFolder 细节”页，显示“iFolder”选项卡。

    “iFolder 细节”页显示当前 iFolder 的 iFolder 细节、拥有或共享 iFolder 的成员列表以及策略设置。

“系统”选项卡


  1. 在万维网 Admin 控制台上，单击“系统”选项卡。

    “系统”页显示系统设置和 iFolder 管理员列表。使用“系统”页，可管理整个系统的策略设置。

  2. 查找要管理的 iFolder 管理员。可以添加或删除 iFolder 管理员。

    还可以管理 Admin 用户的策略设置。

  3. 点击 Admin 用户的名称链接以打开“用户细节”页。

    “用户细节”页打开，显示“用户”选项卡，该选项卡显示该特定用户帐户的用户细节、所拥有的 iFolder 以及共享和策略设置。

“服务器”选项卡


  1. 在万维网 Admin 控制台上，单击“服务器”选项卡。

    “服务器”页显示所选服务器的服务器类型（主或从属）、服务器名称、公用 IP 地址、私用 IP 地址和状态。

  2. 使用搜索工具查找要管理的服务器。

  3. 单击该服务器名称链接以打开“服务器细节”页。

    “服务器细节”页显示服务器细节、服务器类型（主或从属）、DNS 名称、服务器状态、服务器日志、服务器报告、日志级别、LDAP 细节和 iFolder 数据储存区。

“报告”选项卡


  1. 在万维网 Admin 控制台上，单击“报告”。

  2. 先根据所需的频率和时间日程表配置报告，然后根据需要生成输出。

    可以将生成的报告保存到在“报告”页上指定的目录中。
