查看服务器细节

您可查看并编辑下列信息：

参数

描述

名称（仅供查看）

指派给服务器的名称。

类型

服务器类型（主服务器或从属服务器）。单击“设置为主服务器”以将此服务器指定为主服务器。

DNS 名称

iFolder Enterprise Server 的 DNS 名称。

例如：192.168.1.1 或 svr1.domain.com。

公用 URL

与该 iFolder 服务器对应的公共 IP 地址。要更改 IP 地址，请编辑给定的地址并单击“保存”以保存所做更改。

私用 URL

对应于 iFolder 服务器的私用 IP 地址。它允许 iFolder 域范围内的服务器之间进行通讯。私用 URL 和公共 URL 可以相同。要更改给定的 IP 地址，键入新地址并单击“保存”以保存所做更改。

主 URL（仅对从属服务器显示）

与 iFolder 服务器对应的 IP 地址。使用此地址，从属服务器可与 iFolder 域中的主服务器通讯。

要更改 IP 地址，请编辑给定的地址并单击“保存”以保存所做更改。

LDAP 细节

您可以编辑以下有关 LDAP 的信息。单击“保存”可修改项。单击“取消”以取消修改。

<

参数

描述

连接时间

（仅供查看）

显示第一次同步的日期和时间。

状态

报告当前的 LDAP 同步引擎状态。

循环数

（仅供查看）

显示已完成的身份同步循环的总数。

身份同步

以所选间隔从 LDAP 信息更新所选 iFolder 域中的 iFolder 用户。

在“身份同步”字段中指定时间间隔（以分钟计），单击“立即同步”可立即启动 iFolder 用户与 LDAP 用户的同步。

删除成员宽限间隔

指定从 LDAP 中删除特定用户后 iFolder 从服务器中删除该用户信息的时间间隔。如果用户被意外删除，此功能可帮助您恢复数据。


您可以编辑以下有关 LDAP 的信息。单击“编辑”可打开新页，您可在其中修改输入内容。

参数

描述

LDAP 服务器

显示 LDAP 服务器的地址。

LDAP SSL

显示是否启用了 LDAP SSL。还允许您启用或禁用 LDAP SSL 连接。

代理用户

显示代理用户的判别名 (DN)。例如，cn=SimiasProxy,o=novell

iFolder 代理用户是用于访问 LDAP 服务器的身份，以便检索指定树枝或组中的用户列表或 iFolder LDAP 设置中定义的用户列表。iFolder 代理用户是在 iFolder 安装过程中创建的。此身份必须具有对 LDAP 目录的读权限。

代理用户口令

 

此口令用于在 iFolder 与 LDAP 服务器同步用户时向 LDAP 服务器鉴定 iFolder 代理用户。

LDAP 环境

LDAP 环境列出所有的 LDAP 环境。iFolder 仅从列出的 LDAP 环境中搜索用户。


数据储存区

数据储存区表示可以跨越给定服务器中的多个卷（装入点）的 iFolder 储存区域。默认情况下，每个 iFolder 服务器都有一个无法被禁用的默认储存区。使用万维网界面可以添加和配置多个数据储存区，以在其中平衡 iFolder 数据的负载。用户上载 iFolder 时，它检查数据储存区的最大可用空间，并将 iFolder 数据储存到特定数据储存区，从而平衡负载。您可以根据需要添加任意多个数据储存区。拥有多个数据储存区可在大型部署中调整数据储存区容量以满足企业级需求。

您可以查看以下数据储存区信息：

参数

描述

名称

显示为数据储存区指定的唯一名称。

完整路径

显示装入卷的数据储存区的路径。此路径是您在使用万维网界面添加数据储存区时指定的数据路径。

可用空间

显示卷中的可用空间。

已启用

显示给定的数据储存区是否已启用。无法禁用默认的数据储存区。

启用或禁用数据储存区

选择要禁用或启用的数据储存区，然后分别单击“禁用”或“启用”。用户上载 iFolder 时，检查用于储存 iFolder 数据的最大可用空间时始终跳过被禁用的数据储存区。
