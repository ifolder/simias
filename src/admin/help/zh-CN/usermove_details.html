用户重新配置细节

使用此界面可以查看用户配置或重新配置细节，以及用户所拥有的 iFolder 细节。

“用户配置细节”页显示选定用户的 iFolder 帐户的以下用户细节。

参数

说明

用户名

指派给用户帐户的用户名，例如 jsmith 或 john.smith@example.com。

全名

用户帐户的名和姓。

LDAP 环境

LDAP 树环境用于为 iFolder 域配置用户。例如：cn= test, o=novell。

新主服务器

显示要为用户配置的新服务器。

已完成

以百分比形式显示重新配置状态。

重新配置状态

显示以下重新配置状态中的任意状态。

  • 正在初始化
  • 已初始化
  • 正在移动 iFolder
  • 正在重新设置主服务器
  • 正在完成

iFolder 细节

“用户配置细节”页显示选定用户的 iFolder 帐户的以下 iFolder 细节。

参数

说明

类型

常规 iFolder

加密 iFolder

共享 iFolder

名称

指派给 iFolder 的名称。

拥有者

所选 iFolder 的拥有者的用户名。拥有者可以是单个用户，也可以是 LDAP 组。如果 iFolder 的拥有者是 LDAP 组，则 LDAP 组的所有成员都将具有相同的访问权限。对于孤立的 iFolder，在可以重指派或删除 iFolder 之前，iFolder Admin 用户是其拥有者。

单击该拥有者的用户名链接可查看其用户帐户的细节。

iFolder 拥有者具有对该 iFolder 的“完全控制”权限。该拥有者可管理用户的成员资格和访问权限，并且可去除任何成员的“完全控制”权限。

大小

iFolder 大小 (MB)。

状态

显示以下 iFolder 移动状态中的任意状态。

  • 正在初始化
  • 正在转移
  • 已完成
