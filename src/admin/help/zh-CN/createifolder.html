创建新的 iFolder

使用此界面可创建新 iFolder、选择拥有者以及向 iFolder 添加其他成员。

创建新 iFolder
选择新 iFolder 的拥有者
选择新 iFolder 的其他成员

创建新 iFolder

要创建新 iFolder，请执行下面给出的步骤：

  1. 在“iFolder 用户”页上，单击“新建”。

    iFolder 创建页打开，显示“iFolder”页。

  2. 指定以下信息，然后单击“创建”。

    名称
    为 iFolder 指派一个名称，如 ifolder 1。

    说明
    有关 iFolder 的简短说明。

  3. 单击“下一步”可选择 iFolder 的拥有者。

选择新 iFolder 的拥有者


  1. 在“拥有者选择”页面上，使用搜索工具查找要为其指派所有权的所需用户。

  2. 单击“下一步”可选择新 iFolder 的其他成员。

    3. 单击“后退”可进行任意更改。

    单击“取消”可取消所做的更改。

  3. 单击“下一步”可选择 iFolder 的其他成员。

选择新 iFolder 的其他成员。


  1. 在“其他成员选择”页面上，使用搜索工具查找所需的用户。

  2. 选择搜索结果中的用户。

    3. 单击“后退”可进行任意更改。

  3. 单击“创建”可完成 iFolder 的创建。
