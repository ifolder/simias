管理用户帐户

使用此界面在所选的 iFolder 域中管理 iFolder 用户。搜索工具可帮助您查找您要管理的 Novell iFolder ® 3.7 或更高版本的用户帐户。

注意：术语 iFolder 用户 既指单个用户，也指 LDAP 组。

浏览用户

  1. 从用户页上，选择一个名称标准（名字、主服务器、姓氏、用户名）。

  2. 选择一个过滤器标准（“包含”、“开始于”、“结束于”或“等于”）。

  3. 使用下面的一种或多种搜索方法，然后单击“搜索”：

    • 在“搜索用户”字段中键入用户或 LDAP 组的名称。

    • 在“搜索用户”字段中键入一个或多个字母。

    • 在“搜索用户”字段中键入星号 (*) 将返回系统中的所有用户和 LDAP 组的列表。

    • 保留“搜索用户”字段为空将返回系统中的所有用户和 LDAP 组的列表。

    请勿单击该页上的任何地方，直至该页完全刷新完毕，然后您可以浏览、排序或管理“搜索结果”报告中所列的用户。

    注意：当您以二级管理员身份登录时，只能查看您被指定为其二级管理员的用户。此外，您还可以根据提供给您的权限管理用户。

在搜索结果中查找用户

上下滚动以浏览搜索结果并查找您要管理的用户。用户名、名、姓的组合可帮助您标识要找的用户。

类型

指示配置的用户

指示未配置的用户

指示用户是否具有 iFolder Admin 权限

名称
指派给用户帐户的用户名，例如 jsmith 或 john.smith@example.com。单击用户名链接以管理用户策略和该用户的 iFolder。

全名
用户帐户的名和姓。

主服务器
为其配置用户的 iFolder 服务器。

启用
指定服务器的状态。

启用或禁用一个 iFolder 用户

请按照下面给出的步骤执行操作：

  1. 找到您要管理的 iFolder 用户，然后选中该用户旁边的复选框。

    • 单击“启用”可启用该用户。

      这将允许用户登录并同步 iFolder。

    • 单击“禁用”可禁用该用户。

      如果用户在您进行此更改时已经登录，则在该用户注销之前，该用户的会话将继续。此策略将在用户下次试图登录该帐户时生效。要使锁定立即生效，则必须重启动 iFolder 服务器的 Apache 服务，该服务器将中断所有的活动会话，包括用户的会话。

配置/重新配置 iFolder 用户

使用此界面可手动为当前 iFolder 域中可用的不同服务器配置或重新配置 iFolder 用户。

若要配置/重新配置 iFolder 用户，请执行下列任一操作：

使用下拉列表配置

  1. 查找您要配置的 iFolder 用户。

  2. 从用于每个用户的“主目录”下拉列表中选择 iFolder 服务器。

  3. 单击“保存”。

使用配置按钮配置

  1. 找到要管理的 iFolder 用户。

  2. 选中每个用户旁边的复选框。

  3. 单击“配置”打开一个新页，可在该页上为域中的 iFolder 服务器配置/重新配置选定用户。

注意：当用户首次登录时，如果未在用户级别为该用户设置一个主服务器，则会自动将指派给该用户所属 LDAP 组的服务器分配给该用户。

创建新的 iFolder 用户

注意：您仅可以创建非 LDAP 用户。仅当您的 iFolder 服务器未与 LDAP 服务器连接时才可以创建这些用户。

  1. 在“iFolder 用户”页上，单击“新建”。

    “iFolder 用户”创建页会显示“用户”选项卡。

  2. 指定以下信息，然后单击“创建”。

用户名
为用户帐户指派一个用户名，例如 jsmith 或 john.smith@example.com。

名
用户的名。

姓
用户的姓。

全名
用户帐户的名和姓。

口令
为用户帐户指派口令。

重键入口令
重键入口令以进行校验。

访问“用户细节”页

请按照下面给出的步骤执行操作：

  1. 在“iFolder 用户”页上，使用搜索工具查找要管理其 iFolder 帐户的用户。

  2. 单击用户的名称链接以打开“用户细节”页，显示“用户”选项卡。
