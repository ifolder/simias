Adicionar Armazenamento de Dados

Para adicionar um novo Armazenamento de Dados,

  1. Especifique as seguintes informações:

    2. Nome
    Designe um nome exclusivo para o Data Store, como ifolder-store.

    Caminho
    Digite o caminho no qual o novo volume está montado.
    Caso seja um volume remoto (CIFS, NFS, AFP), certifique-se de que ele seja montado a cada reinicialização para que o funcionamento e o equilíbrio de carga sejam adequados.
    É necessário selecionar as permissões do caminho especificado e mudar a propriedade e grupo para apacheuser (wwwrun) e apachegroup (www) respectivamente. A menos que tenha definido a permissão para o diretório no qual o volume será montado, você não poderá criar nem sincronizar pastas do iFolder neste volume.
