Iniciando o Web Admin

O console do Web Admin permite que você gerencie servidores, configurações de política, usuários e pastas do iFolder em um determinado domínio do iFolder 3.7. Você pode habilitar usuários para o domínio do iFolder em usuários LDAP e gerenciar servidores iFolder que participam do domínio. Com o Web Admin, você também pode assegurar o uso obrigatório da criptografia em usuários. O console do Web Admin terá a capacidade de produzir relatórios aprimorados que incluem o status geral do servidor, os relatórios da conta do iFolder e as informações sobre a conta do usuário.
