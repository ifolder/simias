Creación de una carpeta iFolder nueva

Para crear una carpeta iFolder nueva, siga el procedimiento que se indica a continuación:

  1. En la página Usuarios de iFolder, haga clic en Nuevo.

    Se abrirá la página de creación de carpetas iFolder en la página Carpetas iFolder.

  2. Especifique la siguiente información y, a continuación, haga clic en Crear.

    Nombre
    Asigne un nombre a la carpeta iFolder como, por ejemplo, ifolder 1.

    Descripción
    Descripción breve acerca de la carpeta iFolder.

  3. Haga clic en Siguiente para seleccionar un propietario para la carpeta iFolder.

Selección de un propietario para la carpeta iFolder nueva


  1. En la página de selección de propietario, utilice la herramienta de búsqueda para localizar al usuario deseado a quien desea asignar la propiedad.

  2. Haga clic en Siguiente para seleccionar a otros miembros para la carpeta iFolder nueva.

    3. Haga clic en Atrás para realizar los cambios que considere pertinentes.

    Haga clic en Cancelar para cancelar los cambios realizados.

  3. Haga clic en Siguiente para seleccionar a otros miembros para la carpeta iFolder.

Selección de otros miembros para la carpeta iFolder nueva


  1. En la página de selección de otros miembros, utilice la herramienta de búsqueda para localizar a los usuarios deseados.

  2. Seleccione a los usuarios de entre los resultados de la búsqueda.

    3. Haga clic en Atrás para realizar los cambios que considere pertinentes.

  3. Haga clic en Crear para finalizar el proceso de creación de la carpeta iFolder.
