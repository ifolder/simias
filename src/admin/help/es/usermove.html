Estado de la nueva provisión del sistema iFolder

Utilice esta interfaz para ver el estado de la nueva provisión del dominio iFolder actual.

Esta página muestra la siguiente información:

Parámetro

Descripción

Tipo

Muestra si los usuarios son usuarios con provisión o usuarios sin provisión.

Usuario

Muestra el nombre de usuario asignado a cada cuenta de usuario como, por ejemplo, jsmith o john.smith@ejemplo.com.

Ubicación principal actual

Muestra el servidor principal asignado a un usuario con provisión.

Nueva ubicación principal

Muestra el nuevo servidor que debe proveer al usuario.

Completada

Muestra el estado de nueva provisión como porcentaje.

Estado de la nueva provisión

Muestra uno de los siguientes estados de nueva provisión:

  • Iniciándose
  • Iniciada
  • Moviendo la carpeta iFolder
  • Restaurando la ubicación principal
  • Finalizando
