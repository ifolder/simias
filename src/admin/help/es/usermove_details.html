Información de nueva provisión del usuario

Utilice esta interfaz para ver la información de provisión o nueva provisión del usuario y la información de las carpetas iFolder que posee el usuario.

La página Información de provisión del usuario muestra la siguiente información de usuario para la cuenta de iFolder del usuario seleccionado.

Parámetro

Descripción

Usuario

Nombre de usuario asignado a la cuenta de usuario como, por ejemplo, jsmith o john.smith@ejemplo.com.

Nombre completo

Nombre y apellido de la cuenta de usuario.

Contexto LDAP

El contexto del árbol LDAP permite asignar usuarios al dominio iFolder. Por ejemplo, cn=test,o=novell.

Nueva ubicación principal

Muestra el nuevo servidor que debe proveer al usuario.

Completada

Muestra el estado de nueva provisión como porcentaje.

Estado de la nueva provisión

Muestra uno de los siguientes estados de nueva provisión.

  • Iniciándose
  • Iniciada
  • Moviendo la carpeta iFolder
  • Restaurando la ubicación principal
  • Finalizando

Información de una carpeta iFolder

La página Información de provisión del usuario muestra la siguiente información de iFolder para la cuenta iFolder del usuario seleccionado.

Parámetro

Descripción

Tipo

Carpeta iFolder normal

Carpeta iFolder cifrada

Carpeta iFolder compartida

Nombre

Nombre asignado a la carpeta iFolder.

Propietario

Nombre de usuario del propietario de la carpeta iFolder seleccionada. El propietario puede ser un usuario individual o un grupo LDAP. Si un grupo LDAP es el propietario de la carpeta iFolder, todos los miembros de dicho grupo tendrán los mismos derechos de acceso. Para las carpetas iFolder huérfanas, el usuario Admin de iFolder es el propietario hasta que la carpeta iFolder pueda reasignarse o suprimirse.

Haga clic en el enlace del nombre de usuario del propietario para ver la información de su cuenta de usuario.

El propietario de la carpeta iFolder dispone del derecho de control total sobre la carpeta iFolder. El propietario gestiona la pertenencia y los derechos de acceso de los usuarios, y puede eliminar el derecho de Control total de cualquiera de los miembros.

Tamaño

Tamaño de la carpeta iFolder en MB.

Estado

Muestra los estados de movimiento de iFolder.

  • Iniciándose
  • Transfiriendo
  • Completada
