Gestión de una cuenta de usuario

Esta interfaz le permite gestionar los usuarios de iFolder del dominio iFolder seleccionado. La herramienta de búsqueda le ayudará a localizar la cuenta de usuario de Novell iFolder® 3.7 o posterior que desea gestionar.

NOTA: el término usuarios de iFolder hace referencia tanto a usuarios individuales como a grupos LDAP.

Examinación de los usuarios

  1. En la página Usuarios, seleccione un criterio de nombre (Nombre, Servidor principal, Apellido, Nombre de usuario).

  2. Seleccione un criterio de filtrado (Contiene, Empieza por, Acaba por, Igual a).

  3. Utilice uno o varios de los siguientes métodos de búsqueda y, a continuación, haga clic en Buscar:

    • Escriba el nombre del usuario o el grupo LDAP en el campo Buscar usuarios.

    • Escriba una o varias letras en el campo Buscar usuarios.

    • Escriba un asterisco (*) en el campo Buscar usuarios para obtener una lista de todos los usuarios y grupos LDAP del sistema.

    • Si lo prefiere, deje el campo en blanco para obtener una lista de todos los usuarios y grupos LDAP del sistema.

    No haga clic en ningún lugar de la página hasta que ésta se haya actualizado del todo. A continuación, podrá examinar, clasificar o gestionar los usuarios del informe de resultados de búsqueda.

    Nota: cuando inicia sesión como administrador secundario, puede ver sólo aquellos usuarios para los que está designado como administrador secundario. Además, puede gestionar los usuarios dependiendo de los derechos de los que disponga.

Localización de los usuarios en los resultados de búsqueda

Desplácese hacia arriba o hacia abajo para examinar los resultados de búsqueda y localizar al usuario que desea gestionar. La combinación de nombre de usuario y nombre y apellido le permitirá identificar al usuario deseado.

Habilitación o inhabilitación de un usuario de iFolder

Siga los pasos que se indican a continuación:

  1. Localice al usuario de iFolder que desea gestionar y, a continuación, seleccione la casilla de verificación situada junto a éste.

    • Haga clic en Habilitar para habilitar al usuario.

      Esta acción permitirá al usuario entrar en las carpetas iFolder y sincronizarlas.

    • Haga clic en Inhabilitar para inhabilitar al usuario.

      Si el usuario está conectado cuando se efectúa este cambio, la sesión del usuario continuará hasta que éste salga. La directiva surtirá efecto la próxima vez que el usuario intente entrar a la cuenta. Para que el bloqueo se aplique de forma inmediata, deberá reiniciar los servicios de Apache correspondientes al servidor iFolder, el cual desconectará todas las sesiones activas, incluida la del usuario.

Provisión/Nueva provisión de usuarios de iFolder

Utilice esta interfaz para proveer o volver a proveer manualmente a los usuarios de iFolder a los diferentes servidores iFolder disponibles en el dominio de iFolder.

Para proveer o volver a proveer a los usuarios de iFolder, realice una de las tareas siguientes:

Asignación mediante la lista desplegable

  1. Localice a los usuarios de iFolder que desea asignar.

  2. Seleccione el servidor iFolder en la lista desplegable que aparece junto a cada usuario.

  3. Haga clic en Guardar.

Asignación mediante el botón Provisión de servicios

  1. Localice a los usuarios de iFolder que desea gestionar.

  2. Seleccione la casilla de verificación situada junto a cada usuario.

  3. Haga clic en Provisión de servicios para abrir una nueva página en la que poder proveer o volver a proveer a los usuarios seleccionados a un servidor iFolder del dominio.

NOTA: cuando un usuario entra por primera vez y si no se ha definido un servidor principal en el nivel de usuario, el servidor asignado al grupo LDAP al que pertenece el usuario se asigna automáticamente al usuario.

Creación de un usuario de iFolder nuevo

NOTA: sólo es posible crear usuarios que no sean LDAP. Sólo podrá crearlos cuando el servidor iFolder no esté conectado al servidor LDAP.

  1. En la página Usuarios de iFolder, haga clic en Nuevo.

    Se abrirá la página de creación de usuarios de iFolder en la pestaña Usuarios.

  2. Especifique la siguiente información y, a continuación, haga clic en Crear.

Usuario
Asigne un nombre de usuario a la cuenta de usuario como, por ejemplo, jsmith o john.smith@ejemplo.com.

Nombre
Nombre del usuario.

Apellido
Apellido del usuario.

Nombre completo
Nombre y apellido de la cuenta de usuario.

Contraseña
Asigne una contraseña a la cuenta de usuario.

Vuelva a escribir la contraseña
Vuelva a escribir la contraseña para confirmarla.

Acceso a la página Información de usuario

Siga el procedimiento que se indica a continuación:

  1. En la página Usuario de iFolder, utilice la herramienta de búsqueda para localizar al usuario cuya cuenta de iFolder desea gestionar.

  2. Haga clic en el enlace del nombre del usuario para abrir la página Información de usuario en la pestaña Usuarios.
