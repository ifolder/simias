Inicio de Web Admin

La consola de Web Admin permite gestionar servidores, valores de configuración de directivas, usuarios de iFolder y carpetas iFolder en un dominio concreto de iFolder 3.7 o posterior. Es posible habilitar a usuarios para el dominio iFolder a partir de usuarios LDAP y gestionar los servidores iFolder que participen en el dominio iFolder. Web Admin también le permite aplicar el cifrado a los usuarios. La consola de Web Admin ofrece la opción de generar informes mejorados que incluyan el estado general del servidor, los informes de cuenta de iFolder e información sobre la cuenta del usuario.

NOTA: No es posible gestionar servidores Novell iFolder 2.1x con el módulo auxiliar (plug-in) de Novell iFolder 3.7 y módulos auxiliares posteriores en iManager.

Para acceder a la página de gestión de iFolder:

  1. En iManager, expanda la función de iFolder

  2. Haga clic en Lanzar Web Admin para abrir la página de selección de servidor de iFolder.

  3. Especifique el nombre DNS o la dirección IP del servidor empresarial iFolder.

  4. Realice uno de los siguientes pasos:

    • Si ha entrado en iManager con el mismo nombre de usuario que el usuario Admin de iFolder de Web Admin, seleccione Autenticación mediante las credenciales actuales de iManager.

    • Si, por el contrario, ha entrado en iManager con un nombre de usuario distinto del usuario Admin de iFolder de Web Admin, deje la casilla de verificación Autenticación mediante las credenciales actuales de iManager sin seleccionar. En la página de entrada de Web Admin, especifique el nombre de usuario y la contraseña del administrador de iFolder.

  5. Haga clic en Aceptar.

En función de la selección que haya realizado, será conducido a la página de usuarios administradores de Web Admin.
