Inicio de Web Admin

La consola de Web Admin permite gestionar servidores, valores de configuración de directivas, usuarios de iFolder y carpetas iFolder en un dominio concreto de iFolder 3.7 o posterior. Es posible habilitar a usuarios para el dominio iFolder a partir de usuarios LDAP y gestionar los servidores iFolder que participen en el dominio iFolder. Web Admin también le permite aplicar el cifrado a los usuarios. La consola de Web Admin ofrece la opción de generar informes mejorados que incluyan el estado general del servidor, los informes de cuenta de iFolder e información sobre la cuenta del usuario.
