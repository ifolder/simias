Gestión de la consola de Web Admin

Utilice esta interfaz de Web Admin para gestionar usuarios, grupos LDAP, el sistema iFolder, los servidores iFolder y el informe de estadísticas de iFolder.

Pestaña Usuarios

  1. Entre a Web Admin mediante las credenciales correspondientes.

    NOTA:el término usuarios de iFolder hace referencia tanto a usarios individuales como a grupos LDAP.

    La página Usuarios muestra el tipo de usuario (usuario Admin o usuario), el nombre de usuario, el nombre completo del usuario (si está disponible), el servidor al que está asignado el usuario y el estado del usuario (Habilitado o Inhabilitado).

  2. Utilice la herramienta de búsqueda para localizar al usuario cuya cuenta de iFolder desea gestionar.

  3. Haga clic en el enlace del nombre del usuario para abrir la página Información de usuario.

    La página Información de usuario se abrirá en la pestaña Usuarios, la cual muestra la información del usuario, las carpetas iFolder en propiedad y compartidas, y la configuración de directivas para la cuenta del usuario en cuestión.

Pestaña Carpetas iFolder


  1. En la consola de Web Admin, haga clic en la pestaña Carpetas iFolder.

    La página Carpetas iFolder muestra el tipo de carpeta iFolder (usuario Admin o usuario), el nombre de la carpeta iFolder, el propietario de la carpeta iFolder, los miembros, la fecha en la que la carpeta iFolder se ha modificado por última vez y el tamaño de la carpeta iFolder.

  2. Utilice la herramienta de búsqueda para localizar la carpeta iFolder que desea gestionar especificando un criterio de búsqueda.

  3. Haga clic en el enlace de la carpeta iFolder para abrir la página Información de iFolder en la pestaña iFolder.

    La página Información de iFolder mostrará la información de la carpeta iFolder, una lista de los miembros que la poseen o comparten, y la configuración de directivas de la carpeta iFolder seleccionada.

Pestaña Sistemas


  1. En la consola de Web Admin, haga clic en la pestaña Sistemas.

    La página Sistemas muestra la configuración del sistema y una lista de los administradores de iFolder. La página Sistema permite gestionar la configuración de directivas para todo el sistema.

  2. Localice al administrador de iFolder que desea gestionar. Puede añadir o suprimir un administrador de iFolder.

    También puede gestionar la configuración de directivas para el usuario Admin.

  3. Haga clic en el enlace del nombre del usuario Admin para abrir la página Información de usuario.

    La página Información de usuario se abrirá en la pestaña Usuarios, la cual muestra la información del usuario, las carpetas iFolder en propiedad y compartidas, y la configuración de directivas para la cuenta del usuario en cuestión.

Pestaña Servidores


  1. En la consola de Web Admin, haga clic en la pestaña Servidores.

    La página Servidores muestra el tipo de servidor (Principal o Esclavo), el nombre del servidor, la dirección IP pública, la dirección IP privada y el estado del servidor seleccionado.

  2. Utilice la herramienta de búsqueda para localizar el servidor que desea gestionar.

  3. Haga clic en el enlace del nombre del servidor para abrir la página Información del servidor.

    La página Información del servidor muestra la información del servidor, el tipo de servidor (Principal o Esclavo), el nombre DNS, el estado del servidor, los registros del servidor, los informes del servidor, el nivel de registro, la información de LDAP y el almacén de datos de iFolder.

Pestaña Informes


  1. En la consola de Web Admin, haga clic en Informes.

  2. Configure los informes en función de la frecuencia y la programación deseadas y, a continuación, genere el resultado de la forma deseada.

    Puede guardar el informe generado en el directorio especificado en la página Informes.
