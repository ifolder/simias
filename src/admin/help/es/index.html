Acceso a carpetas y archivos de las carpetas iFolder

Utilice esta interfaz para acceder a las carpetas y a los archivos de las carpetas iFolder disponibles en esta cuenta. Es posible visualizar una lista de las carpetas y los archivos de cada carpeta iFolder, así como cargar y descargar archivos, crear carpetas y suprimir carpetas y archivos. Para crear, suprimir o compartir carpetas iFolder, debe utilizarse el Cliente iFolder para equipos Linux*, Windows* o Macintosh*, según corresponda.

En esta ayuda se tratan los temas siguientes:

Entrada y salida
Navegación por las carpetas iFolder
Gestión de carpetas y archivos de una carpeta iFolder

Entrada y salida

Si se termina la conexión del sistema mientras trabaja con las carpetas iFolder, el acceso Web le devuelve a la página de entrada. Puede volver a entrar para seguir trabajando sin tener que cerrar el navegador.

Para salir haga clic en Salir, situado en la parte superior derecha y, a continuación, cierre el navegador. Si no se desconecta, la sesión permanece abierta hasta que se agota el tiempo de espera, lo que supone un riesgo de seguridad.

Navegación por las carpetas iFolder

En la página de carpetas iFolder aparecen todas las carpetas iFolder disponibles en la cuenta actual. Las carpetas iFolder disponibles constan de las suyas propias y de las que otros usuarios comparten con usted. Es posible acceder a las carpetas iFolder compartidas mediante el acceso Web, aunque no se haya instalado iFolder en un equipo local. La columna Propietario especifica el nombre del propietario de la carpeta iFolder para que resulte más sencillo ubicar la carpeta iFolder buscada.

Haga clic en el nombre de la carpeta iFolder para visualizar una lista de las carpetas y archivos, cargar y descargar archivos, crear carpetas y suprimir carpetas y archivos.

Gestión de carpetas y archivos de una carpeta iFolder

En una carpeta iFolder es posible realizar las siguientes tareas:

Ver una lista de las carpetas y archivos
La página iFolder enumera las carpetas y archivos en orden alfabético. Desplácese para encontrar el directorio o archivo buscado.

Desplazarse por un directorio
Haga clic en el nombre de la carpeta para abrir la carpeta y ver una lista con su contenido. Para volver a un directorio de un nivel más elevado, haga clic en el icono de carpeta Arriba situado en la barra de tareas.

Abrir o guardar un archivo
Haga clic en el nombre del archivo que desea abrir o guardar.

  • Haga clic en Abrir para ver el contenido del archivo. Es necesario que el equipo tenga una aplicación compatible para abrir el archivo. Si desea trabajar con el archivo, descárguelo, modifíquelo y, a continuación, cárguelo.

  • Haga clic en Guardar para guardar el archivo en el disco duro local o en una unidad de red asignada. Desplácese hasta la ubicación en la que desea colocar el archivo y, a continuación, haga clic en Guardar. El archivo se descargará en el equipo.

Crear una nueva carpeta
En el campo Nueva carpeta, especifique el nombre de la carpeta nueva y, a continuación, haga clic en Crear. Los nombres de las carpetas distinguen entre mayúsculas y minúsculas. Cuando se actualiza la página, la carpeta aparece en la lista ordenada por orden alfabético.

Suprimir carpetas o archivos
Seleccione el recuadro de verificación situado junto a las carpetas o archivos que desea suprimir y, a continuación, haga clic en Suprimir para confirmar esta opción.

Cargar un archivo
En el campo Cargar archivo, haga clic en Examinar, busque la carpeta local que desea cargar y, a continuación, haga clic en Abrir > Cargar. Cuando se actualiza la página, el archivo aparece en la lista ordenado por orden alfabético.
