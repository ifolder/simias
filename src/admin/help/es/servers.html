Gestión de servidores iFolder

Esta interfaz le permite gestionar los servidores iFolder para una configuración multiservidor. La herramienta de búsqueda le ayudará a localizar el servidor que desea gestionar.

Búsqueda de servidores iFolder

  1. Seleccione un criterio de filtrado (Contiene, Empieza por, Acaba por, Igual a).

  2. Utilice uno o varios de los siguientes métodos de búsqueda y, a continuación, haga clic en Buscar:

    • Escriba el nombre del servidor en el campo Buscar servidores.

    • Escriba una o varias letras en el campo Buscar servidores.

    • Escriba un asterisco (*) en el campo Buscar servidores para obtener una lista con todos los servidores del sistema.

    • Si lo prefiere, deje el campo en blanco para obtener una lista con todos los servidores del sistema.

    No haga clic en ningún lugar de la página hasta que ésta se haya actualizado del todo. A continuación, podrá examinar, clasificar o gestionar los servidores del informe de resultados de búsqueda.

    Desplácese hacia arriba o hacia abajo para examinar los resultados de búsqueda y localizar el servidor que desea gestionar.

Acceso a la página Información del servidor

Siga el procedimiento que se indica a continuación:

  1. En la página Servidor, utilice la herramienta de búsqueda para localizar el servidor por medio del criterio de búsqueda.

  2. Haga clic en el enlace del nombre del servidor para abrir la página Información del servidor en la página Servidores.

    Importante: la información que aparece en esta página es únicamente específica del servidor al que se encuentra conectado en estos momentos. Para realizar cualquier modificación, deberá entrar en la consola Web Admin del servidor correspondiente.
