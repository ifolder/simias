Entrada a Web Admin

Entre en Web Admin de Novell iFolderï¿½ 3.7 para gestionar carpetas iFolder y usuarios. El nombre de usuario y la contraseña del administrador de iFolder se autentican con los servicios de directorio LDAP.

También es posible especificar el idioma admitido que se utilizará. Por lo general, se trata del mismo idioma que el código de idioma del servidor empresarial de iFolder y del equipo local. Asegúrese de que ha configurado el ajuste del idioma del navegador para que admita el idioma deseado.

  1. Especifique el nombre de usuario y la contraseña de la cuenta de iFolder a la que desea acceder.

  2. Utilice la lista desplegable Idiomas para especificar el idioma que desea utilizar para acceder a los archivos.

  3. Haga clic en Entrada.
