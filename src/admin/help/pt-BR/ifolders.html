Gerenciando pastas do iFolder

Use esta interface para gerenciar a propriedade, as políticas e a associação de uma pasta do iFolder selecionada para a conta do usuário no domínio da pasta do iFolder fornecida. A ferramenta de pesquisa ajuda a localizar a conta que você deseja gerenciar.

Nota: Quando efetuar login como um administrador secundário, você só poderá ver os iFolders que pertencem a usuários para os quais foi designado como administrador secundário. Além disso, você só poderá gerenciar iFolders dependendo dos direitos que estiverem disponíveis.

Clique em Todas para ver todas as pastas do iFolder do domínio atual. Clique em Órfã para ver todas as pastas do iFolder órfãs.

Recuperando uma lista de pastas do iFolder no servidor.

Use um ou mais dos seguintes métodos de pesquisa e clique em Procurar:

  • Selecione É igual a como o critério de filtro e digite o nome do iFolder que você deseja localizar no campo Procurar iFolders.

  • Selecione um critério de filtro (Começa com, Termina com, Contém, É igual a) para o nome do iFolder e digite uma ou mais letras no campo Procurar iFolders.

  • Digite um asterisco (*) no campo Procurar iFolders para retornar uma lista de todos os iFolders no sistema.

  • Deixe o campo Procurar iFolders vazio para retornar uma lista de todos os iFolders no sistema.

Não clique na página até que ela seja completamente atualizada; em seguida, você poderá procurar, classificar ou gerenciar as pastas do iFolder listadas no relatório de Resultados da Pesquisa.

Procurando as pastas do iFolder nos resultados da pesquisa

Role para cima e para baixo para procurar os resultados da pesquisa e localizar o iFolder que deseja gerenciar. A combinação do proprietário e do nome do iFolder facilita a identificação do iFolder desejado.

Habilitando, desabilitando e excluindo um iFolder


  1. Localize o iFolder que você deseja gerenciar e marque a caixa de seleção ao lado do iFolder.

  2. Selecione uma ação para executar no iFolder:

    • Clique em Apagar para apagar o iFolder.

    • Clique em Habilitar para habilitar o iFolder.

      Isso permite que o usuário efetue login e sincronize iFolders.

    • Clique em Desabilitar para desabilitar o iFolder.

      Se o usuário estiver conectado quando você fizer a mudança, a sessão do usuário continuará até o usuário se desconectar. A política fará efeito na próxima vez em que o usuário tentar efetuar login na conta. Para que o bloqueio faça efeito imediatamente, é preciso reiniciar os serviços Apache do servidor iFolder, que desconecta todas as sessões ativas, inclusive a sessão do usuário.

NOTA:Por padrão, todas as pastas do iFolder ficam habilitadas.

Criando um novo iFolder


  1. Na página Usuários do iFolder, clique em Novo.

    A página de criação do iFolder é aberta na página iFolders.

  2. Especifique as informações a seguir e clique em Criar.

Nome
Designe um nome para o iFolder como ifolder 1.

Descrição
Uma breve descrição sobre o iFolder.

Acessando a página Detalhes dos iFolders


  1. Use a ferramenta de pesquisa para localizar o iFolder que você deseja.

  2. Clique no link do iFolder para abrir a página Detalhes do iFolder na guia iFolder.

    A página Detalhes do iFolder exibirá os detalhes do iFolder, uma lista de membros proprietários ou que compartilham o iFolder e configurações de política para o iFolder atual.
