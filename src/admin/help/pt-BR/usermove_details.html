Detalhes de Reprovisionamento do Usuário

Use esta interface para ver os detalhes de aprovisionamento ou reprovisionamento e do iFolder que o usuário possui.

A página Detalhes de Aprovisionamento do Usuário exibirá os seguintes detalhes do usuário para a conta do iFolder do usuário selecionado.

Parâmetro

Descrição

Nome do Usuário

O nome de usuário designado à conta do usuário, como jsilva ou jorge.silva@exemplo.com.

Nome completo

O nome e o sobrenome da conta do usuário.

Contexto do LDAP

O contexto de árvore do LDAP é usado para os usuários de aprovisionamento no domínio do iFolder. Por exemplo, cn=test, o=novell.

Novo Servidor Pessoal

Mostra o novo servidor para provisionar o usuário.

Concluído

Mostra o status de reprovisionamento em porcentagem.

Estado do Reprovisionamento

Mostra qualquer um dos estados de reprovisionamento a seguir:

  • Inicializando
  • Inicializado
  • Movendo o iFolder
  • Redefinindo a Home Page
  • Finalizando

Detalhes do iFolder

A página Detalhes de Aprovisionamento do Usuário exibirá os seguintes detalhes do iFolder para a conta do iFolder do usuário selecionado.

Parâmetro

Descrição

Tipo

iFolder Normal

iFolder Criptografado

iFolder Compartilhado

Nome

O nome designado ao iFolder.

Proprietário

O nome de usuário do proprietário do iFolder selecionado. O proprietário pode ser um usuário individual ou um grupo LDAP. Se um grupo LDAP for proprietário do iFolder, todos os membros do grupo terão os mesmos direitos de acesso. Para iFolders órfãos, o usuário do Admin do iFolder é considerado o proprietário até que o iFolder seja reatribuído ou apagado.

Clique no link do nome de usuário do proprietário para ver os detalhes da conta de usuário dele.

O proprietário do iFolder possui direito Controle Total para o iFolder. O proprietário gerencia os direitos de associação e acesso dos usuários e pode remover o direito Controle Total de qualquer membro.

Tamanho

Tamanho do iFolder em MB.

Status

Mostra qualquer um dos seguintes estados de movimento do iFolder.

  • Inicializando
  • Transferindo
  • Concluído
