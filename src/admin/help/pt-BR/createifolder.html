Criar um novo iFolder

Para criar um novo iFolder, siga o procedimento abaixo:

  1. Na página Usuários do iFolder, clique em Novo.

    A página de criação do iFolder é aberta na página iFolders.

  2. Especifique as informações a seguir e clique em Criar.

    Nome
    Designe um nome para o iFolder como ifolder 1.

    Descrição
    Uma breve descrição sobre o iFolder.

  3. Clique em Avançar para selecionar um proprietário para o iFolder.

Selecionando um proprietário para o novo iFolder


  1. Na página de seleção do proprietário, use a ferramenta de pesquisa para localizar o usuário ao qual você deseja designar a propriedade.

  2. Clique em Avançar para selecionar membros adicionais ao novo iFolder.

    3. Clique em Voltar para fazer mudanças.

    Clique em Cancelar para cancelar as mudanças feitas.

  3. Clique em Avançar para selecionar membros adicionais para o iFolder.

Selecionando membros adicionais para o novo iFolder


  1. Na página de seleção de membros adicionais, use a ferramenta de pesquisa para localizar os usuários desejados.

  2. Selecione os usuários no resultado da pesquisa.

    3. Clique em Voltar para fazer mudanças.

  3. Clique em Criar para concluir a criação do iFolder.
