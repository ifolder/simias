Status do reprovisionamento do sistema iFolder

Use esta interface para ver o status de reprovisionamento do domínio iFolder atual.

Essa página exibe os seguintes detalhes:

Parâmetro

Descrição

Tipo

Mostra se os usuários são provisionados ou não-provisionados.

Nome do Usuário

Mostra o nome de usuário atribuído à conta do usuário, como jsilva ou jorge.silva@exemplo.com.

Home Page Atual

Mostra o servidor pessoal atribuído a um usuário provisionado.

Novo Servidor Pessoal

Mostra o novo servidor para provisionar o usuário.

Concluído

Mostra o status de reprovisionamento em porcentagem.

Estado do Reprovisionamento

Mostra qualquer um dos estados de reprovisionamento a seguir:

  • Inicializando
  • Inicializado
  • Movendo o iFolder
  • Redefinindo a Home Page
  • Finalizando
