Relatórios

Use essa interface para habilitar relatórios e a geração de relatórios para iFolders e Diretórios. Ela gera relatórios com base na freqüência com que você seleciona.

  1. Selecione Habilitar relatórios para habilitar relatórios.

  2. Selecione uma das freqüências nas opções oferecidas (Diariamente, Semanalmente, Mensalmente).

  3. Selecione a hora em que deseja gerar o relatório.

  4. Selecione uma das opções fornecidas para ser o local no qual deseja gravar relatórios (iFolder do relatório, Diretório do relatório)

    Se você selecionar Diretório do relatório, o relatório gerado será gravado automaticamente no diretório do relatório no caminho de dados padrão.

    Por exemplo, <caminho de dados>/simias/report.

    Se você selecionar iFolder do Relatório, o relatório gerado será gravado no iFolder do Relatório.

    O iFolder do Relatório, chamado < nome do domínio>-<nome do servidor>-Relatórios é automaticamente criado para cada servidor do usuário admin.

  5. Selecione o formato para a geração do relatório.

  6. Clique em Gravar para gravar as configurações.

    7. Clique em Cancelar para cancelar as configurações.
