Gerenciando servidores iFolder

Use esta interface para gerenciar os servidores iFolder para uma configuração de vários servidores. A ferramenta de pesquisa ajuda você a localizar o servidor que você deseja gerenciar.

Pesquisando servidores iFolder

  1. Selecione um critério de filtro (Contém, Começar com, Terminar com, É igual a).

  2. Use um ou mais dos seguintes métodos de pesquisa e clique em Procurar:

    • Digite o nome do servidor no campo Procurar Servidores.

    • Digite uma ou mais letras no campo Procurar Servidores.

    • Digite um asterisco (*) no campo Procurar Servidores para retornar uma lista de todos os Servidores no sistema.

    • Deixe o campo Procurar Servidores em branco para retornar uma lista de todos os Servidores no sistema.

    Não clique na página até ela ser totalmente atualizada, em seguida, você poderá procurar, classificar ou gerenciar os servidores listados no relatório Procurar Resultados.

    Role para baixo e para cima para procurar resultados e localizar o Servidor que você deseja gerenciar.

Acessando a página Detalhes do Servidor

Siga o procedimento abaixo:

  1. Na página Servidor, use a ferramenta de pesquisa para localizar o servidor usando o critério de pesquisa.

  2. Clique no link do nome do Servidor para abrir a página Detalhes do Servidor na página Servidores.

    Importante:As informações que você pode ver nesta página são específicas somente para o servidor no qual você efetuou login atualmente. Para fazer qualquer modificação, você precisa efetuar login no respectivo console do Web Admin do servidor.
