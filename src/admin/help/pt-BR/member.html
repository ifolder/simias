Gerenciando o console do Web Admin

Use a interface do Web Admin para gerenciar os usuários do iFolder, o sistema iFolder, os servidores, as pastas do iFolder e o relatório de estatísticas do iFolder.

Guia Usuários

  1. Efetue login no Web Admin usando as credenciais do Web Admin.

    NOTA: O termo usuários do iFolder refere-se tanto aos usuários individuais quanto aos Grupos LDAP.

    A página Usuários exibe o tipo de usuário (usuário admin ou usuário), o nome de usuário, o nome completo do usuário (se disponível), o servidor ao qual o usuário está aprovisionado e o status do usuário (Habilitado ou Desabilitado).

  2. Use a funcionalidade de pesquisa para localizar o usuário cuja conta do iFolder você deseja gerenciar.

  3. Clique no link do nome do usuário para abrir a página Detalhes do Usuário.

    A página Detalhes do Usuário abre a guia Usuários, que exibe os detalhes do usuário, os iFolders compartilhados e de sua propriedade e as configurações de política para essa determinada conta de usuário.

Guia iFolders


  1. No console do Web Admin, clique na guia iFolders.

    A página iFolders exibe o tipo de iFolder (usuário admin ou usuário), o nome do iFolder, o proprietário do iFolder, os membros, a data em que o iFolder foi modificado pela última vez e o tamanho do iFolder.

  2. Use a ferramenta de pesquisa para localizar o iFolder que você deseja gerenciar usando o critério de pesquisa.

  3. Clique no link do iFolder para abrir a página Detalhes do iFolder na guia iFolder.

    A página Detalhes do iFolder exibe os detalhes do iFolder, a lista de membros que possuem ou compartilham os iFolders e as configurações de política do iFolder atual.

Guia Sistemas


  1. No console do Web Admin, clique na guia Sistemas.

    A página Sistemas exibe as configurações do sistema e a lista dos administradores do iFolder. Com a página Sistema você pode gerenciar as configurações de política para o sistema inteiro.

  2. Localize o Administrador do iFolder que você deseja gerenciar. Você pode adicionar ou apagar o Administrador do iFolder.

    Você também pode gerenciar as configurações de política do usuário Admin.

  3. Clique no link do nome do usuário Admin para abrir a página Detalhes do Usuário.

    A página Detalhes do Usuário abre a guia Usuários, que exibe os detalhes do usuário, os iFolders compartilhados e de sua propriedade e as configurações de política para essa determinada conta de usuário.

Guia Servidores


  1. No console do Web Admin, clique na guia Servidores.

    A página Servidores exibe o tipo de servidor (Master ou Escravo), o nome do servidor, o endereço IP público, o endereço IP particular e o status do servidor selecionado.

  2. Use a ferramenta de pesquisa para localizar o Servidor que você deseja gerenciar.

  3. Clique no link do nome do Servidor para abrir a página Detalhes do Servidor.

    A página Detalhes do Servidor exibe detalhes do servidor, o tipo de servidor (Master ou Escravo), o nome DNS, o status do servidor, os registros do servidor, os relatórios do servidor, o nível de registro, os detalhes do LDAP e o Data Store do iFolder.

Guia Relatórios


  1. No console do Web Admin, clique em Relatórios.

  2. Configure o relatório de acordo com a freqüência e a programação de horário que deseja e gere a saída conforme desejado.

    Você pode gravar o relatório gerado no diretório especificado na página Relatórios.
