Vendo detalhes do servidor

É possível exibir e editar as seguintes informações:

Parâmetro

Descrição

Nome (Ver Apenas)

O nome designado ao Servidor.

Tipo

O tipo de servidor (Master ou Escravo). Clique em Definir como Mestre para designar este servidor como um servidor mestre.

Nome DNS

O nome DNS do servidor empresarial do iFolder.

Por exemplo: 192.168.1.1 ou svr1.domain.com.

URL Público

O endereço IP público correspondente ao servidor iFolder. Para mudar o endereço IP, edite o endereço fornecido e clique em Gravar para gravar as mudanças efetuadas.

URL Particular

O endereço IP particular correspondente ao servidor iFolder. Isso permite a comunicação entre os servidores no domínio do iFolder. O URL particular e o URL público podem ser os mesmos. Para mudar o endereço IP, digite o novo endereço e clique em Gravar para gravar as mudanças efetuadas.

URL Master (exibido somente para os servidores Escravos)

O endereço IP correspondente ao servidor iFolder. Usando esse endereço, o servidor escravo se comunica com o servidor master no domínio do iFolder.

Para mudar o endereço IP, edite o endereço fornecido e clique em Gravar para gravar as mudanças efetuadas.

Parâmetro

Mostra o número total de ciclos de sincronização de identidade concluídos.

Sincronização de Identidade

Atualiza os usuários do iFolder no domínio do iFolder selecionado nas informações de LDAP, durante o intervalo que você escolheu.

Especifique o intervalo de tempo em minutos no campo Sincronização de Identidade e clique em Sincronizar Agora para iniciar imediatamente a sincronização dos usuários do iFolder com os usuários LDAP.

Intervalo extra para membro apagado

Especifica o intervalo de tempo para o iFolder apagar as informações do usuário do servidor, uma vez que esse determinado usuário foi apagado do LDAP. Esse recurso ajuda você a recuperar os dados se o usuário for apagado acidentalmente.


Você pode editar as seguintes informações relacionadas ao LDAP. Clique em Editar para abrir uma nova página em que as entradas possam ser modificadas.

Parâmetro

Descrição

Servidor LDAP

Mostra o endereço do Servidor LDAP.

SSL LDAP

Mostra se o SSL LDAP está habilitado. Também permite que você habilite ou desabilite a conexão SSL LDAP.

Usuário Proxy

Mostra o DN (Distinguished Name - Nome Exclusivo) do usuário Proxy. Por exemplo, cn=SimiasProxy,o=novell

O usuário Proxy do iFolder é a identidade usada para acessar o servidor LDAP a fim de recuperar uma lista de usuários nos containers ou grupos especificados, ou usuários que são definidos nas configurações LDAP do iFolder. O usuário Proxy do iFolder é criado durante a instalação. Essa identidade precisa ter o direito de leitura no diretório LDAP.

Senha do Usuário Proxy

 

A senha é utilizada para autenticar o usuário Proxy do iFolder no servidor LDAP quando o iFolder sincroniza os usuários com esse servidor.

Contexto do LDAP

O Contexto do LDAP lista todos os contextos LDAP. O iFolder pesquisa usuários somente nos contextos LDAP listados.


Data Store

O Data Store representa o armazenamento do iFolder que pode se segmentar em vários volumes (pontos de montagem) em um determinado servidor. Por padrão, cada servidor iFolder possui um armazenamento padrão que não pode ser desabilitado. Com a interface da Web, você pode adicionar e configurar vários Data Stores nos quais os dados do iFolder são carregados de forma equilibrada. Quando o usuário carrega um iFolder, ele verifica o Data Store com o máximo de espaço disponível e armazena os dados do iFolder nesse Data Store, equilibrando assim a carga. Você pode adicionar quantos Data Stores quiser. A vantagem de se ter vários Data Stores é que se torna possível escalonar a capacidade de armazenamento de dados em uma grande distribuição a fim de atender aos requisitos de nível empresarial.

Você pode ver as seguintes informações do Data Store:

Parâmetro

Descrição

Nome

Mostra o nome exclusivo que você especificou para o Data Store.

Caminho completo

Mostra o caminho para o Data Store, onde o volume está montado. Esse é o caminho de dados que você especificou ao adicionar o armazenamento de dados usando a interface da Web.

Libere espaço

Mostra o espaço disponível no volume.

Habilitado

Mostra se o Data Store fornecido está habilitado. O Data Store padrão não pode estar desabilitado.

Habilitar ou desabilitar Data Store

Selecione o Data Store que você deseja desabilitar ou habilitar e clique em Desabilitar ou Habilitar respectivamente. Quando um usuário carrega um iFolder, os Data Stores desabilitados são sempre ignorados ao verificar o espaço máximo livre disponível para o armazenamento de dados do iFolder.
