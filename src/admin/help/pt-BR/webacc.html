Iniciando o Web Admin

O console do Web Admin permite que você gerencie servidores, configurações de política, usuários e pastas do iFolder em um determinado domínio do iFolder 3.7 e posterior. Você pode habilitar usuários para o domínio do iFolder em usuários LDAP e gerenciar servidores que participam do domínio do iFolder. Com o Web Admin, você também pode assegurar o uso obrigatório da criptografia em usuários. O console do Web Admin terá a capacidade de produzir relatórios aprimorados que incluem o status geral do servidor, os relatórios da conta do iFolder e as informações sobre a conta do usuário.

NOTA: Você não pode gerenciar servidores do Novell iFolder 2.1x com o plug-in do Novell iFolder 3.7 e posterior para iManager.

Para acessar a página Gerenciamento do iFolder:

  1. No iManager, expanda a função do iFolder

  2. Clique em Iniciar o Web Admin para abrir o servidor iFolder na página de seleção.

  3. Especifique o nome DNS do servidor empresarial do iFolder ou endereço IP.

  4. Siga um destes procedimentos:

    • Se você tiver efetuado login no iManager com o mesmo nome de usuário que o usuário Admin do iFolder do Web Admin, selecione Autenticar Usando as Credenciais Atuais do iManager.

    • Se você tiver efetuado login no iManager com um nome de usuário diferente do usuário Admin do iFolder do Web Admin, deixe a caixa de diálogo Autenticar Usando as Credenciais Atuais do iManager desmarcada. Na página de login do Web Admin, especifique o nome de usuário e a senha do Admin do iFolder.

  5. Clique em OK.

Com base na seleção acima, você é direcionado para a página de usuários do Web Admin.
