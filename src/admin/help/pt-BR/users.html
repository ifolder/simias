Gerenciando uma conta de usuário

Use esta interface para gerenciar os usuários do iFolder no domínio do iFolder selecionado. A ferramenta de pesquisa ajuda você a localizar a conta de usuário do Novell iFolder® 3.7 ou posterior que você deseja gerenciar.

NOTA: O termo usuários do iFolder refere-se tanto aos usuários individuais quanto aos grupos LDAP.

Procurando os usuários

  1. Na página Usuários, selecione um critério de nome (Nome, Servidor inicial, Sobrenome, Nome de usuário).

  2. Selecione um critério de filtro (Contém, Começar com, Terminar com, É igual a).

  3. Use um ou mais dos seguintes métodos de pesquisa e clique em Pesquisar:

    • Digite o nome do usuário ou grupo LDAP no campo Pesquisar Usuários.

    • Digite uma ou mais letras no campo Pesquisar Usuários.

    • Digite um asterisco (*) no campo Pesquisar Usuários para retornar uma lista de todos os usuários e grupos LDAP no sistema.

    • Deixe o campo Pesquisar Usuários em branco para retornar uma lista de todos os usuários e grupos LDAP no sistema.

    Não clique na página até que seja totalmente atualizada. Em seguida, você poderá procurar, classificar ou gerenciar os usuários listados no relatório Pesquisar Resultados.

    Nota: Quando você efetuar login como um administrador secundário, só poderá ver os usuários para os quais foi designado como administrador secundário. Você também poderá gerenciar usuários, dependendo dos seus direitos.

Procurando os usuários nos Resultados da Pesquisa

Habilitando ou desabilitando um usuário do iFolder

Siga as etapas abaixo:

  1. Localize o usuário do iFolder que você deseja gerenciar e marque a caixa de seleção ao lado do usuário.

    • Clique em Habilitar para habilitar o servidor.

      Isso permitirá que o usuário faça login e sincronize iFolders.

    • Clique em Desabilitar para desabilitar o usuário.

      Se o usuário estiver conectado quando você fizer a mudança, a sessão do usuário continuará até o usuário se desconectar. A política fará efeito na próxima vez em que o usuário tentar efetuar login na conta. Para que o bloqueio faça efeito imediatamente, é preciso reiniciar os serviços Apache do servidor iFolder, que desconecta todas as sessões ativas, inclusive a sessão do usuário.

Aprovisionamento/Reprovisionamento de Usuários do iFolder

Use esta interface para provisionar ou reprovisionar manualmente os usuários do iFolder para diferentes servidores disponíveis no domínio iFolder atual.

Para provisionar/reprovisionar os usuários do iFolder, execute qualquer uma das seguintes ações:

Aprovisionamento usando a lista suspensa

  1. Localize os usuários do iFolder que você deseja aprovisionar.

  2. Selecione o servidor iFolder na lista suspensa da home page fornecida para cada usuário.

  3. Clique em Gravar.

Aprovisionamento usando o botão Aprovisionar

  1. Localize os usuários do iFolder que você deseja gerenciar.

  2. Marque a caixa de seleção ao lado de cada usuário.

  3. Clique em Provisionar para abrir uma nova página na qual você pode provisionar/reprovisionar os usuários selecionados a um servidor iFolder do domínio.

NOTA: Se no primeiro login o servidor pessoal não estiver definido no nível do usuário, o servidor atribuído ao grupo LDAP ao qual o usuário pertence lhe será atribuído automaticamente.

Criando um novo usuário do iFolder

NOTA:Você só pode criar usuários que não sejam LDAP. Isso será feito apenas quando o servidor iFolder não estiver conectado ao servidor LDAP.

  1. Na página Usuários do iFolder, clique em Novo.

    A página de criação do Usuário do iFolder se abre na guia Usuários

  2. Especifique as informações a seguir e clique em Criar.

Acessando a página Detalhes do Usuário

Siga o procedimento abaixo:

  1. Na página Usuário do iFolder, use a ferramenta de pesquisa para localizar o usuário cuja conta do iFolder você deseja gerenciar.

  2. Clique no link do nome do usuário para abrir a página Detalhes do Usuário na guia Usuários.
