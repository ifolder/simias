Spouštění aplikace Web Admin

Konzola Web Admin vám umožní spravovat servery, nastavení zásad, uživatele složek iFolder a složky iFolder v dané doméně iFolder 3.7 nebo novější. Můžete zapnout uživatele protokolu LDAP pro doménu iFolder a spravovat servery iFolder začleněné do domény iFolder. Pomocí konzoly Web Admin můžete rovněž u uživatelů uplatnit šifrování. Konzola Web Admin bude mít možnost vydat rozšířené zprávy, které obsahují celkový stav serveru, zprávy účtu iFolder a informace o uživatelském účtu.

POZNÁMKA: Nemůžete spravovat servery Novell iFolder 2.1x pomocí modulu plug-in Novell iFolder 3.7 nebo novějšího v aplikaci iManager.

Chcete-li přistoupit ke stránce Správa složky iFolder, postupujte následovně:

  1. V aplikaci iManager rozbalte roli iFolder.

  2. Klepnutím na tlačítko Spustit konzolu Web Admin otevřete stránku pro výběr serveru iFolder.

  3. Zadejte název DNS podnikového serveru iFolder nebo adresu IP.

  4. Použijte jednu z následujících možností:

    • Pokud jste přihlášeni k aplikaci iManager se stejným uživatelským jménem jako správce složky iFolder konzoly Web Admin, zaškrtněte zaškrtávací políčko Autorizovat pomocí aktuálního pověření aplikace iManager.

    • Pokud jste přihlášeni k aplikaci iManager s jiným uživatelským jménem než správce složky iFolder konzoly Web Admin, nechejte zaškrtávací políčko Autorizovat pomocí aktuálního pověření aplikace iManager nezaškrtnuté. Na přihlašovací stránce konzoly Web Admin zadejte uživatelské jméno a heslo správce složky iFolder.

  5. Klepněte na tlačítko OK.

V závislosti na předchozím výběru jste přesměrováni na uživatelskou stránku Web Admin.
