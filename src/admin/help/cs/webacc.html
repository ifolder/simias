Spouštění aplikace Web Admin

Konzola Web Admin vám umožní spravovat servery, nastavení zásad, uživatele složek iFolder a složky iFolder v dané doméně iFolder 3.7. Můžete zapnout uživatele protokolu LDAP pro doménu iFolder a spravovat servery iFolder začleněné do domény iFolder. Pomocí konzoly Web Admin můžete rovněž u uživatelů uplatnit šifrování. Konzola Web Admin bude mít možnost vydat rozšířené zprávy, které obsahují celkový stav serveru, zprávy účtu iFolder a informace o uživatelském účtu.
