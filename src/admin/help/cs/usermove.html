Stav opakovaného poskytování systému iFolder

Toto rozhraní slouží k zobrazení stavu opakovaného poskytování aktuální domény iFolder.

Na této stránce jsou uvedeny následující údaje:

Parametr

Popis

Typ

Zobrazuje, zda je uživatel poskytnut nebo neposkytnut.

Uživatelské jméno

Zobrazuje uživatelské jméno přiřazené uživatelskému účtu, například pnovak nebo petr.novak@priklad.cz

Aktuální domovský server

Zobrazuje domovský server přiřazený poskytnutému uživateli.

Nový domovský server

Zobrazuje nový server pro poskytování uživateli.

Dokončeno

Zobrazuje stav opakovaného poskytování uživatelů v procentech.

Stav opakovaného poskytování

Zobrazuje některý z následujících stavů opakovaného poskytování:

  • Probíhá inicializace
  • Inicializováno
  • Přesouvání složky iFolder
  • Obnovování domovského serveru
  • Dokončování
