Přihlášení do aplikace Web Admin

Chcete-li spravovat složky iFolder a uživatele, přihlašte se do aplikace Novell iFolder® 3.7 Web Admin. Uživatelské jméno a heslo správce složky iFolder se ověřují proti adresářovým službám LDAP.

Lze rovněž zadat, který z podporovaných jazyků se použije. Typicky je to stejný jazyk jako jazyk kódu podnikového serveru iFolder a místního počítače. Ujistěte se, že nastavení jazyku vašeho prohlížeče podporuje požadovaný jazyk.

  1. Zadejte uživatelské jméno a heslo účtu iFolder, ke kterému chcete přistupovat.

  2. Použijte rozevírací seznam Jazyky ke zvolení, který jazyk chcete použít pro přístup k vašim souborům.

  3. Klepněte na tlačítko Přihlásit.
