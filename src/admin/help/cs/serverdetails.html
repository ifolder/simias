Prohlížení podrobností o serveru

Můžete zobrazovat a upravovat následující údaje:

Parametr

Popis

Název (pouze zobrazení)

Název přiřazený serveru.

Typ

Typ serveru (hlavní nebo podřízený). Chcete-li tento server určit jako hlavní server, klepněte na možnost Nastavit jako hlavní.

Název DNS

Název DNS podnikového serveru iFolder.

Například: 192.168.1.1 nebo svrl.domain.com.

Veřejná URL

Veřejná adresa IP odpovídající serveru iFolder. Pokud chcete změnit adresu IP, upravte ji a provedené změny uložte klepnutím na tlačítko Uložit.

Soukromá URL

Skoukromá adresa IP odpovídající serveru iFolder. Ta umožní komunikaci mezi servery v rámci domény iFolder. Soukromá URL a veřejná URL může být stejná. Pokud chcete změnit adresu IP, zadejte novou adresu a provedené změny uložte klepnutím na tlačítko Uložit.

Hlavní adresa URL (zobrazuje se pouze pro podřízené servery)

Adresa IP odpovídající serveru iFolder. Prostřednictvím této adresy komunikuje podřízený server s hlavním serverem v doméně serveru iFolder.

Pokud chcete změnit adresu IP, upravte ji a provedené změny uložte klepnutím na tlačítko Uložit.

Synchronizace identifikačních dat

Aktualizuje uživatele složky iFolder ve vybrané doméně složky iFolder z údajů protokolu LDAP v zadaném intervalu.

Zadejte do pole Synchronizace identifikačních údajů časový interval v minutách a klepnutím na tlačítko Synchronizovat nyní proveďte okamžitou synchronizaci uživatelů složky iFolder s uživateli protokolu LDAP.

Interval odložení odstranění uživatele

Udává časový interval složky iFolder pro odstranění informací o uživateli ze serveru, jakmile se daný uživatel z protokolu LDAP odstraní. Tato funkce pomáhá obnovit data, pokud dojde nedopatřením k odstranění uživatele.


Následující informace týkající se protokolu LDAP můžete upravovat. Klepnutím na tlačítko Upravit otevřete novou stránku, na které můžete záznamy upravit.

Parametr

Popis

Server LDAP

Zobrazuje adresu serveru LDAP.

Zabezpečení SSL protokolu LDAP

Ukazuje, zda je zapnuté zabezpečení SSL protokolu LDAP, nebo ne. Rovněž umožňuje zapnout nebo blokovat připojení LDAP SSL.

Uživatel proxy

Ukazuje jednoznačné uživatelské jméno (DN) uživatele proxy. Například, en=SimiasProxy, o=novell

Uživatel proxy složky iFolder je identifikační údaj používaný pro přístup k serveru LDAP pro získání seznamu uživatelů v zadaných kontejnerech nebo skupinách, nebo uživatelů, kteří jsou definováni v nastavení protokolu LDAP složky iFolder. Uživatel proxy složky iFolder se vytváří během instalace složky iFolder. Tento identifikační údaj musí mít k adresáří LDAP právo na čtení.

Heslo uživatele proxy

 

Heslo se používá k ověření uživatele proxy služby iFolder na serveru LDAP při synchronizaci uživatelů se serverem LDAP.

Příkaz LDAP Context

Příkaz LDAP Context vypíše všechny kontexty protokolu LDAP. Aplikace iFolder vyhledává uživatele pouze v uvedených kontextech protokolu LDAP.


Datové úložiště

Datové úložiště reprezentuje úložiště složky iFolder, které se může rozkládat přes několik svazků (bodů připojení) na daném serveru. Implicitně má každý server iFolder výchozí úložiště, které nelze blokovat. Pomocí webového rozhraní můžete přidat a konfigurovat více datových úložišť, do kterých se data složky iFolder rozloží. Když uživatel vyšle složku iFolder, zjistí se datové úložiště s maximálním volným místem a data složky iFolder se uloží do této složky, a tím se vyváží zatížení. Datových úložišť můžete přidat libovolné množství. Při více datových úložištích je možné přizpůsobit kapacitu datového úložiště pro rozsáhlá rozložení, aby se splnily požadavky na podnikové úrovni.

Můžete zobrazit následující údaje o datovém úložišti:

Parametr

Popis

Název

Ukazuje jedinečný název, který je datovému úložišti přiřazen.

Úplná cesta

Ukazuje cestu k datovému úložišti, na které je svazek umístěn. Jedná se o cestu, která se zadala při přidávání datového úložiště pomocí webového rozhraní.

Volné místo

Ukazuje dostupné místo svazku.

Zapnuto

Ukazuje, zda je datové úložiště zapnuté, nebo ne. Výchozí datové úložiště nelze blokovat.

Zapnutí nebo blokování datového úložiště

Vyberte datové úložiště, které chcete blokovat nebo zapnout, a klepněte na tlačítko Blokovat, případně Zapnout. Když uživatel vysílá složku iFolder, blokovaná úložiště se vždy přeskočí při kontrole maximálního dostupného volného místa pro uložení dat složky iFolder.
