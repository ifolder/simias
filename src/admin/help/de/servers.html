Verwalten von iFolder-Servern

Verwenden Sie diese Schnittstelle zum Verwalten der iFolder-Server in einer Einrichtung mit mehreren Servern. Das Suchwerkzeug unterstützt Sie bei der Suche nach dem zu verwaltenden Server.

Durchsuchen von iFolder-Servern

  1. Wählen Sie ein Filterkriterium aus (Enthält, Beginnt mit, Endet mit, Ist gleich).

  2. Wenden Sie eine oder mehrere der folgenden Suchmethoden an und klicken Sie dann auf "Suchen":

    • Geben Sie den Namen des Servers in das Feld "Server suchen" ein.

    • Geben Sie mindestens einen Buchstaben in das Suchfeld ein.

    • Geben Sie ein Sternchen (*) in das Feld "Server suchen" ein, damit eine Liste aller im System vorhandenen Server wiedergegeben wird.

    • Lassen Sie das Feld "Server suchen" leer, damit eine Liste aller im System vorhandenen Server wiedergegeben wird.

    Klicken Sie nicht auf die Seite, bis sie vollständig neu geladen ist. Anschließend können Sie die im Suchergebnis angezeigte Serverliste durchsuchen, sortieren oder verwalten.

    Blättern Sie im Suchergebnis, bis Sie den zu verwaltenden Server gefunden haben.

Zugreifen auf die Seite "Serverdetails"

Befolgen Sie die unten angegebenen Schritte:

  1. Verwenden Sie das Suchwerkzeug auf der Seite "Server", um den gewünschten Server mithilfe eines Suchkriteriums zu suchen.

  2. Klicken Sie auf den Servernamenlink, um die Seite "Serverdetails" zu öffnen.

    WICHTIG:Die Informationen, die Sie auf dieser Seite anzeigen können, beziehen sich nur auf den Server, bei dem Sie aktuell angemeldet sind. Änderungen können Sie vornehmen, indem Sie sich in der Web Admin-Konsole des entsprechenden Servers anmelden.
