Wiederbereitstellungsstatus des iFolder-Systems

Mit dieser Schnittstelle können Sie den Wiederbereitstellungsstatus der aktuellen iFolder-Domäne anzeigen.

Auf dieser Seite werden die folgenden Details angezeigt:

Parameter

Beschreibung

Typ

Zeigt, ob die Benutzer bereitgestellt oder nicht bereitgestellt sind.

Benutzername

Der jedem Benutzerkonto zugewiesene Benutzername, beispielsweise hans.schmidt@beispiel.de.

Aktuelle Basis

Zeigt den Stammserver an, der einem bereitgestellten Benutzer zugewiesen ist.

Neue Basis

Zeigt den neuen Server an, der dem Benutzer bereitgestellt werden soll.

Abgeschlossen

Zeigt den Wiederbereitstellungsstatus als einen Prozentwert an.

Wiederbereitstellungsstatus

Zeigt einen der folgenden Wiederbereitstellungszustände an:

  • Initialisierung läuft
  • Fehler bei der Initialisierung
  • Initialisiert
  • iFolder wird verschoben
  • Home wird zurückgesetzt
  • Finalisierung
Löschen Klicken Sie auf diese Schaltfläche, um unerledigte Benutzeranfragen zum Verschieben zu löschen. Die Schaltfläche ist aktiviert, wenn bei einem Verschiebungsvorgang eines Benutzers während der Initialisierungsphase ein Problem auftritt.

Verwandte Themen

Verwalten eines Benutzerkontos
Verwalten von iFolder-Ordnern
Verwalten von iFolder-Ordnerdetails
