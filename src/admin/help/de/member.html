Verwalten der Web Admin-Konsole

Verwenden Sie diese Web Admin-Schnittstelle, um iFolder-Benutzer, LDAP-Gruppen, das iFolder-System, die iFolder-Server sowie iFolder-Ordner selbst und den iFolder-Bericht zu verwalten

Registerkarte "Benutzer"
Registerkarte "iFolder-Ordner"
Registerkarte "System"
Registerkarte "Server"

Registerkarte "Benutzer"

  1. Melden Sie sich mit dem Web Admin-Berechtigungsnachweis in Web Admin an.

    HINWEIS: Der Begriff iFolder-Benutzer bezieht sich sowohl auf einzelne Benutzer als auch auf Ldap-Gruppen.

    Auf der Seite "Benutzer" werden der Benutzertyp (Admin oder Benutzer), der Benutzername, der vollständige Name des Benutzers (falls verfügbar), der Server, auf dem der Benutzer registriert ist, sowie der Status des Benutzers ("Aktiviert" oder "Deaktiviert") angezeigt.

  2. Verwenden Sie die Suchfunktion, um den Benutzer zu finden, dessen iFolder-Konto Sie verwalten möchten.

  3. Klicken Sie auf den Benutzernamenlink, um die Seite "Benutzerdetails" zu öffnen.

    Die Seite "Benutzerdetails" wird mit der Registerkarte "Benutzer" geöffnet, auf der die Benutzerdetails, die iFolder-Ordner, deren Eigentümer dieser Benutzer ist oder in denen er Mitglied ist, sowie die Richtlinieneinstellungen für dieses Benutzerkonto angezeigt werden.

Registerkarte "iFolder-Ordner"


  1. Klicken Sie in der Web Admin-Konsole auf die Registerkarte "iFolder-Ordner".

    Auf der Seite "iFolder-Ordner" werden der Typ des iFolder-Ordners (Adminbenutzer oder Benutzer) sowie der Name, der Eigentümer, die Mitglieder, das Datum, an dem der iFolder-Ordner zum letzten Mal geändert wurde, und die Größe des iFolder-Ordners angezeigt.

  2. Verwenden Sie das Suchwerkzeug, um den zu verwaltenden iFolder-Ordner mithilfe eines Suchkriteriums zu suchen.

  3. Klicken Sie auf den iFolder-Link, um die Registerkarte "iFolder" der Seite "iFolder-Details" zu öffnen.

    Auf der Seite "iFolder-Details" werden die Details des iFolder-Ordners angezeigt, eine Liste mit den Mitgliedern und dem Eigentümer sowie die Richtlinieneinstellungen.

Registerkarte "System"


  1. Klicken Sie in der Web Admin-Konsole auf die Registerkarte "System".

    Auf der Seite "System" werden die Systemeinstellungen und eine Liste der iFolder-Administratoren angezeigt. Auf der Seite "System" können Sie die Richtlinieneinstellungen für das gesamte System verwalten.

  2. Suchen Sie den iFolder-Administrator, den Sie verwalten möchten. Sie können iFolder-Administratoren hinzufügen oder löschen.

    Außerdem können Sie die Richtlinieneinstellungen für den Adminbenutzer verwalten.

  3. Klicken Sie auf den Admin-Benutzernamenlink, um die Seite "Benutzerdetails" zu öffnen.

    Die Seite "Benutzerdetails" wird mit der Registerkarte "Benutzer" geöffnet, auf der die Benutzerdetails, die iFolder-Ordner, deren Eigentümer dieser Benutzer ist oder in denen er Mitglied ist, sowie die Richtlinieneinstellungen für dieses Benutzerkonto angezeigt werden.

Registerkarte "Server"


  1. Klicken Sie auf der Admin-Konsole auf die Registerkarte "Server".

    Die Seite "Server" zeigt den Servertyp (Master oder Slave), den Servernamen, die öffentliche IP-Adresse, die private IP-Adresse und den Status des ausgewählten Servers an.

  2. Verwenden Sie das Suchwerkzeug, um den zu verwaltenden Server zu suchen.

  3. Klicken Sie auf den Servernamenlink, um die Seite "Serverdetails" zu öffnen.

    Auf der Seite "Serverdetails" werden Serverdetails, der Servertyp (Master oder Slave), der DNS-Name, der Serverstatus, Serverprotokolle, Serverberichte, der Protokollierumfang, LDAP-Details und der iFolder-Datenspeicher angezeigt.

Registerkarte "Berichte"


  1. Klicken Sie in der Web Admin-Konsole auf "Berichte".

  2. Konfigurieren Sie Frequenz und Zeitplan der Berichterstellung und generieren Sie anschließend die Ausgabe wie gewünscht.

    Sie können den generierten Bericht in dem auf der Seite "Berichte" angegebenen Verzeichnis speichern.
