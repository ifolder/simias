Details zur Wiederbereitstellung für Benutzer

Mit dieser Schnittstelle zeigen Sie die Details zur Bereitstellung oder Wiederbereitstellung des Benutzers und die Details der iFolder an, die der Benutzer besitzt.

Auf der Seite "Benutzerbereitstellungsdetails" werden die folgenden Benutzerdetails für das iFolder-Konto des ausgewählten Benutzers angezeigt.

Parameter

Beschreibung

Benutzername

Der dem Benutzerkonto zugewiesene Benutzername, beispielsweise hans.schmidt@beispiel.de.

Vollständiger Name

Der Vorname und der Nachname des Benutzerkontos.

LDAP-Kontext

Der LDAP-Baumkontext wird für die Bereitstellung von Benutzern in der iFolder-Domäne verwendet. Beispiel: cn= test, o=novell.

Neue Basis

Zeigt den neuen Server an, der dem Benutzer bereitgestellt werden soll.

Abgeschlossen

Zeigt den Wiederbereitstellungsstatus als einen Prozentwert an.

Wiederbereitstellungsstatus

Zeigt einen der folgenden Wiederbereitstellungszustände an:

  • Initialisierung läuft
  • Initialisiert
  • iFolder wird verschoben
  • Home wird zurückgesetzt
  • Finalisierung

iFolder-Ordnerdetails

Auf der Seite "Benutzerbereitstellungsdetails" werden die folgenden iFolder-Details für das iFolder-Konto des ausgewählten Benutzers angezeigt.

Parameter

Beschreibung

Typ

Normaler iFolder-Ordner

Verschlüsselter iFolder-Ordner

Freigegebener iFolder-Ordner

Name

Der dem iFolder-Ordner zugewiesene Name.

Eigentümer

Der Benutzername des Eigentümers des ausgewählten iFolder-Ordners. Der Eigentümer kann ein einzelner Benutzer oder eine LDAP-Gruppe sein. Wenn eine LDAP-Gruppe Eigentümer des iFolders ist, haben alle Mitglieder der LDAP-Gruppe dieselben Zugriffsrechte. Bei iFolder-Ordnern ohne Eigentümer wird der Adminbenutzer zum Eigentümer, bis der iFolder-Ordner neu zugewiesen oder gelöscht werden kann.

Klicken Sie auf den Link des Benutzernamens des Eigentümers, um die Kontodetails des entsprechenden Benutzers anzuzeigen.

Der Eigentümer des iFolder-Ordners besitzt die Rechte "Vollständige Kontrolle" für den iFolder-Ordner. Der Eigentümer verwaltet die Mitgliedschaft und die Zugriffsrechte der Benutzer und hat die Möglichkeit, die Rechte "Vollständige Kontrolle" eines beliebigen Mitglieds aufzuheben.

Größe

Größe des iFolder-Ordners in MB.

Status

Zeigt einen der folgenden iFolder-Verschiebungszustände an:

  • Initialisierung läuft
  • Übertragung
  • Abgeschlossen

Verwandte Themen
