Starten von Web Admin

In der Web Admin-Konsole können Sie Server, Richtlinieneinstellungen, iFolder-Benutzer und iFolder-Ordner einer bestimmten iFolder-Domäne der Version 3.7 oder höher verwalten. Sie können LDAP-Benutzer für die Verwendung einer iFolder-Domäne aktivieren und die in der iFolder-Domäne teilnehmenden iFolder-Server verwalten. Mit Web Admin können Sie auch die Verschlüsselung von Benutzerdaten erzwingen. Die Web Admin-Konsole bietet umfassende Berichte wie beispielsweise Berichte zum Serverstatus, zu iFolder-Konten oder zu Benutzerkontoinformationen.

HINWEIS: Novell iFolder 2.1x-Server können nicht mit dem Plugin für iManager der Novell iFolder-Version 3.7 oder höher verwaltet werden.

So greifen Sie auf die iFolder-Verwaltungsseite zu:

  1. Erweitern Sie in iManager die Funktion "iFolder

  2. Klicken Sie auf "Web Admin starten", um die Seite zum Auswählen eines iFolder-Servers zu öffnen.

  3. Geben Sie den DNS-Namen oder die IP-Adresse des iFolder Enterprise-Servers an.

  4. Führen Sie einen der folgenden Vorgänge aus:

    • Wenn Sie sich in iManager mit demselben Benutzernamen wie demjenigen des iFolder-Adminbenutzers von Web Admin angemeldet haben, wählen Sie "Unter Verwendung des aktuellen iManager-Berechtigungsnachweises authentifizieren" aus.

    • Wenn Sie sich in iManager mit einem anderen als dem Benutzernamen des iFolder-Adminbenutzers von Web Admin angemeldet haben, lassen Sie das Kontrollkästchen "Unter Verwendung des aktuellen iManager-Berechtigungsnachweises authentifizieren" leer. Geben Sie auf der Anmeldeseite zu Web Admin den Benutzernamen und das Passwort des iFolder-Administrators ein.

  5. Klicken Sie auf "OK".

Je nach Ihrer Auswahl im oben beschriebenen Schritt werden Sie zur Seite "Web Admin-Benutzer" weitergeleitet.
