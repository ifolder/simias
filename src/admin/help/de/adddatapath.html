Datenablage hinzufügen

So fügen Sie eine neue Datenablage hinzu:

  1. Legen Sie die folgenden Werte fest:

    2. Name
    Weisen Sie der Datenablage einen eindeutigen Namen zu, wie z. B. ifolder-Datenablage.

    Pfad
    Geben Sie den Pfad des neuen Volumes ein.
    Wenn es sich um ein Remote-Volume handelt (CIFS, NFS, AFP), stellen Sie sicher, dass das Volume bei jedem Start ordnungsgemäß gemountet und der Lastausgleich gewährleistet ist.
    Sie müssen die Berechtigungen für den angegebenen Pfad überprüfen und die Eigentümerschaft und Gruppe in apacheuser (wwwrun) bzw. apachegroup (www) ändern. Sie können iFolder-Ordner in diesem Volume nur dann erstellen oder synchronisieren, wenn Sie die entsprechende Berechtigung für das Verzeichnis festgelegt haben, in dem das Volume gemountet ist.
