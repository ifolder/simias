Berichte

Verwenden Sie diese Schnittstelle, um die Berichterstellung zu aktivieren und Berichte für iFolder-Ordner und Verzeichnisse generieren zu können. Die Berichte werden mit der von Ihnen ausgewählten Häufigkeit generiert.

  1. Wählen Sie die Option "Berichterstellung aktivieren" aus, um die Berichterstellung zu aktivieren.

  2. Wählen Sie die Häufigkeiten aus den verfügbaren Optionen aus (Täglich, Wöchentlich, Monatlich).

  3. Wählen Sie die Uhrzeit, zu der Sie den Bericht generieren möchten.

  4. Wählen Sie den Standort, an dem Sie die Bereichte speichern möchten, aus den vorhandenen Optionen aus (Berichts-iFolder, Berichtsverzeichnis).

    Wenn Sie "Berichtsverzeichnis" auswählen, wird der generierte Bericht automatisch im Berichtsverzeichnis unter dem Standarddatenpfad gespeichert.

    Beispiel: <datenpfad>/simias/bericht.

    Wenn Sie "Bericht-iFolder-Ordner" auswählen, wird der generierte Bericht automatisch im Bericht-iFolder-Ordner gespeichert.

    Der Bericht-iFolder-Ordner mit dem Namen <domänenname>-<servername>-Berichte wird automatisch auf jedem Server für den Adminbenutzer erstellt.

  5. Wählen Sie das Format, in dem der Bericht generiert werden soll.

  6. Klicken Sie auf "Speichern", um die Einstellungen zu speichern.

    7. Klicken Sie auf "Abbrechen", um die festgelegten Einstellungen rückgängig zu machen.
