Verwalten eines Benutzerkontos

Verwenden Sie diese Schnittstelle zum Verwalten von iFolder-Benutzern in der ausgewählten Domäne. Das Suchwerkzeug unterstützt Sie beim Suchen des zu verwaltenden Novell iFolder® 3.7-Benutzerkontos.

HINWEIS: Der Begriff iFolder-Benutzer bezieht sich sowohl auf einzelne Benutzer als auch auf LDAP-Gruppen.

Suchen nach Benutzern

  1. Wählen Sie auf der Seite "Benutzer" ein Namenskriterium (Vorname, Stammserver, Nachname, Benutzername) aus.

  2. Wählen Sie ein Filterkriterium aus (Enthält, Beginnt mit, Endet mit, Ist gleich).

  3. Wenden Sie eine oder mehrere der folgenden Suchmethoden an und klicken Sie dann auf "Suchen":

    • Geben Sie den Namen des Benutzers bzw. der LDAP-Gruppe in das Feld "Benutzer suchen" ein.

    • Geben Sie mindestens einen Buchstaben in das Suchfeld ein.

    • Geben Sie ein Sternchen (*) in das Feld "Benutzer suchen" ein, damit eine Liste aller im System vorhandenen Benutzer und LDAP-Gruppen ausgegeben wird.

    • Lassen Sie das Feld "Benutzer suchen" leer, damit eine Liste aller im System vorhandenen Benutzer und LDAP-Gruppen wiedergegeben wird.

    Klicken Sie nicht auf die Seite, bis sie vollständig neu geladen ist. Anschließend können Sie die im Suchergebnis angezeigte Benutzerliste durchsuchen, sortieren oder verwalten.

    Hinweis: Wenn Sie als sekundärer Administrator angemeldet sind, können Sie nur die Benutzer anzeigen, für die Sie als sekundärer Administrator designiert sind. Sie können außerdem die Benutzer entsprechend den Ihnen zur Verfügung stehenden Rechten verwalten.

Auffinden der Benutzer in den Suchergebnissen

Blättern Sie im Suchergebnis, bis Sie den zu verwaltenden Benutzer gefunden haben. Zusammen mit dem Namen des Benutzers kann Ihnen die Angabe des Vor- und Nachnamens dabei helfen, den richtigen Ordner zu identifizieren.

Aktivieren oder Deaktivieren eines iFolder-Benutzers

Befolgen Sie die unten angegebenen Schritte:

  1. Suchen Sie den zu verwaltenden iFolder-Benutzer und aktivieren Sie anschließend das Kontrollkästchen neben diesem Benutzer.

    • Klicken Sie auf "Aktivieren", um den Benutzer zu aktivieren.

      Dies ermöglicht dem Benutzer, sich anzumelden und iFolder-Ordner zu synchronisieren.

    • Klicken Sie auf "Deaktivieren", um den Benutzer zu deaktivieren.

      Wenn ein Benutzer angemeldet ist, während Sie diese Änderung vornehmen, wird die Sitzung des Benutzers fortgesetzt, bis dieser sich abmeldet. Die Richtlinie wird angewendet, sobald der Benutzer das nächste Mal versucht, sich bei dem Konto anzumelden. Um die Zugriffssperre sofort anzuwenden, müssen Sie die Apache-Services des iFolder-Servers neu starten. Auf diese Weise werden alle aktiven Sitzungen inklusive der Sitzung des betroffenen Benutzers beendet.

Bereitstellung/+++Neubereitstellung+++ von iFolder-Benutzern

Mit dieser Schnittstelle können Sie die iFolder-Benutzer für mehrere Server manuell bereitstellen, die in der aktuellen iFolder-Domäne verfügbar sind.

Gehen Sie zur Bereitstellung/+++Neubereitstellung+++ der iFolder-Benutzer wie folgt vor:

Bereitstellung mithilfe einer Dropdown-Liste

  1. Suchen Sie die iFolder-Benutzer, die Sie bereitstellen möchten.

  2. Wählen Sie den iFolder-Server in der für die einzelnen Benutzer angegebenen Dropdown-Liste aus.

  3. Klicken Sie auf "Speichern".

Bereitstellen mithilfe der Schaltfläche "Bereitstellung"

  1. Suchen Sie die iFolder-Benutzer, die Sie verwalten möchten.

  2. Aktivieren Sie das Kontrollkästchen neben den einzelnen Benutzern.

  3. Klicken Sie auf die Schaltfläche "Bereitstellung", um eine neue Seite zu öffnen, in der Sie die ausgewählten Benutzer für einen iFolder-Server in der Domäne bereitstellen oder +++neu bereitstellen+++ können.

HINWEIS: Wenn ein Benutzer sich zum ersten Mal anmeldet und wenn auf der Benutzerebene kein Stammserver für den Benutzer festgelegt ist, wird dem Benutzer automatisch der Server zugeordnet, der der LDAP-Gruppe, der der Benutzer angehört, zugewiesen wurde.

Erstellen eines neuen iFolder-Benutzers

HINWEIS:Sie können nur Nicht-LDAP-Benutzer erstellen. Sie können Sie nur erstellen, wenn Ihr iFolder-Server nicht mit dem LDAP-Server verbunden ist.

  1. Klicken Sie auf der Seite iFolder-Benutzer" auf "Neu".

    Die Seite zum Erstellen von Benutzern wird geöffnet und die Registerkarte "Benutzer" wird angezeigt.

  2. Geben Sie die folgenden Informationen an und klicken Sie dann auf "Erstellen".

Benutzername
Weisen Sie dem Benutzerkonto einen Benutzernamen zu, beispielsweise hans.schmidt oder hans.schmidt@beispiel.de.

Vorname
Vorname des Benutzers.

Nachname
Nachname des Benutzers.

Vollständiger Name
Der Vorname und der Nachname des Benutzerkontos.

Passwort
Weisen Sie dem Benutzerkonto ein Passwort zu.

Passwort wiederholen
Geben Sie das Passwort zur Bestätigung noch einmal ein.

Zugreifen auf die Seite "Benutzerdetails"

Befolgen Sie die unten angegebenen Schritte:

  1. Verwenden Sie das Suchwerkzeug auf der Seite "iFolder-Benutzer", um den Benutzer, dessen iFolder-Konto Sie verwalten möchten, zu suchen.

  2. Klicken Sie auf den Benutzernamenlink, um die Seite "Benutzerdetails" mit angezeigter Registerkarte "Benutzer" zu öffnen.
