Uruchamianie programu Web Admin

Konsola Web Admin umożliwia zarządzanie serwerami, ustawieniami założeń, użytkownikami programu iFolder oraz folderami iFolder w danej domenie programu iFolder 3.7 lub nowszego. Można dodawać użytkowników do domeny iFolder spośród użytkowników usługi LDAP i zarządzać serwerami iFolder znajdującymi się w domenie iFolder. Przy użyciu programu Web Admin można również wymuszać szyfrowanie na kontach użytkowników. Konsola programu Web Admin umożliwia generowanie rozbudowanych raportów dotyczących ogólnego stanu serwera, kont programu iFolder oraz informacji o kontach użytkowników.
