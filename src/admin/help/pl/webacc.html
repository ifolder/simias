Uruchamianie programu Web Admin

Konsola programu Web Admin umożliwia zarządzanie serwerami, ustawieniami założeń, użytkownikami programu iFolder oraz folderami iFolder w danej domenie programu iFolder 3.7 lub nowszego. Można dodawać użytkowników do domeny iFolder spośród użytkowników usługi LDAP i zarządzać serwerami iFolder znajdującymi się w domenie iFolder. Przy użyciu programu Web Admin można również wymuszać szyfrowanie na kontach użytkowników. Konsola programu Web Admin umożliwia generowanie rozbudowanych raportów dotyczących ogólnego stanu serwera, kont programu iFolder oraz informacji o kontach użytkowników.

UWAGA: Przy użyciu dodatku typu plug-in Novell iFolder 3.7 lub nowszego do programu iManager nie można zarządzać serwerami Novell iFolder 2.1x.

Aby uzyskać dostęp do strony Zarządzanie systemem iFolder:

  1. W programie iManager rozwiń rolę iFolder

  2. Kliknij przycisk Uruchom program Web Admin, aby otworzyć stronę wyboru serwera iFolder.

  3. Podaj nazwę DNS lub adres IP serwera iFolder Enterprise.

  4. Wykonaj jedną z następujących czynności:

    • Zaznacz opcję Uwierzytelniaj przy użyciu bieżących poświadczeń programu iManager, jeśli logowanie do programu iManager nastąpiło przy użyciu nazwy użytkownika, który ma prawa administratora systemu iFolder w programie Web Admin.

    • Pozostaw pole wyboru Uwierzytelniaj przy użyciu bieżących poświadczeń programu iManager niezaznaczone, jeśli logowanie do programu iManager nastąpiło przy użyciu nazwy innego użytkownika niż użytkownik, który ma prawa administratora systemu iFolder w programie Web Admin. Na stronie logowania programu Web Admin podaj nazwę i hasło administratora systemu iFolder.

  5. Kliknij przycisk OK.

Na podstawie powyższego wyboru nastąpi przejście do strony administratorów programu Web Admin.
