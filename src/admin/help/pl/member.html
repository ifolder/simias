Zarządzanie konsolą programu Web Admin

Ten interfejs Programu Web Admin służy do zarządzania użytkownikami programu iFolder, grupami LDAP, systemem iFolder, serwerami, iFolderami i raportem statystycznym systemu iFolder.

Karta Użytkownicy

  1. Zaloguj się do programu Web Admin przy użyciu poświadczeń Web Admin.

    UWAGA: Pojęcie użytkownicy systemu iFolder oznacza zarówno użytkowników indywidualnych, jak i grupy LDAP.

    Strona Użytkownicy zawiera typ użytkownika (administrator lub użytkownik), jego nazwę, imię i nazwisko (jeśli są dostępne), status (włączony lub wyłączony) oraz serwer, do którego jest on przypisany.

  2. Użyj narzędzia wyszukiwania do zlokalizowania użytkownika, którego konto programu iFolder ma być zarządzane.

  3. Kliknij łącze w postaci nazwy użytkownika, aby otworzyć stronę Szczegółowe informacje o użytkowniku.

    Strona Szczegółowe informacje o użytkowniku zostanie otwarta na karcie Użytkownicy, która zawiera szczegóły użytkownika, posiadane i współużytkowane iFoldery oraz ustawienia założeń konta danego użytkownika.

Karta iFoldery


  1. W konsoli Web Admin kliknij kartę iFoldery.

    Strona iFoldery zawiera typ iFolderu (należący do administratora lub użytkownika), jego nazwę, właściciela, członków, datę ostatniej modyfikacji i rozmiar.

  2. Użyj narzędzia wyszukiwania i kryteriów wyszukiwania do zlokalizowania iFolderu, który ma być zarządzany.

  3. Kliknij łącze iFolderu, aby otworzyć stronę Szczegóły iFolderu na karcie iFolder.

    Strona Szczegóły iFolderu zawiera szczegóły iFolderu, listę członków, którzy są właścicielami iFolderu lub go współużytkują, oraz ustawienia założeń bieżącego iFolderu.

Karta System


  1. W konsoli Web Admin kliknij kartę System.

    Strona System zawiera ustawienia systemu i listę administratorów systemu iFolder. Przy użyciu strony System można zarządzać ustawieniami założeń całego systemu.

  2. Zlokalizuj administratora systemu iFolder, który ma być zarządzany. Można dodawać i usuwać administratorów systemu iFolder.

    Można również zarządzać ustawieniami założeń administratora.

  3. Kliknij łącze w postaci nazwy administratora, aby otworzyć stronę Szczegółowe informacje o użytkowniku.

    Strona Szczegółowe informacje o użytkowniku zostanie otwarta na karcie Użytkownicy, która zawiera szczegóły użytkownika, posiadane i współużytkowane iFoldery oraz ustawienia założeń konta danego użytkownika.

Karta Serwery


  1. W konsoli Web Admin kliknij kartę Serwery.

    Strona Serwery zawiera typ wybranego serwera (Nadrzędny lub Podrzędny), jego nazwę, publiczny adres IP, prywatny adres IP oraz stan.

  2. Użyj narzędzia wyszukiwania do zlokalizowania serwera, który ma być zarządzany.

  3. Kliknij łącze w postaci nazwy serwera, aby otworzyć stronę Szczegóły serwera.

    Strona Szczegóły serwera zawiera szczegóły serwera, typ serwera (Nadrzędny lub Podrzędny), jego nazwę DNS, stan, dzienniki, raporty, poziom dziennika, szczegóły usługi LDAP oraz magazyn danych iFolder.

Karta Raporty


  1. W konsoli Web Admin kliknij kartę Raporty.

  2. Skonfiguruj raportowanie zgodnie z wymaganą częstotliwością i planem, a następnie wygeneruj raport w żądany sposób.

    Wygenerowany raport można zapisać w katalogu określonym na stronie Raporty.
