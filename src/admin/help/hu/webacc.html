A Web Admin szolgáltatás indítása

A Web Admin konzollal kezelheti a kiszolgálókat, a házirend-beállításokat, az iFolder-felhasználókat és az iFolder mappákat egy adott, 3.7-es vagy újabb verziójú iFolder-tartományban. Engedélyezheti az LDAP-felhasználók közül az iFolder-tartomány felhasználóit, és kezelheti az iFolder-tartományban részt vevő iFolder-kiszolgálókat. A Web Admin konzollal a felhasználókra kényszeríthető továbbá a titkosítás használata. A Web Admin konzollal az általános kiszolgálóállapotot, az iFolder fiókokról szóló jelentéseket és a felhasználói fiók adatait is tartalmazó jelentések készíthetők.

MEGJEGYZÉS: A Novell iFolder 2.1x verziójú kiszolgálókat nem lehet kezelni az iManager programhoz tartozó Novell iFolder 3.7-es vagy újabb verziójú beépülő modullal.

Az iFolder kezelése lap elérése:

  1. Bontsa ki az iManager programban az iFolder szerepkört.

  2. Az iFolder kiszolgáló kiválasztását lehetővé tevő lap megnyitásához kattintson A Web Admin szolgáltatás indítása hivatkozásra.

  3. Adja meg az iFolder Enterprise kiszolgáló DNS-nevét vagy IP-címét.

  4. A következő lehetőségek közül választhat:

    • Ha az iManager programba ugyanazzal a felhasználónévvel van bejelentkezve, mint a Web Admin szolgáltatás iFolder rendszergazdai felhasználója, akkor válassza a Hitelesítés az iManager aktuális hitelesítő adataival lehetőséget.

    • Ha az iManager programba másik felhasználónévvel van bejelentkezve, mint a Web Admin szolgáltatás iFolder rendszergazdai felhasználója, akkor hagyja üresen a Hitelesítés az iManager aktuális hitelesítő adataival jelölőnégyzetet. A Bejelentkezés a Web Admin szolgáltatásba lapon adja meg az iFolder rendszergazdai felhasználónevét és jelszavát.

  5. Kattintson az OK gombra.

A fenti beállítások alapján a rendszer a rendszergazdai felhasználók webes lapjára irányítja.
