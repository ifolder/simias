A Web Admin konzol kezelése

Ezen a Web Admin kezelőfelületen lehet kezelni az iFolder felhasználókat, az LDAP-csoportokat, az iFolder rendszert, -kiszolgálókat, iFolder mappákat és az iFolder szolgáltatás statisztikai jelentését.

Felhasználók fül

  1. Jelentkezzen be a Web Admin szolgáltatásba a Web Admin hitelesítő adataival.

    MEGJEGYZÉS: Az iFolder felhasználók kifejezés az egyéni felhasználókat és az LDAP-csoportokat egyaránt jelenti.

    A Felhasználó lap megjeleníti a felhasználó típusát (rendszergazda vagy felhasználó), felhasználónevét, a felhasználó teljes nevét, a kiszolgálót, amelyhez a felhasználónak hozzáférése van, valamint a felhasználó állapotát (engedélyezve vagy letiltva).

  2. A keresési funkció használatával keresheti meg a felhasználót, akinek fiókját kezelni kívánja.

  3. Kattintson a felhasználó nevének hivatkozására a Felhasználó adatai lap megnyitásához.

    A Felhasználó adatai lap megnyitja a Felhasználók lapot, amely megjeleníti a felhasználó adatait, a tulajdonában álló iFolder mappákat, valamint az adott felhasználói fiókhoz tartozó megosztási és házirend-beállításokat.

iFolder mappák lap


  1. Kattintson a Web Admin konzol iFolder mappák lapfülére.

    Az iFolder mappák lap megjeleníti az iFolder típusát (rendszergazda vagy felhasználó), az iFolder nevét, tulajdonosát, tagjait, az iFolder mappa legutóbbi módosításának dátumát és a méretét.

  2. A keresőeszköz használatával a keresési feltételek alkalmazásával keresheti meg a kezelni kívánt iFolder mappát.

  3. Kattintson az iFolder nevének hivatkozására, ekkor az iFolder lapon megjelenik az iFolder mappa adatai lap.

    Az iFolder mappa adatai lap megjeleníti az iFolder mappa adatait, a tagok listáját, akiknek tulajdonában van, illetve akikkel meg van osztva, valamint az aktuális iFolder mappa házirend-beállításait.

Rendszerek fül


  1. Kattintson a Web Admin konzol Rendszerek lapfülére.

    A Rendszerek lap a rendszerbeállításokat és az iFolder rendszergazdák listáját jeleníti meg. A Rendszer lapon az egész rendszerre vonatkozó házirend-beállításokat lehet kezelni.

  2. Keresse meg a kezelni kívánt iFolder rendszergazdát. Hozzáadhatja vagy törölheti az iFolder mappa rendszergazdáját.

    Ezen kívül kezelheti a rendszergazdai felhasználóra vonatkozó házirend-beállításokat is.

  3. Kattintson a rendszergazdai felhasználó nevének hivatkozására a Felhasználó adatai lap megnyitásához.

    A Felhasználó adatai lap megnyitja a Felhasználók lapot, amely megjeleníti a felhasználó adatait, a tulajdonában álló iFolder mappákat, valamint az adott felhasználói fiókhoz tartozó megosztási és házirend-beállításokat.

Kiszolgálók fül


  1. Kattintson a Web Admin konzol Kiszolgálók lapfülére.

    A Kiszolgálók lap megjeleníti a kiszolgáló típusát (Fő vagy Szolga), a kiszolgáló nevét, a nyilvános és magán IP-címét, valamint a kijelölt kiszolgáló állapotát.

  2. A keresőeszköz használatával keresheti meg a kezelni kívánt kiszolgálót.

  3. Kattintson a kiszolgáló nevének hivatkozására a Kiszolgáló adatai lap megnyitásához.

    A Kiszolgáló adatai lap megjeleníti a kiszolgáló adatait, a kiszolgáló típusát (mester vagy szolga), a DNS-nevet, a kiszolgáló állapotát, a kiszolgáló naplóit, a kiszolgáló jelentéseit, a naplózás szintjét, az LDAP-adatokat és az iFolder-adattárat.

Jelentések fül


  1. Kattintson a Web Admin konzol Jelentések lapfülére.

  2. Konfigurálja a jelentést a kívánt gyakoriságnak és ütemtervnek megfelelően, majd hozza létre a kimenetet az igényeknek megfelelően.

    A létrehozott jelentést a Jelentések lapon megadott könyvtárba lehet menteni.
