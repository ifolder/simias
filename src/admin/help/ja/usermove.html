iFolderシステムの再プロビジョニング状態

このインタフェースは、現在のiFolderドメインの再プロビジョニング状態を表示する場合に使用します。

このページには以下の詳細が表示されます。

パラメータ

説明

タイプ

ユーザがプロビジョニングされているか、プロビジョニングされていないかを表示します。

［ユーザ名］

jsmithやjohn.smith@example.comなど、ユーザアカウントに割り当てられているユーザ名を表示します。

Current Home (現在のホーム)

プロビジョニング済みユーザに割り当てられているホームサーバを表示します。

New Home (新しいホーム)

ユーザにプロビジョニングする新しいサーバを表示します。

完了

プロビジョニングの状態をパーセントで表示します。

Reprovision State (再プロビジョニングステータス)

再プロビジョニングステータスを次のいずれかで表示します。

  • 初期化中
  • Initialization Failed (初期化に失敗)
  • Initialized (初期化済)
  • Moving iFolder (iFolderを移動中)
  • Resetting Home (ホームをリセット中)
  • Finalizing (最終処理を実行中)
削除 未完了のユーザ移動要求を削除するには、このボタンをクリックします。初期化段階でユーザ移動プロセスに問題が発生した場合、このボタンが有効になります。
