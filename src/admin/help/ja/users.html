ユーザアカウントの管理

このインタフェースでは、選択したiFolderドメイン内のiFolderユーザを管理できます。検索ツールを使用すると、管理するNovell iFolder® 3.7以降のユーザアカウントを簡単に探すことができます。

注意: 「iFolderユーザ」とは、個々のユーザおよびLDAPGroupの両方を指します。

ユーザの参照

  1. ［ユーザ］ページで、名前の条件(［名］、［Home Server (ホームサーバ)］、［姓］、［ユーザ名］)を選択します。

  2. フィルタ条件(［含む］、［開始文字］、［終了文字］、または［等しい］)を選択します。

  3. 次の検索方法を1つまたは複数使用して、［検索］をクリックします。

    • ［Search Users (ユーザの検索)］フィールドに、ユーザまたはLDAPGroupの名前を入力します。

    • ［Search Users (ユーザの検索)］フィールドに、1つ以上の文字を入力します。

    • システム上のすべてのユーザおよびLDAPGroupのリストを表示するには、［Search Users (ユーザの検索)］フィールドにアスタリスク(*)を入力します。

    • システム上のすべてのユーザおよびLDAPGroupのリストを表示するには、［Search Users (ユーザの検索)］フィールドを空白のままにします。

    ページが完全に更新されるまではページ内をクリックしないでください。更新されたら、検索結果レポートに表示されているユーザを参照、ソート、または管理できます。

    注意: セカンダリ管理者としてログインしている場合、セカンダリ管理者として指定を受けたユーザのみを表示できます。また、管理できるユーザは、使用できる権利によって異なります。

iFolderユーザの有効化または無効化

次の手順に従います。

  1. 管理するiFolderユーザを探して、そのユーザの横のチェックボックスをオンにします。

    • ユーザを有効にするには、［有効］をクリックします。

      これにより、ユーザはログインしてiFolderを同期できるようになります。

    • ユーザを無効にするには、［無効］をクリックします。

      この変更を行うときにユーザがログインしている場合、そのユーザのセッションは、ユーザがログアウトするまで維持されます。ポリシーは、ユーザが次回アカウントにログインしようとしたときに有効になります。ロックアウトを直ちに有効にするには、iFolderサーバのApacheサービスを再起動して、ユーザのセッションを含むすべてのアクティブセッションを終了する必要があります。

iFolderユーザのプロビジョニング/再プロビジョニング

このインタフェースでは、現在のiFolderドメイン内で使用可能な別のiFolderサーバに、iFolderユーザを手動でプロビジョニングまたは再プロビジョニングできます。

iFolderユーザをプロビジョニングまたは再プロビジョニングするには、次の手順に従います。

ドロップダウンリストによるプロビジョニング

  1. プロビジョニングするiFolderユーザを探します。

  2. ユーザごとに［ホーム］ドロップダウンリストからiFolderサーバを選択します。

  3. ［保存］をクリックします。

［プロビジョン］ボタンによるプロビジョニング

  1. 管理対象のiFolderユーザを探します。

  2. 各ユーザの横にあるチェックボックスをオンにします。

  3. ［プロビジョニング］をクリックすると、新しいページが開き、選択したユーザをドメイン内のiFolderサーバにプロビジョニングまたは再プロビジョニングできます。

注意: ユーザがはじめてログインしたときに、そのユーザのホームサーバがユーザレベルで設定されていない場合、そのユーザが属するLDAPGroupに割り当てられたサーバが自動的にそのユーザに割り当てられます。

新しいiFolderユーザの作成

注: LDAP以外のユーザのみ作成できます。ユーザを作成できるのは、iFolderサーバがLDAPサーバに接続されていない場合だけです。

  1. ［iFolderユーザ］ページで、［新規］をクリックします。

    ［ユーザ］タブに［iFolder User creation (iFolderユーザの作成)］ページが開きます。

  2. 次の情報を指定して［作成］をクリックします。

ユーザ名
jsmithやjohn.smith@example.comなどのユーザ名をユーザアカウントに割り当てます。

名前
ユーザの名前。

名字
ユーザの名字。

フルネーム
ユーザアカウントの名と姓。

パスワード
ユーザアカウントにパスワードを割り当てます。

パスワードの再入力
確認のためにもう一度パスワードを入力します。

［Users Details (ユーザ詳細)］ページへのアクセス

次の手順に従います。

  1. ［iFolderユーザ］ページで検索ツールを使用して、iFolderアカウントを管理するユーザを探します。

  2. ユーザ名のリンクをクリックして、［ユーザ］タブに［User Details (ユーザ詳細)］ページを開きます。
