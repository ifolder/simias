Web Adminへのログイン

Novell iFolder® 3.7 Web Adminへログインすると、iFolderとユーザを管理できます。iFolder管理者のユーザ名とパスワードは、LDAPディレクトリサービスを介して認証されます。

また、使用するサポート言語も指定できます。この言語は通常、iFolderエンタープライズサーバおよびローカルコンピュータの言語コードと同じ言語です。目的の言語をサポートするように、ブラウザの言語設定を設定してください。

  1. アクセスするiFolderアカウントのユーザ名とパスワードを指定します。

  2. ［使用言語］ドロップダウンリストを使用して、ファイルへのアクセスに使用する言語を指定します。

  3. ［ログイン］をクリックします。
