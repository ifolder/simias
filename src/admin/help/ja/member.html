Web Adminコンソールの管理

このWeb Adminインタフェースでは、iFolderユーザ、LDAPGroup、iFolderシステム、サーバ、iFolderおよびiFolder統計レポートを管理できます。

［ユーザ］タブ
［iFolder］タブ
［システム］タブ
［サーバ］タブ
［レポート］タブ

［ユーザ］タブ

  1. Web Adminの資格情報を使用して、Web Adminにログインします。

    注意: 「iFolderユーザ」とは、個々のユーザおよびLDAPGroupの両方を指します。

    ［ユーザ］ページには、ユーザのタイプ(管理者ユーザまたはユーザ)、ユーザ名、ユーザのフルネーム(使用可能な場合)、ユーザがプロビジョニングされているサーバ、およびユーザステータス(有効または無効)が表示されます。

  2. iFolderアカウントを管理するユーザを探すには、検索機能を使用します。

  3. ユーザ名のリンクをクリックして、［User Details(ユーザ詳細)］ページを開きます。

    ［User Details(ユーザ詳細)］ページが［ユーザ］タブに開き、ユーザ詳細、所有しているiFolder、共有しているiFolder、およびこの特定のユーザアカウントのポリシー設定が表示されます。

［iFolder］タブ


  1. Web Adminコンソールで、［iFolder］タブをクリックします。

    ［iFolder］ページに、iFolderタイプ(管理者ユーザまたはユーザ)、iFolder名、iFolderの所有者、メンバー、iFolderの最終変更日、およびiFolderのサイズが表示されます。

  2. 検索ツールで検索条件を使用して、管理するiFolderを探します。

  3. iFolderのリンクをクリックして、［iFolder］タブに［iFolder Details(iFolder詳細)］ページを開きます。

    ［iFolder Details(iFolder詳細)］ページに、iFolder詳細、iFolderを所有または共有しているメンバーのリスト、および現在のiFolderのポリシー設定が表示されます。

［システム］タブ


  1. Web Adminコンソールで、［システム］タブをクリックします。

    ［システム］ページに、システム設定およびiFolder管理者のリストが表示されます。［システム］ページでは、システム全体のポリシー設定を管理できます。

  2. 管理するiFolder管理者を探します。iFolder管理者を追加したり削除したりできます。

    管理者ユーザのポリシー設定も管理できます。

  3. 管理者ユーザの名前のリンクをクリックして、［User Details(ユーザ詳細)］ページを開きます。

    ［User Details(ユーザ詳細)］ページが［ユーザ］タブに開き、ユーザ詳細、所有しているiFolder、共有しているiFolder、およびこの特定のユーザアカウントのポリシー設定が表示されます。

［サーバ］タブ


  1. Web Adminコンソールで、［サーバ］タブをクリックします。

    ［サーバ］ページに、選択したサーバのサーバタイプ(マスタまたはスレーブ)、サーバ名、パブリックIPアドレス、プライベートIPアドレス、およびステータスが表示されます。

  2. 検索ツールを使用して、管理するサーバを探します。

  3. サーバ名のリンクをクリックして、［Servers Details(サーバ詳細)］ページを開きます。

    ［Server Details(サーバ詳細)］ページに、サーバ詳細、サーバタイプ(マスタまたはスレーブ)、DNS名、サーバステータス、サーバログ、サーバレポート、ログレベル、LDAP詳細、およびiFolderデータストアが表示されます。

［レポート］タブ


  1. Web Adminコンソールで、［レポート］タブをクリックします。

  2. 目的の頻度と時間スケジュールに従ってレポーティングを設定し、必要に応じて出力を生成します。

    生成されたレポートは、［レポート］ページで指定したディレクトリに保存できます。
