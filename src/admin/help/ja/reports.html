レポート

このインタフェースでは、レポーティングを有効にしたり、iFolderとディレクトリのレポートを生成したりできます。レポートは、選択した頻度に従って生成されます。

  1. レポーティングを有効にするには、［Enable Reporting (レポーティングを有効にする)］を選択します。

  2. オプションから頻度(［毎日］、［毎週］、または［毎月］)を選択します。

  3. レポートを生成する時刻を選択します。

  4. 指定されたオプション(［Report iFolder (iFolderのレポート)］、［Report Directory (ディレクトリのレポート)］)からレポートを保存する場所を選択します。

    ［Report Directory (ディレクトリのレポート)］を選択した場合、生成されたレポートは、デフォルトのデータパスの下にあるreportディレクトリに自動的に保存されます。

    たとえば、<data path>/simias/reportに保存されます。

    ［Report iFolder (iFolderのレポート)］を選択した場合、生成されたレポートは、Report iFolderに保存されます。

    管理者ユーザのサーバごとに、<domain name>-<server name>-Reportsという名前のReport iFolderが自動的に作成されます。

  5. 生成するレポートのフォーマットを選択します。

  6. ［保存］をクリックして、設定を保存します。

    7. 設定をキャンセルするには、［キャンセル］をクリックします。
