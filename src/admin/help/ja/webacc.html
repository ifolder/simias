Web Adminの起動

Web Adminコンソールを使用すると、iFolder 3.7およびそれ以降のドメイン内のサーバ、ポリシー設定、iFolderユーザ、およびiFolderを管理できます。また、LDAPユーザからiFolderドメインに対してユーザを有効にしたり、iFolderドメインに参加しているiFolderサーバを管理したりすることができます。Web Adminでは、ユーザに対して暗号化を強制することもできます。Web Adminコンソールには、サーバ全体のステータスを含む拡張レポート、iFolderアカウントレポート、およびユーザアカウント情報を生成する機能が用意されています。

注意: iManagerのNovell iFolder 3.7およびそれ以降のプラグインでNovell iFolder 2.1xサーバを管理することはできません。

［iFolder Management(iFolder管理)］ページにアクセスするには、次の手順に従います。

  1. iManagerで、iFolder のロールを展開します。

  2. ［Launch Web Admin(Web Adminの起動)］をクリックして、［iFolder server selection(iFolderサーバの選択)］ページを開きます。

  3. iFolderエンタープライズサーバのDNS名またはIPアドレスを指定します。

  4. 次のいずれかの操作を行います。

    • Web AdminのiFolder管理者ユーザと同じユーザ名でiManagerにログインしている場合は、［現在のiManagerアカウント情報を使用して認証します］チェックボックスをオンにします。

    • Web AdminのiFolder管理者ユーザとは異なるユーザ名でiManagerにログインしている場合は、［現在のiManagerアカウント情報を使用して認証します］チェックボックスをオフのままにします。Web Adminのログインページで、iFolder管理者のユーザ名とパスワードを指定します。

  5. ［OK］をクリックします。

上記の選択内容に基づいて、Web Adminのユーザページが表示されます。
