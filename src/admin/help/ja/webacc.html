Web Adminの起動

Web Adminコンソールを使用すると、iFolder 3.7以降のドメイン内のサーバ、ポリシー設定、iFolderユーザ、およびiFolderを管理できます。また、LDAPユーザからiFolderドメインに対してユーザを有効にしたり、iFolderドメインに参加しているiFolderサーバを管理したりすることができます。Web Adminでは、ユーザに対して暗号化を強制することもできます。Web Adminコンソールには、サーバ全体のステータスを含む拡張レポート、iFolderアカウントレポート、およびユーザアカウント情報を生成する機能が用意されています。
