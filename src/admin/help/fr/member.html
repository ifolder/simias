Gestion de la console Web Admin

Cette interface Web Admin permet de gérer les utilisateurs, les groupes LDAP et le système iFolder, les serveurs, les dossiers iFolder et le rapport sur les statistiques iFolder.

Onglet Utilisateurs

  1. Loguez-vous à Web Admin en utilisant les références Web Admin.

    REMARQUE : le terme utilisateurs iFolder fait référence à la fois aux utilisateurs individuels et aux groupes LDAP.

    La page Utilisateurs affiche le type d'utilisateur (utilisateur Admin ou utilisateur), le nom d'utilisateur, le nom complet de l'utilisateur (si disponible), le serveur auquel est relié l'utilisateur et l'état de l'utilisateur (activé ou désactivé).

  2. Utilisez la fonctionnalité de recherche pour localiser l'utilisateur dont vous souhaitez gérer le compte iFolder.

  3. Cliquez sur le lien du nom de l'utilisateur pour ouvrir la page Détails de l'utilisateur.

    La page Détails de l'utilisateur s'ouvre dans l'onglet Utilisateurs qui affiche les détails de l'utilisateur, les dossiers iFolder qu'il possède et partage ainsi que les paramètres de stratégie pour le compte utilisateur sélectionné.

Onglet Dossiers iFolder


  1. Sur la console Web Admin, cliquez sur l'onglet Dossiers iFolder.

    La page Dossiers iFolder affiche le type (utilisateur admin ou utilisateur), le nom, le propriétaire, les membres, la taille et la date de dernière modification du dossier iFolder.

  2. Utilisez l'outil de recherche pour localiser le dossier iFolder à gérer en utilisant les critères de recherche.

  3. Cliquez sur le lien du dossier iFolder pour ouvrir la page Détails iFolder dans l'onglet iFolder.

    La page Détails iFolder affiche les détails du dossier iFolder actuel, une liste des membres qui le possèdent ou le partagent et ses paramètres de stratégie.

Onglet Système


  1. Sur la console Web Admin, cliquez sur l'onglet Système.

    La page Système affiche les paramètres du système et la liste des administrateurs iFolder. La page Système permet de gérer les paramètres de stratégie pour l'ensemble du système.

  2. Localisez l'administrateur iFolder à gérer. Vous pouvez ajouter ou supprimer un administrateur iFolder.

    Vous pouvez également gérer les paramètres de stratégie pour l'utilisateur Admin.

  3. Cliquez sur le lien du nom de l'utilisateur Admin pour ouvrir la page Détails de l'utilisateur.

    La page Détails de l'utilisateur s'ouvre dans l'onglet Utilisateurs qui affiche les détails de l'utilisateur, les dossiers iFolder qu'il possède et partage ainsi que les paramètres de stratégie pour le compte utilisateur sélectionné.

Onglet Serveurs


  1. Sur la console Web Admin, cliquez sur l'onglet Serveurs.

    La page Serveurs affiche le type (Maître ou Esclave), le nom, l'adresse IP publique, l'adresse IP privée et l'état du serveur sélectionné.

  2. Utilisez l'outil de recherche pour localiser le serveur à gérer.

  3. Cliquez sur le lien du nom du serveur pour ouvrir la page Détails du serveur.

    La page Détails du serveur affiche les détails du serveur, son type (Maître ou Esclave), son nom DNS, son état, ses journaux, ses rapports, le niveau de consignation, les détails LDAP et la zone de stockage de données iFolder.

Onglet Rapports


  1. Sur la console Web Admin, cliquez sur Rapports.

  2. Configurez la création de rapports selon la fréquence et les heures souhaitées, puis générez la sortie comme vous le voulez.

    Vous pouvez enregistrer le rapport généré dans le répertoire spécifié sur la page Rapports.
