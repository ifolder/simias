Détails du reprovisioning de l'utilisateur

Cette interface permet d'afficher les détails de provisioning ou de reprovisioning de l'utilisateur ainsi que les détails des dossiers iFolder détenus par l'utilisateur.

La page Détails de provisioning de l'utilisateur affiche les détails de l'utilisateur suivants pour le compte iFolder de l'utilisateur sélectionné.

Paramètre

Description

Nom d'utilisateur

Affiche le nom d'utilisateur assigné au compte utilisateur, p. ex. jdupont ou jacques.dupont@exemple.com.

Nom complet

Affiche les nom et prénom du compte utilisateur.

Contexte LDAP

Le contexte de l'arborescence LDAP est utilisé pour le provisioning des utilisateurs dans le domaine iFolder. Par exemple, cn= test, o=novell.

Nouveau serveur privé

Affiche le nouveau serveur à provisionner pour l'utilisateur.

Terminé

Affiche le statut redéploiement sous forme de pourcentage.

État redéploiement

Affiche l'un des états redéploiement suivants :

  • Initialisation en cours
  • Initialisé
  • Déplacement de iFolder
  • Réinitialisation du serveur privé
  • Finalisation

Détails iFolder

La page Détails de provisioning de l'utilisateur affiche les détails iFolder suivants pour le compte iFolder de l'utilisateur sélectionné.

Paramètre

Description

Type

Dossier iFolder normal

Dossier iFolder codé

Dossier iFolder partagé

Nom

Affiche le nom assigné au dossier iFolder

Propriétaire

Affiche le nom d'utilisateur du propriétaire du dossier iFolder sélectionné. Le propriétaire peut être un utilisateur individuel ou un groupe LDAP. Si un groupe LDAP est le propriétaire du dossier iFolder, tous les membres du groupe LDAP disposent alors des mêmes droits d'accès. Pour les dossiers iFolder orphelins, l'utilisateur Admin iFolder est considéré comme propriétaire jusqu'à ce que le dossier iFolder puisse être réassigné ou supprimé.

Cliquez sur le lien du nom d'utilisateur du propriétaire pour afficher les détails de son compte utilisateur.

Le propriétaire du dossier iFolder possède le droit Contrôle total sur le dossier iFolder. Le propriétaire gère l'adhésion et les droits d'accès des utilisateurs et peut supprimer le droit Contrôle total pour n'importe quel membre.

Taille

Taille du dossier iFolder en Mo.

État

Affiche l'un des états de mouvement suivants :

  • Initialisation en cours
  • Transfert
  • Terminé
