Rapports

Cette interface permet d'activer la création de rapports et de générer des rapports pour des dossiers iFolder et des répertoires. Les rapports sont générés en fonction de la fréquence que vous sélectionnez.

  1. Sélectionnez l'option Activer les rapports pour activer la création de rapports.

  2. Sélectionnez la fréquence parmi les options données (Tous les jours, Toutes les semaines, Tous les mois).

  3. Sélectionnez l'heure à laquelle vous souhaitez générer le rapport.

  4. Sélectionnez l'emplacement d'enregistrement des rapports parmi les options données (Rapport iFolder, Répertoire de rapports).

    Si vous sélectionnez Répertoire des rapports, le rapport généré est automatiquement enregistré dans le répertoire des rapports sous le chemin d'accès aux données par défaut.

    Par exemple, <chemin données>/simias/rapport.

    Si vous sélectionnez Rapport iFolder, le rapport généré est enregistré dans le dossier iFolder Rapport.

    Le dossier iFolder Rapport, nommé <nom domaine>-<nom serveur>-Rapports est créé automatiquement pour chaque serveur de l'utilisateur Admin.

  5. Sélectionnez le format de génération du rapport.

  6. Cliquez sur Enregistrer pour enregistrer les paramètres.

    7. Cliquez sur Annuler pour annuler les paramètres.
