Gestion des dossiers iFolder

Cette interface permet de gérer la propriété, les stratégies et l'adhésion d'un dossier iFolder sélectionné pour un compte d'utilisateur iFolder se trouvant dans le domaine iFolder donné. L'outil de recherche vous aide à localiser le compte à gérer.

Remarque : lorsque vous êtes logué en tant qu'administrateur secondaire, vous ne pouvez afficher que les dossiers iFolder des utilisateurs pour lesquels vous avez été désigné comme administrateur secondaire. De même, vous ne pouvez gérer les dossiers iFolder qu'en fonction des droits qui vous sont assignés.

Recherche d'un dossier iFolder

Types de dossiers iFolder

L'icône indique que le dossier iFolder est normal.
L'icône indique que le dossier iFolder est partagé.
L'icône indique que le dossier iFolder est codé.

Cliquez sur Tout pour afficher tous les dossiers iFolder dans le domaine iFolder en cours. Cliquez sur Orphelin pour afficher tous les dossiers iFolder orphelins.

Récupération d'une liste des dossiers iFolder se trouvant sur le serveur

Utilisez une ou plusieurs des méthodes de recherche suivantes, puis cliquez sur Rechercher :

  • Sélectionnez Égale comme critère de filtre, puis entrez dans le champ Rechercher dossiers iFolder le nom du dossier iFolder à localiser.

  • Sélectionnez un critère de filtre (Commence par, Se termine par, Contient, Égale) pour le nom du dossier iFolder, puis entrez une ou plusieurs lettres dans le champ Rechercher dossiers iFolder.

  • Tapez un astérisque (*) dans le champ Rechercher dossiers iFolder pour obtenir une liste de tous les dossiers iFolder se trouvant sur le système.

  • Laissez le champ Rechercher dossiers iFolder vide pour obtenir une liste de tous les dossiers iFolder se trouvant sur le système.

Ne cliquez pas sur la page avant que celle-ci ne soit complètement rafraîchie. Une fois l'actualisation terminée, vous pouvez parcourir, trier ou gérer les dossiers iFolder listés dans les résultats de la recherche.

Consultation des dossiers iFolder dans les résultats de la recherche

Faites défiler la liste pour parcourir les résultats de la recherche et localiser le dossier iFolder à gérer. La combinaison du nom et du propriétaire du dossier iFolder peut vous aider à identifier le dossier iFolder souhaité.

Activation, désactivation et suppression d'un dossier iFolder


  1. Localisez le dossier iFolder à gérer, puis cochez la case en regard de celui-ci.

  2. Sélectionnez une opération à effectuer sur le dossier iFolder :

    • Cliquez sur Supprimer pour supprimer le dossier iFolder

    • Cliquez sur Activer pour activer le dossier iFolder.

      L'utilisateur peut alors se loguer et synchroniser les dossiers iFolder.

    • Cliquez sur Désactiver pour désactiver le dossier iFolder.

      Si l'utilisateur est logué pendant que vous effectuez cette modification, sa session reste active jusqu'à ce qu'il se délogue. La stratégie prendra effet la prochaine fois que l'utilisateur tentera de se loguer au compte. Pour que le verrouillage prenne effet immédiatement, vous devez redémarrer les services Apache pour le serveur iFolder. Toutes les sessions actives seront alors déconnectées, y compris celle de l'utilisateur

REMARQUE : par défaut, tous les dossiers iFolder sont activés.

Création d'un nouveau dossier iFolder


  1. Sur la page Utilisateurs iFolder, cliquez sur Nouveau.

    La page de création de dossiers iFolder s'ouvre sur la page Dossiers iFolder.

  2. Spécifiez les informations suivantes, puis cliquez sur Créer.

Nom
Assignez un nom au dossier iFolder, p. ex. ifolder 1.

Description
Affiche une brève description du dossier iFolder

Accès à la page Détails iFolder


  1. Utilisez l'outil de recherche pour localiser le dossier iFolder à gérer.

  2. Cliquez sur le lien du dossier iFolder pour ouvrir la page Détails iFolder dans l'onglet iFolder.

    La page Détails iFolder affiche les détails du dossier iFolder actuel, une liste des membres qui le possèdent ou le partagent et ses paramètres de stratégie.
