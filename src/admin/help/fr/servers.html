Gestion des serveurs iFolder

Cette interface permet de gérer les serveurs iFolder pour une configuration multiserveur. L'outil de recherche vous aide à localiser le serveur à gérer.

Recherche de serveurs iFolder

  1. Sélectionnez un critère de filtre (Contient, Commence par, Se termine par, Égale).

  2. Utilisez une ou plusieurs des méthodes de recherche suivantes, puis cliquez sur Rechercher :

    • Entrez le nom du serveur dans le champ Rechercher serveurs.

    • Tapez une ou plusieurs lettres dans le champ Rechercher serveurs.

    • Tapez un astérisque (*) dans le champ Rechercher serveurs pour obtenir une liste de tous les serveurs se trouvant sur le système.

    • Laissez le champ Rechercher serveurs vide pour obtenir une liste de tous les serveurs se trouvant sur le système.

    Ne cliquez pas sur la page avant que celle-ci ne soit complètement rafraîchie. Une fois l'actualisation terminée, vous pouvez parcourir, trier ou gérer les serveurs listés dans le rapport des résultats de la recherche.

    Faites défiler la liste pour parcourir les résultats de la recherche et localiser le serveur à gérer.

Accès à la page Détails du serveur.

Suivez la procédure décrite ci-dessous.

  1. Sur la page Serveur, utilisez l'outil de recherche pour localiser le serveur au moyen des critères de recherche.

  2. Cliquez sur le lien du nom du serveur pour ouvrir la page Détails du serveur à la page Serveurs.

    Important : les informations affichées sur cette page concernent uniquement le serveur auquel vous êtes logué. Pour effectuer des modifications, vous devez vous loguer à la console Web Admin propre au serveur.
