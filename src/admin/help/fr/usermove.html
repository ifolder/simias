Statut redéploiement du système iFolder

Cette interface permet d'afficher le statut redéploiement du domaine iFolder actuel.

Cette page affiche les détails suivants :

Paramètre

Description

Type

Affiche si les utilisateurs sont provisionnés ou non.

Nom d'utilisateur

Affiche le nom d'utilisateur assigné à chaque compte utilisateur, p. ex. jdupont ou jacques.dupont@exemple.com.

Serveur privé actuel

Affiche le serveur privé assigné à un utilisateur provisionné.

Nouveau serveur privé

Affiche le nouveau serveur à provisionner pour l'utilisateur.

Terminé

Affiche le statut redéploiement sous forme de pourcentage.

État redéploiement

Affiche l'un des états redéploiement suivants :

  • Initialisation en cours
  • Initialisé
  • Déplacement de iFolder
  • Réinitialisation du serveur privé
  • Finalisation
