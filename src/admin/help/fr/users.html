Gestion d'un compte utilisateur

Cette interface permet de gérer des utilisateurs iFolder dans le domaine iFolder sélectionné. L'outil de recherche vous aide à localiser le compte utilisateur Novell iFolder® 3.7 ou version ultérieure à gérer.

REMARQUE : le terme utilisateurs iFolder fait référence à la fois aux utilisateurs individuels et aux groupes LDAP.

Consultation des utilisateurs

  1. Sur la page Utilisateurs, sélectionnez un critère de nom (prénom, serveur privé, nom, nom d'utilisateur).

  2. Sélectionnez un critère de filtre (Contient, Commence par, Se termine par, Égale).

  3. Utilisez une ou plusieurs des méthodes de recherche suivantes, puis cliquez sur Rechercher :

    • Entrez le nom de l'utilisateur ou du groupe LDAP dans le champ Rechercher utilisateurs.

    • Tapez une ou plusieurs lettres dans le champ Rechercher utilisateurs.

    • Tapez un astérisque (*) dans le champ Rechercher utilisateurs pour obtenir une liste de tous les utilisateurs et groupes LDAP du système.

    • Laissez le champ Rechercher utilisateurs vide pour obtenir une liste de tous les utilisateurs et groupes LDAP du système.

    Ne cliquez pas sur la page avant que celle-ci ne soit complètement rafraîchie. Une fois l'actualisation terminée, vous pouvez parcourir, trier ou gérer les utilisateurs listés dans le rapport des Résultats de la recherche.

    Remarque : lorsque vous êtes connecté en tant qu'administrateur secondaire, vous ne pouvez afficher que les utilisateurs pour lesquels vous êtes désigné comme tel. De même, vous ne pouvez gérer les utilisateurs qu'en fonction des droits qui vous sont assignés.

Localisation des utilisateurs dans les résultats de la recherche

Faites défiler la liste pour parcourir les résultats de la recherche et localiser l'utilisateur à gérer. La combinaison du nom d'utilisateur, du prénom et du nom vous aide à identifier l'utilisateur que vous recherchez.

Activation ou désactivation d'un utilisateur iFolder

Suivez la procédure décrite ci-dessous.

  1. Localisez l'utilisateur iFolder à gérer, puis cochez la case en regard de celui-ci.

    • Cliquez sur Activer pour activer l'utilisateur.

      L'utilisateur peut alors se connecter et synchroniser ses dossiers iFolder.

    • Cliquez sur Désactiver pour désactiver l'utilisateur.

      Si l'utilisateur est connecté pendant que vous effectuez cette modification, sa session reste active jusqu'à ce qu'il se déconnecte. La stratégie prendra effet la prochaine fois que l'utilisateur tentera de se connecter au compte. Pour que le verrouillage prenne effet immédiatement, vous devez redémarrer les services Apache pour le serveur iFolder. Toutes les sessions actives seront alors déconnectées, y compris celle de l'utilisateur

Provisioning/reprovisioning d'utilisateurs iFolder

Cette interface permet de provisionner ou de reprovisionner manuellement les utilisateurs iFolder sur les différents serveurs disponibles dans le domaine iFolder actuel.

Pour provisionner/reprovisionner les utilisateurs iFolder, effectuez l'une des opérations suivantes :

Provisioning à l'aide de la liste déroulante

  1. Localisez les utilisateurs iFolder à provisionner.

  2. Sélectionnez le serveur iFolder dans la liste déroulante située en regard de chaque utilisateur.

  3. Cliquez sur Enregistrer.

Provisioning à l'aide du bouton Provisioning

  1. Localisez les utilisateurs iFolder à gérer.

  2. Cochez la case située à côté de chaque utilisateur.

  3. Cliquez sur Déploiement pour ouvrir une nouvelle page dans laquelle vous pouvez provisionner/reprovisionner les utilisateurs sélectionnés à un serveur iFolder du domaine.

REMARQUE : si aucun serveur privé n'est défini au niveau utilisateur lors de son premier login, le serveur assigné au groupe LDAP auquel il appartient est automatiquement affecté à l'utilisateur.

Création d'un nouvel utilisateur iFolder

REMARQUE :vous ne pouvez créer que des utilisateurs non LDAP. Vous pouvez les créer uniquement lorsque votre serveur iFolder n'est pas connecté au serveur LDAP.

  1. Sur la page Utilisateurs iFolder, cliquez sur Nouveau.

    La page de création d'utilisateurs iFolder s'ouvre dans l'onglet Utilisateurs

  2. Spécifiez les informations suivantes, puis cliquez sur Créer.

Nom d'utilisateur
Assignez un nom d'utilisateur au compte utilisateur, p. ex. jdupont ou jacques.dupont@exemple.com.

Prénom
Prénom de l'utilisateur.

Nom de famille
Nom de l'utilisateur.

Nom complet
Affiche les nom et prénom du compte utilisateur.

Mot de passe
Assignez un mot de passe au compte utilisateur.

Retype password
Réintroduisez le mot de passe pour le vérifier.

Accès à la page Détails des utilisateurs

Suivez la procédure décrite ci-dessous.

  1. Sur la page des utilisateurs iFolder, utilisez l'outil de recherche pour localiser l'utilisateur dont vous souhaitez gérer le compte iFolder.

  2. Cliquez sur le lien du nom de l'utilisateur pour ouvrir la page Détails de l'utilisateur dans l'onglet Utilisateurs.
