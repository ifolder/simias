Création d'un nouveau dossier iFolder

Pour créer un nouveau dossier iFolder, suivez la procédure décrite ci-dessous :

  1. Sur la page Utilisateurs iFolder, cliquez sur Nouveau.

    La page de création de dossiers iFolder s'ouvre sur la page Dossiers iFolder.

  2. Spécifiez les informations suivantes, puis cliquez sur Créer.

    Nom
    Assignez un nom au dossier iFolder, p. ex. ifolder 1.

    Description
    Une brève description du dossier iFolder.

  3. Cliquez sur Suivant pour sélectionner un propriétaire pour le dossier iFolder.

Sélection d'un propriétaire pour le nouveau dossier iFolder


  1. Sur la page Sélection du propriétaire, utilisez l'outil de recherche pour localiser l'utilisateur à qui vous souhaitez assigner la propriété.

  2. Cliquez sur Suivant pour sélectionner des membres supplémentaires pour le nouveau dossier iFolder.

    3. Cliquez sur Précédent pour apporter des modifications.

    Cliquez sur Annuler pour annuler les modifications effectuées.

  3. Cliquez sur Suivant pour sélectionner des membres supplémentaires pour le dossier iFolder.

Sélection de membres supplémentaires pour le nouveau dossier iFolder


  1. Sur la page Sélection de membres supplémentaires, utilisez l'outil de recherche pour localiser les utilisateurs souhaités.

  2. Sélectionnez les utilisateurs à partir des résultats de la recherche.

    3. Cliquez sur Précédent pour apporter des modifications.

  3. Cliquez sur Créer pour terminer la création du dossier iFolder.
