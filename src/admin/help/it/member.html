Gestione della console di Amministratore Web

Utilizzare questa interfaccia di Amministratore Web per gestire utenti, gruppi LDAP, sistema, server, cartelle e statistiche iFolder.

Scheda Utenti

  1. Eseguire il login ad Amministratore Web utilizzandone le credenziali.

    NOTA: Il termine utenti iFolder fa riferimento sia a singoli utenti che a gruppi LDAP.

    Nella pagina Utenti vengono visualizzati il tipo di utente (utente o utente amministratore), il nome utente, il nome completo dell'utente (se disponibile), il server in cui è stato eseguito il provisioning dell'utente e lo stato dell'utente (Abilitato o Disabilitato).

  2. Utilizzare la funzionalità di ricerca per individuare la cartella iFolder che si desidera gestire.

  3. Fare clic sul collegamento relativo al nome utente per aprire la pagina Dettagli utente.

    Nella scheda Utenti all'interno della pagina Dettagli utente vengono visualizzati i dettagli dell'utente, le cartelle iFolder di sua proprietà, nonché le impostazioni condivise e delle norme per il conto utente.

Scheda Cartelle iFolder


  1. Nella console di Amministratore Web fare clic sulla scheda Cartelle iFolder.

    Nella pagina Cartelle iFolder vengono visualizzati il tipo di cartella iFolder (utente o utente amministratore), il nome della cartella, il proprietario, i membri la data dell'ultima modifica e le dimensioni.

  2. Utilizzare lo strumento di ricerca per individuare la cartella iFolder che si desidera gestire mediante il criterio di ricerca.

  3. Fare clic sul collegamento relativo alla cartella iFolder per aprire la pagina Dettagli iFolder nella scheda iFolder.

    Nella pagina Dettagli iFolder verranno visualizzati i dettagli iFolder, un elenco dei membri che sono proprietari o condividono la cartella iFolder e le impostazioni delle norme per quest'ultima.

Scheda Sistema


  1. Nella console di Amministratore Web fare clic sulla scheda Sistemi.

    Nella pagina Sistema vengono visualizzati l'elenco degli amministratori iFolder e le impostazioni del sistema. Con la pagina Sistema è possibile gestire le impostazioni delle norme per l'intero sistema.

  2. Individuare l'amministratore iFolder che si desidera gestire. È possibile aggiungere o cancellare l'amministratore.

    È inoltre possibile gestire le impostazioni delle norme per l'utente amministratore.

  3. Fare clic sul collegamento relativo al nome utente amministratore per aprire la pagina Dettagli utente.

    Nella scheda Utenti all'interno della pagina Dettagli utente vengono visualizzati i dettagli dell'utente, le cartelle iFolder di sua proprietà, nonché le impostazioni condivise e delle norme per il conto utente.

Scheda Server


  1. Nella console di Amministratore Web fare clic sulla scheda Server.

    Nella pagina Server vengono visualizzati il tipo di server (master o slave), il nome del server, l'indirizzo IP pubblico e privato e lo stato del server selezionato.

  2. Utilizzare lo strumento di ricerca per individuare il server che si desidera gestire.

  3. Fare clic sul collegamento relativo al nome del server per aprire la pagina Dettagli server.

    Nella pagina Dettagli server vengono visualizzati i dettagli e il tipo di server (master o slave), il nome DNS, lo stato del server, i log, i rapporti, il livello di log, i dettagli LDAP e l'archivio dati della cartella iFolder.

Scheda Rapporti


  1. Nella console di Amministratore Web fare clic su Rapporti.

  2. Configurare la generazione di rapporti in base alla pianificazione di frequenza e tempo desiderata, quindi generare l'output.

    È possibile salvare il rapporto generato nella directory specificata nella pagina Rapporti.
