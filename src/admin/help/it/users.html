Gestione di un conto utente

Utilizzare questa interfaccia per gestire gli utenti iFolder nel dominio iFolder selezionato. Lo strumento di ricerca consente di individuare il conto utente Novell iFolder® 3.7 o versione successiva che si desidera gestire.

NOTA: con il termine utenti iFolder si fa riferimento sia ai singoli utenti sia ai gruppi LDAP.

Sfogliatura degli utenti

  1. Nella pagina Utenti selezionare un criterio relativo ai nomi (Nome, Home Server, Cognome, Nome utente).

  2. Selezionare un criterio di filtro (Contiene, Inizia con, Termina con, È uguale a).

  3. Utilizzare uno o più dei seguenti metodi di ricerca, quindi fare clic su Cerca:

    • Digitare il nome dell'utente o del gruppo LDAP nel campo per la ricerca di utenti.

    • Digitare una o più lettere nel campo di ricerca.

    • Digitare un asterisco (*) nel campo per la ricerca di utenti per ottenere un elenco di tutti gli utenti e i gruppi LDAP presenti nel sistema.

    • Lasciare vuoto il campo per la ricerca di utenti per ottenere un elenco di tutti gli utenti e i gruppi LDAP presenti nel sistema.

    Non fare clic sulla pagina finché non si è completamente aggiornata; a questo punto sarà possibile cercare il percorso degli utenti elencati nei risultati della ricerca oppure ordinarli o gestirli.

    Nota: quando il login è stato eseguito come amministratore secondario, è possibile visualizzare solo gli utenti per i quali si è stati designati come amministratore secondario. Inoltre, è possibile gestire gli utenti a seconda dei diritti disponibili.

Individuazione degli utenti nei risultati della ricerca

Scorrere la pagina verso l'alto e verso il basso per cercare il percorso dei risultati della ricerca e individuare l'utente che si desidera gestire. La combinazione di nome utente e nome e cognome consente di identificare l'utente desiderato.

Abilitazione o disabilitazione di un utente iFolder

Seguire la procedura riportata di seguito:

  1. Individuare l'utente iFolder che si desidera gestire, quindi selezionare la casella di controllo vicino all'utente.

    • Fare clic su Abilita per abilitare l'utente.

      Questa operazione consente all'utente di eseguire il login e sincronizzare le cartelle iFolder.

    • Fare clic su Disabilita per disabilitare l'utente.

      Se l'utente esegue il login in fase di modifica, la sessione dell'utente continua finché l'utente non esegue il logout. La norma diverrà effettiva al successivo tentativo di login dell'utente al conto. Affinché il blocco divenga effettivo da subito, è necessario riavviare i servizi Apache per il server iFolder, che disconnette tutte le sessioni attive, inclusa quella dell'utente.

Eseguire o rieseguire il provisioning di utenti iFolder

Questa interfaccia consente di eseguire o rieseguire manualmente il provisioning degli utenti in server diversi disponibili nel dominio iFolder attuale.

Per eseguire/rieseguire il provisioning di utenti iFolder, eseguire una o più delle operazioni seguenti:

Provisioning mediante elenco a discesa

  1. Individuare gli utenti iFolder di cui eseguire il provisioning.

  2. Selezionare il server iFolder dall'elenco a discesa Home per ogni utente.

  3. Fare clic su Salva.

Provisioning mediante il pulsante Esegui provisioning

  1. Individuare l'amministratore iFolder che si desidera gestire.

  2. Selezionare la casella di controllo accanto a ogni utente.

  3. Fare clic su Esegui provisioning per aprire una nuova pagina in cui eseguire/rieseguire il provisioning degli utenti selezionati in un server iFolder nel dominio.

NOTA: quando un utente esegue il login per la prima volta e per l'utente non è stato impostato un home server a livello dell'utente, a quest'ultimo verrà automaticamente assegnato il server del gruppo LDAP a cui egli appartiene.

Creazione di un nuovo utente iFolder

NOTA: è possibile creare solo utenti non LDAP e solo quando il server iFolder non è connesso al server LDAP.

  1. Fare clic su Nuovo nella pagina Utenti iFolder.

    La pagina di creazione degli utenti iFolder viene aperta alla scheda Utenti

  2. Specificare le seguenti informazioni, quindi fare clic su Crea.

Nome utente
Assegnare un nome utente al conto utente, ad esempio mrossi o mario.rossi@esempio.com.

Nome
Nome dell'utente.

Cognome
Cognome dell'utente.

Nome completo
Nome e cognome del conto utente.

Password
Assegnare una password al conto utente.

Conferma password
Digitare nuovamente la password per verificarla.

Accesso alla pagina Dettagli utente

Seguire la procedura riportata di seguito:

  1. Utilizzare lo strumento di ricerca nella pagina degli utenti iFolder per individuare l'utente di cui si desidera gestire il conto iFolder.

  2. Fare clic sul collegamento relativo al nome utente per aprire la pagina Dettagli utente alla scheda Utenti.
