Gestione dei server iFolder

Utilizzare questa interfaccia per gestire i server iFolder in caso di una configurazione con più server. Lo strumento di ricerca consente di individuare il server che si desidera gestire.

Ricerca di server iFolder

  1. Selezionare un criterio di filtro (Contiene, Inizia con, Termina con, È uguale a).

  2. Utilizzare uno o più dei seguenti metodi di ricerca, quindi fare clic su Cerca:

    • Digitare il nome del server nel campo di ricerca corrispondente.

    • Digitare una o più lettere nel campo di ricerca.

    • Digitare un asterisco (*) nel campo di ricerca dei server per ottenere un elenco di tutti i server presenti nel sistema.

    • Lasciare vuoto il campo di ricerca dei server per ottenere un elenco di tutti i server presenti nel sistema.

    Non fare clic sulla pagina finché non si è completamente aggiornata; a questo punto sarà possibile cercare il percorso dei server elencati nei risultati della ricerca oppure ordinarli o gestirli.

    Scorrere la pagina verso l'alto e verso il basso per cercare il percorso dei risultati della ricerca e individuare il server che si desidera gestire.

Accesso alla pagina Dettagli server

Seguire la procedura riportata di seguito:

  1. Nella pagina Server utilizzare lo strumento di ricerca per individuare il server mediante il criterio di ricerca.

  2. Fare clic sul collegamento relativo al nome del server per aprire la pagina Dettagli server alla pagina Server.

    Importante: le informazioni visualizzate in questa pagina sono specifiche solo del server a cui è stato attualmente eseguito il login. Per apportare qualsiasi modifica, è necessario eseguire il login alla rispettiva console di Amministratore Web del server.
