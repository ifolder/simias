Stato del nuovo provisioning del sistema iFolder

Questa interfaccia consente di visualizzare lo stato del nuovo provisioning del dominio iFolder attuale.

In questa pagina vengono visualizzati i dettagli seguenti:

Parametro

Descrizione

Tipo

Indica se si tratta di utenti di cui è stato eseguito oppure non eseguito il provisioning.

Nome utente

Visualizza il nome utente assegnato al ciascun conto utente, ad esempio mrossi o mario.rossi@esempio.com.

Home server attuale

Indica l'home server assegnato all'utente di cui è stato eseguito il provisioning.

Nuovo home server

Indica il nuovo server in cui eseguire il provisioning dell'utente.

Completato

Indica lo stato del nuovo provisioning in percentuale.

Stato nuovo provisioning

Visualizza uno degli stati seguenti per il nuovo provisioning:

  • Inizializzazione in corso
  • Inizializzato
  • Spostamento cartelle iFolder in corso
  • Reimpostazione home in corso
  • Finalizzazione in corso
