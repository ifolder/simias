Rapporti

Utilizzare questa interfaccia per abilitare la generazione e generare rapporti per la cartella iFolder e le directory. I rapporti vengono generati in base alla frequenza selezionata.

  1. Selezionare l'opzione per la generazione di rapporti per abilitare tale funzione.

  2. Selezionare la frequenza tra le opzioni specificate (Ogni giorno, Ogni settimana, Ogni mese).

  3. Selezionare l'ora in cui si desidera generare il rapporto.

  4. Selezionare l'ubicazione in cui si desidera salvare i rapporti dalle opzioni indicate (iFolder rapporti, directory rapporti)

    Se si seleziona l'opzione Directory rapporto, il rapporto generato viene automaticamente salvato nella directory del rapporto al percorso dati di default.

    Ad esempio, <percorso dati>/simias/report.

    Se si seleziona l'opzione Rapporto iFolder, il rapporto generato viene salvato nell'omonima cartella.

    Il rapporto, denominato <nome dominio>-<nome server>-Reports viene creato automaticamente per ciascun server per l'utente amministratore.

  5. Selezionare il formato per la generazione del rapporto.

  6. Fare clic su Salva per salvare le impostazioni.

    7. Fare clic su Annulla per annullare le impostazioni.
