Avvio di Amministratore Web

La console di Amministratore Web consente di gestire server, impostazioni delle norme, utenti e cartelle iFolder in un dominio iFolder 3.7 o versione successiva specifico. È possibile abilitare utenti al dominio iFolder dagli utenti LDAP e gestire i server iFolder che partecipano al dominio iFolder. Con Amministratore Web è inoltre possibile imporre la cifratura degli utenti. La console è in grado di produrre rapporti avanzati che includono lo stato complessivo del server, i rapporti del conto iFolder e le informazioni relative al conto utente.
