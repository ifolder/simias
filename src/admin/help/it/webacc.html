Avvio di Amministratore Web

La console di Amministratore Web consente di gestire server, impostazioni delle norme, utenti e cartelle iFolder in un dominio iFolder 3.7 o versione successiva specifico. È possibile abilitare utenti al dominio iFolder dagli utenti LDAP e gestire i server iFolder che partecipano al dominio iFolder. Con Amministratore Web è inoltre possibile imporre la cifratura degli utenti. La console è in grado di produrre rapporti avanzati che includono lo stato complessivo del server, i rapporti del conto iFolder e le informazioni relative al conto utente.

NOTA: non possibile gestire i server Novell iFolder 2.1x con il plug-in Novell iFolder 3.7 e versioni successive per iManager.

Per accedere alla pagina di gestione iFolder:

  1. In iManager espandere il ruolo iFolder

  2. Fare clic sull'opzione di avvio di Amministratore Web per aprire la pagina di selezione dei server iFolder.

  3. Specificare il nome DNS o l'indirizzo IP del server aziendale iFolder.

  4. Effettuare una delle seguenti operazioni:

    • Se è stato eseguito il login a iManager con lo stesso nome utente dell'utente amministratore di iFolder di Amministratore Web, selezionare l'opzione Esegui autenticazione mediante credenziali di iManager attuali.

    • Se è stato eseguito il login a iManager con un nome utente diverso rispetto all'utente amministratore di iFolder di Amministratore Web, lasciare deselezionata la casella di controllo Esegui autenticazione mediante credenziali di iManager attuali. Specificare nella pagina di login di Amministratore Web il nome utente e la password dell'amministratore iFolder.

  5. Fare clic su OK.

Con questa selezione, l'utente viene indirizzato alla pagina degli utenti di Amministratore Web.
