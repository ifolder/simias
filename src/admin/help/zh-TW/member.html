管理 Web 管理員主控台

使用此「Web 管理員」介面來管理 iFolder 使用者、LdapGroup、iFolder 系統、伺服器、iFolder 和 iFolder 統計報告。

使用者索引標籤

  1. 使用「Web 管理員」身份證明登入「Web 管理員」。

    注意：iFolder 使用者一詞代表的包括個別使用者和 LdapGroup。

    「使用者」頁會顯示使用者的類型 (管理員使用者或使用者)、使用者名稱、使用者的全名 (如果可用的話)、提供使用者的伺服器，以及使用者狀態 (「啟用」或「停用」)。

  2. 使用搜尋功能找出您想要管理 ifolder 帳戶的使用者。

  3. 按一下使用者名稱連結，以開啟「使用者詳細資料」頁。

    「使用者詳細資料」頁會開啟「使用者」索引標籤，其中會顯示使用者詳細資料、擁有的 iFolder，以及這個特定使用者帳戶的共用和規則設定。

iFolder 索引標籤


  1. 在「Web 管理員」主控台上按一下「iFolder」索引標籤。

    「iFolder」頁會顯示 iFolder 類型 (管理員使用者或使用者)、iFolder 名稱、iFolder 擁有者、成員、iFolder 上次修改的日期，以及 iFolder 大小。

  2. 請使用搜尋工具找出您想要使用搜尋標準管理的 iFolder。

  3. 按一下 iFolder 的連結，以開啟「iFolder 詳細資料」的「iFolder」索引標籤。

    「iFolder 詳細資料」頁會顯示 iFolder 詳細資料、擁有或共用 iFolder 的成員清單，以及目前 iFolder 的規則設定。

系統索引標籤


  1. 在「Web 管理員」主控台上，按一下「系統」索引標籤。

    「系統」頁會顯示系統設定和 iFolder 管理員的清單。您可以使用「系統」頁管理整個系統的規則設定。

  2. 找出您想要管理的「iFolder 管理員」。您可以加入或刪除「iFolder 管理員」。

    您也可以管理 Admin 使用者的規則設定。

  3. 按一下 Admin 使用者的名稱連結，以開啟「使用者詳細資料」頁。

    「使用者詳細資料」頁會開啟「使用者」索引標籤，其中會顯示使用者詳細資料、擁有的 iFolder，以及這個特定使用者帳戶的共用和規則設定。

伺服器索引標籤


  1. 在「Web 管理員」主控台上，按一下「伺服器」索引標籤。

    「伺服器」頁會顯示伺服器類型 (「主要」或「附屬」)、伺服器名稱、公用 IP 位址，以及選定伺服器的狀態。

  2. 請使用搜尋工具找出您要管理的「伺服器」。

  3. 按一下「伺服器」名稱連結，開啟「伺服器詳細資料」頁。

    「伺服器詳細資料」頁會顯示伺服器詳細資訊、伺服器類型 (「主要」或「附屬」)、DNS 名稱、伺服器狀態、伺服器記錄、伺服器報告、記錄層級、LDAP (輕量目錄存取協定) 詳細資料和「iFolder 資料儲存」。

報告索引標籤


  1. 在「Web 管理員」主控台上，按一下「報告」。

  2. 根據所需的頻率和時間排程設定報告的組態，然後再視需要產生輸出。

    您可以將產生的報告儲存在「報告」頁上指定方目錄中。
