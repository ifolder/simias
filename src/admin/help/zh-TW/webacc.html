啟動 Web 管理員

「Web 管理員」主控台可讓您管理伺服器、規則設定、iFolder 使用者，以及指定 iFolder 3.7 或更新版本之網域內的 iFolder。您可以讓使用者從 LDAP 使用者進入 iFolder 網域，並管理參與 iFolder 網域的使用者。您也可以使用「Web 管理員」對使用者強制執行加密。「Web 管理員」主控台將會提供產生增強型報表的功能，其中包括整體的伺服器狀態、iFolder 帳戶報告和使用者帳戶資訊。

注意：您不能使用 Novell iFolder 3.7 及更新的 iManager 外掛程式來管理 Novell iFolder 2.1x 伺服器。

若要存取「iFolder 管理」頁：

  1. 在 iManager 中，展開「iFolder 」角色。

  2. 按一下「啟動 Web 管理員」，開啟「iFolder 伺服器選擇」頁。

  3. 指定「iFolder 企業」伺服器的 DNS 名稱或 IP 位址。

  4. 請執行下列其中一個步驟：

    • 如果您已使用與「Web 管理員」之「iFolder 管理員」使用者相同的使用者名稱登入 iManager，請選取「使用目前的 iManager 身份證明進行驗證」。

    • 如果您使用與「Web 管理員」之「iFolder 管理員」使用者不同的使用者名稱登入 iManager，請讓「使用目前的 iManager 身份證明進行驗證」核取方塊保持為未選取狀態。在「Web 管理員登入」頁中，指定「iFolder 管理員」的使用者名稱和密碼。

  5. 按一下「確定」。

根據上述的選擇，您將會被導向至「Web 管理員使用者」頁。
