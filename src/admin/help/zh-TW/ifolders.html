管理 iFolder

使用此介面為使用者位於指定 iFolder 領域上的 iFolder 使用者帳戶管理選定 iFolder 的擁有權、規則和成員資格。搜尋工具可協助您找出想要管理的帳戶。

注意：當您以次要管理員身分登入時，只能檢視您被指定為其次要管理員之使用者所擁有的 iFolder。此外，您只能根據您對 iFolder 擁有的管理權限來管理 iFolder。

搜尋 iFolder

iFolder 類型

指出 iFolder 為「一般」的 iFolder。
指出 iFolder 為「共享」的 iFolder。
指出 iFolder 為「加密」的 iFolder。

按一下「全部」以檢視目前 iFolder 領域的所有 iFolder。按一下「錯誤鏡像複製區」以檢視所有錯誤鏡像複製 iFolder。

取回伺服器上的 iFolder 清單

使用下列一或多種搜尋方法，然後按一下「搜尋」：

  • 選取「等於」做為過濾器標準，然後在「搜尋 iFolder」欄位中輸入您要尋找的 iFolder 名稱。

  • 為 iFolder 的名稱選取過濾器標準 (「開始於」、「結束以」、「包含」、「等於」)，然後在「搜尋 iFolder」欄位中輸入一或多個字母。

  • 在「搜尋 iFolder」欄位內輸入星號 (*)，以傳回系統上所有 iFolder 的清單。

  • 讓「搜尋 iFolder」欄位保留為空白，以傳回系統上所有 iFolder 的清單。

在頁面完全重新整理之前，請勿按下頁面的任何地方；接著您便可以瀏覽、排序或管理「搜尋結果」報告內列出的 iFolder。

在搜尋結果內瀏覽 iFolder

向上及向下捲動以瀏覽搜尋結果並找出您想要管理的 iFolder。iFolder 名稱和擁有者的組合有助於辨識所需的 iFolder。

啟用、停用及刪除 iFolder


  1. 找出您要管理的 iFolder，然後選取 iFolder 旁的核取方塊。

  2. 選取要在 iFolder 上執行的動作。

    • 按一下「刪除」以刪除 iFolder。

    • 按一下「啟用」以啟用 iFolder。

      如此一來，使用者便可以登入並同步化 iFolder。

    • 按一下「停用」以停用 iFolder。

      當您在進行這項變更時，如果使用者已登入，則使用者的會期將會持續到使用者登出為止。規則會在下一次使用者嘗試登入帳戶時生效。若要鎖定使變更立即生效，您必須重新啟動 iFolder 伺服器的 Apache 服務，如此將會中斷所有使用中會期，包括使用者的會期。

注意：所有的 iFolder 預設都是啟用的。

建立新的 iFolder


  1. 在「iFolder 使用者」頁上按一下「新增」。

    此時「iFolder 建立」頁會開啟「iFolder」頁。

  2. 指定下列資訊，然後按一下「建立」。

名稱
為 iFolder 指定名稱，如 ifolder 1。

描述
iFolder 的簡短描述。

存取 iFolder 詳細資料頁


  1. 使用搜尋工具找出您要管理的 iFolder。

  2. 按一下 iFolder 連結，開啟「iFolder 詳細資料」頁的「iFolder」索引標籤。

    「iFolder 詳細資料」頁將會顯示 iFolder 詳細資料、擁有或共用 iFolder 的成員清單，以及目前 iFolder 的規則設定。
