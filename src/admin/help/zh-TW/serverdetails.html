檢視伺服器詳細資料

您可以檢視並編輯以下狀態資訊：

參數

描述

名稱 (僅檢視)

為伺服器指定的名稱。

類型

伺服器類型 (「主要」或「附屬」)。按一下「設為主伺服器」將此伺服器指定為主伺服器。

DNS 名稱

「iFolder 企業」伺服器的 DNS 名稱。

例如：192.168.1.1 或 svr1.domain.com。

公用 URL

對應至 iFolder 伺服器的公用 IP 位址。若要變更 IP 位址，請編輯給予的位址，然後按一下「儲存」以儲存您所做的變更。

私人 URL

對應至 iFolder 伺服器的私人 IP 位址。如此可讓 iFolder 領域內各伺服器間進行通訊。私人 URL 和公用 URL 可以相同。若要變更給予的 IP 位址，請輸入新的位址，然後按一下「儲存」以儲存您所做的變更。

主 URL (僅為從屬伺服器顯示)

對應至 iFolder 伺服器的 IP 位址。透過此位址，從屬伺服器會與此 iFolder 領域的主伺服器通訊。

若要變更 IP 位址，請編輯給予的位址，然後按一下「儲存」以儲存您所做的變更。

LDAP 詳細資料

您可以編輯下列 LDAP 相關資訊。按一下「儲存」以修改項目。按一下「取消」以取消您的修改。

<

參數

描述

開機時間

(僅能檢視)

顯示第一次同步的日期和時間。

狀態

報告目前 LDAP 同步引擎的狀態。

週期

(僅能檢視)

顯示已完成身份同步週期的總數。

身份同步化

在選取的週期內，從 LDAP 資訊更新選定 iFolder 領域內的 iFolder 使用者。

在「身份同步化」欄位內指定時間間隔 (以分鐘為單位)，然後按一下「立即同步化」以立即啟動 iFolder 使用者與 LDAP 使用者的同步作業。

刪除成員的寬限間隔

指定 iFolder 在特定使用者從 LDAP 刪除後，要從伺服器刪除使用者資訊的時間間隔。如果該使用者是不小心刪除的，則這個功能可協助您復原資料。


您可以編輯下列 LDAP 相關資訊。按一下「編輯」以開啟新頁面來修改項目。

參數

描述

LDAP 伺服器

顯示「LDAP 伺服器」的位址。

LDAP SSL

顯示是否已啟用「LDAP SSL」。這個功能也可讓您啟用或停用「LDAP SSL」連接。

代理使用者

顯示代理使用者的「可辨識名稱 (DN)」。例如，cn=SimiasProxy,o=novell

「iFolder 代理」使用者是一個用來存取 LDAP 伺服器的身份，以取回指定容器群組內的使用者清單，或定義於「iFolder LDAP」設定中的使用者。「iFolder 代理」使用者會在 iFolder 安裝期間建立。這個身份必須具有對 LDAP 目錄的「讀取」權限。

代理使用者密碼

 

iFolder 與 LDAP 伺服器同步化使用者時，此密碼是用來與 LDAP 伺服器驗證 iFolder 代理使用者之用。

LDAP 網路位置

「LDAP 網路位置」會列出所有的 LDAP 網路位置。iFolder 只會從列出的 LDAP 網路位置搜尋使用者。


資料儲存

「資料儲存」代表可在指定伺服器內跨多個卷冊 (裝載點) 延伸的 iFolder 儲存。根據預設，每部 iFolder 伺服器都具有預設的儲存，這個儲存不能停用。使用 Web 介面，您就可以加入及設定多個「資料儲存」的組態，包含各個載入平衡的 iFolder 資料。當使用者上載 iFolder 時，會以最大的可用空間檢查「資料儲存」，並將 iFolder 資料儲存在特定的「資料儲存」中，以平衡負載。您可以視需要加入多個「資料儲存」。擁有多個「資料儲存」可在大型部署中擴充資料儲存的容量，以符合企業層級的需求。

您可以檢視下列「資料儲存」資訊：

參數

描述

名稱

顯示您為「資料儲存」指定的獨特名稱。

完整路徑

顯示「資料儲存」的路徑，這個路徑即是裝上卷冊的路徑。這是您使用 Web 介面加入資料儲存時指定的資料路徑。

可用空間

顯示卷冊內可用的空間。

已啟用

顯示指定的「資料儲存」是否已啟用。您不能停用預設的「資料儲存」。

啟用或停用資料儲存

選取您要停用或啟用的「資料儲存」，然後個別按一下「停用」或「啟用」。當使用者上載 iFolder 時，會略過停用的「資料儲存」，同時會檢查儲存 iFolder 資料的最大可用空間。
