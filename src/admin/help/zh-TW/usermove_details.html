使用者重新供應詳細資料

使用這個介面可檢視使用者供應或重新供應詳細資料，和使用者擁有的 iFolder 詳細資料。

「使用者供應詳細資料」頁會為選定使用者的 iFolder 帳戶顯示下列使用者詳細資料。

參數

描述

使用者名稱

為使用者帳戶指定的使用者名稱，例如：jsmith 或 john.smith@example.com。

全名

使用者帳戶的名字和姓氏。

LDAP 網路位置

在 iFolder 領域內用於供應使用者之 LDAP 網路樹的網路位置。例如 cn= test, o=novell。

新主伺服器

顯示要為使用者供應的新伺服器。

已完成

以百分比顯示重新供應狀態。

重新供應狀態

顯示下列任何重新供應狀態。

  • 啟始化中
  • 已啟始化
  • 移動 iFolder 中
  • 重設主伺服器中
  • 完成中

iFolder 詳細資料

「使用者供應詳細資料」頁會為選定使用者的 iFolder 帳戶顯示下列 iFolder 詳細資料。

參數

描述

類型

一般的 iFolder

加密的 iFolder

共享 iFolder

名稱

指定給 iFolder 的名稱。

擁有者

選定 iFolder 之擁有者的使用者名稱。擁有者可以是個別使用者或 LDAPGroup。如果 LDAPGroup 為 iFolder 的擁有者，則所有此 LDAPGroup 的成員都擁有相同的存取權限。對於殘留下來的 iFolder，「iFolder 管理員」使用者會成為擁有者，直到 iFolder 可以重新指派或刪除為止。

按一下擁有者的使用者名稱連結，以檢視他 (或她) 的帳戶詳細資料。

iFolder 擁有者具有 iFolder 的「完全控制」權限。擁有者可管理成員資格和使用者的存取權限，並且可以移除任何成員的「完全控制」權限。

大小

iFolder 的大小，以 MB 為單位。

狀態

顯示下列任何 iFolder 移動狀態。

  • 啟始化中
  • 傳送中
  • 已完成
