管理使用者帳戶

使用此介面管理選定 iFolder 領域內的 iFolder 使用者。搜尋工具可協助您找出您要管理的 Novell iFolder® 3.7 或更新版本的使用者帳號。

注意：iFolder 使用者一詞代表的包括個別使用者和 LdapGroup。

瀏覽使用者

  1. 從「使用者」頁面，選取名稱準則 (名字、主目錄伺服器、姓氏、使用者名稱)。

  2. 選取過濾器標準 (「包含」、「開始於」、「結束以」)。

  3. 使用下列一或多種搜尋方法，然後按一下「搜尋」：

    • 在「搜尋使用者」欄位內輸入使用者或 LDAPGroup 名稱。

    • 在「搜尋伺使用者」欄位內輸入一或多個字母。

    • 在「搜尋使用者」欄位內輸入星號 (*) 以傳回系統上所有使用者和 LDAPGroup 的清單。

    • 讓「搜尋使用者」欄位保持為空白，以傳回系統上所有使用者及 LDAPGroup 的清單。

    在頁面完全重新整理之前，請勿按下頁面的任何地方，接著您便可以瀏覽、排序或管理「搜尋結果」報告內列出的使用者。

    注意：當您以次要管理員身分登入時，只能檢視您被指定為其次要管理員的使用者。此外，您可以根據您所擁有的權限來管理那些使用者。

在搜尋結果內找出使用者

向上及向下捲動以瀏覽搜尋結果並找出您想要管理的使用者。使用者名稱和名字及姓氏的組合有助於識別所需的使用者。

啟用或停用 iFolder 使用者

請遵循下列步驟進行：

  1. 找出您要管理的 iFolder 使用者，然後選取該使用者旁的核取方塊。

    • 按一下「啟用」以啟用使用者。

      如此一來，使用者便可以登入並同步化 iFolder。

    • 按一下「停用」以停用使用者。

      當您在進行這項變更時，如果使用者已登入，則使用者的會期將會持續到使用者登出為止。規則會在下一次使用者嘗試登入帳戶時生效。若要鎖定使變更立即生效，您必須重新啟動 iFolder 伺服器的 Apache 服務，如此將會中斷所有使用中會期，包括使用者的會期。

供應/重新供應 iFolder 使用者

使用這個介面可手動供應或重新供應 iFolder 使用者給 iFolder 領域中可用的不同伺服器。

若要供應/重新供應 iFolder 使用者，請執行下列任一步驟：

使用下拉式清單提供

  1. 找出您想要提供的 iFolder 使用者。

  2. 從每個使用者所擁有的首頁下拉式清單中選取 iFolder 伺服器。

  3. 按一下「儲存」。

使用「供應」按鈕提供

  1. 找出您想要管理的 iFolder 使用者。

  2. 選取使用者旁的核取方塊。

  3. 按一下「供應」以開啟新頁面，此處可供您將選定的使用者供應/重新供應給領域內的 iFolder 伺服器。

注意：使用者第一次登入時，如果未在使用者層級設定使用者的主伺服器，則設定給使用者所屬 LDAPGroup 的伺服器會自動配置給此使用者。

建立新的 iFolder 使用者

注意：您只可以建立非 LDAP 的使用者。只有在 iFolder 伺服器未與 LDAP 伺服器連接時，才能建立這些使用者。

  1. 在「iFolder 使用者」頁上按一下「新增」。

    「建立 iFolder 使用者」頁會開啟「使用者」索引標籤

  2. 指定下列資訊，然後按一下「建立」。

使用者名稱

存取使用者詳細資料頁

請遵循下列程序進行：

  1. 在「ifolder 使用者」頁上，使用搜尋工具找出您要管理其 ifolder 帳戶的使用者。

  2. 按一下使用者的名稱連結，開啟「詳細資料」頁的「使用者」索引標籤。
