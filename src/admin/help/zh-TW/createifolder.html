建立新的 iFolder

若要建立新的 iFolder，請依照以下程序進行：

  1. 在「iFolder 使用者」頁上按一下「新增」。

    此時「iFolder 建立」頁會開啟「iFolder」頁。

  2. 指定下列資訊，然後按一下「建立」。

    名稱
    為 iFolder 指定名稱，如 ifolder 1。

    描述
    iFolder 的簡短描述。

  3. 按一下「下一步」，選取 iFolder 的擁有者。

選取新增 iFolder 的擁有者


  1. 在「擁有者選擇」頁上，使用搜尋工具找出所需的使用者，以便指定擁有權。

  2. 按一下「下一步」選取新的 iFolder 的其他成員。

    3. 按一下「上一步」進行任何變更。

    按一下「取消」取消您所做的變更。

  3. 按一下「下一步」選取 iFolder 的其他成員。

選取新增 iFolder 的其他成員


  1. 在「其他成員選擇」頁上，使用搜尋工具找出所需的使用者。

  2. 從搜尋結果選取使用者。

    3. 按一下「上一步」進行任何變更。

  3. 按一下「建立」完成 iFolder 的建立程序。
