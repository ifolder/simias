Управление учетной записью пользователя

Используйте данный интерфейс для управления пользователями iFolder в выбранном домене iFolder. Найти необходимую учетную запись пользователя Novell iFolder® 3.7 или более поздней версии можно с помощью средства поиска.

ПРИМЕЧАНИЕ. Под пользователями iFolder подразумеваются как индивидуальные пользователи, так и группы LDAPGroup.

Просмотр пользователей

  1. На странице пользователей выберите критерий имени ("Имя", "Основной сервер", "Фамилия", "Имя пользователя").

  2. Выберите критерий фильтрации ("Начинается с", "Заканчивается на", "Содержит", "Равно").

  3. Воспользуйтесь одним или несколькими из перечисленных методов поиска, а затем щелкните "Поиск":

    • Введите имя пользователя или группы LDAPGroup в поле "Поиск пользователей".

    • Введите одну или несколько букв в поле "Поиск пользователей".

    • Введите символ звездочки (*) в поле "Поиск пользователей" для возврата к списку всех пользователей и групп LDAPGroup в системе.

    • Оставьте поле "Поиск пользователей" пустым для возврата к списку всех пользователей и групп LDAPGroup в системе.

    Не щелкайте какие-либо участки страницы до ее полного обновления; после этого вы сможете просмотреть или отсортировать пользователей в результатах поиска и выполнить другие действия над ними.

    Примечание. Выполнив вход как дополнительный администратор, вы сможете просматривать только тех пользователей, для которых вы назначены в качестве дополнительного администратора. Кроме того, управление пользователями осуществляется лишь в пределах прав, которыми вы наделены для такого управления.

Просмотр пользователей в результатах поиска

Выполняйте прокрутку вверх и вниз для просмотра результатов поиска и выбора нужного пользователя. Определить нужного пользователя можно по его имени пользователя, имени и фамилии.

Тип

обозначает пользователей с персонифицированным доступом к информации.

обозначает пользователей с измененным доступом к информации.

Имя
Имя пользователя, назначенное учетной записи пользователя, например "jsmith" или "john.smith@example.com". Щелкните ссылку "Имя пользователя", чтобы настроить для пользователя политики и папки iFolder.

Полное Имя
Имя и фамилия в соответствии с учетной записью пользователя.

Основной сервер
Сервер iFolder, к которому пользователь имеет персонифицированный доступ.

Разрешено
Состояние сервера.

Включение или отключение пользователя iFolder

Выполните следующие действия:

  1. Найдите нужного пользователя iFolder, а затем установите расположенный рядом с ним флажок.

    • Щелкните "Включить" для разрешения доступа к iFolder.

      Это позволяет пользователю входить в папки iFolder и выполнять их синхронизацию.

    • Щелкните "Запретить" для запрета доступа к iFolder.

      Сеанс пользователя, вошедшего в систему на момент установки данного значения, продолжается до тех пор, пока не будет выполнен выход. Данная политика вступает в силу при следующей попытке пользователя войти в систему с использованием своей учетной записи. Чтобы блокировка вступила в силу немедленно, необходимо перезапустить сервисы Apache для сервера iFolder. При этом выполняется отключение всех активных сеансов, включая сеансы пользователей.

Персонификация и изменение доступа для пользователей iFolder

Используйте этот интерфейс для персонификации и изменения доступа пользователей iFolder к различным серверам, доступным в домене iFolder.

Чтобы персонифицировать и изменить доступ пользователей iFolder к информации, выполните одно из следующих действий:

Персонификация доступа к информации с помощью раскрывающегося списка

  1. Найдите пользователей iFolder, для которых необходимо персонифицировать доступ.

  2. Выберите сервер iFolder в раскрывающемся списке "Домашний" рядом с каждым пользователем.

  3. Щелкните кнопку "Сохранить".

Персонификация доступа к информации с помощью кнопки "Персонификация доступа".

  1. Найдите необходимых пользователей iFolder.

  2. Установите флажок для каждого пользователя.

  3. Щелкните "Персонификация доступа", чтобы открыть новую страницу, где можно персонифицировать или изменить доступ для выбранных пользователей к серверу iFolder в домене.

ПРИМЕЧАНИЕ. При первом входе пользователя, для которого на уровне пользователя не установлен домашний сервер, этому пользователю автоматически назначается сервер, назначенный группе LDAPGroup, к которой принадлежит этот пользователь.

Создание нового пользователя iFolder

ПРИМЕЧАНИЕ. Можно создавать только пользователей без поддержки LDAP. Их можно создавать только в том случае, если сервер iFolder не подключен к серверу LDAP.

  1. На странице "Пользователи iFolder" щелкните "Создать".

    На вкладке "Пользователи" открывается страница "Пользователь iFolder".

  2. Укажите следующую информацию и щелкните "Создать".

Имя пользователя
Назначьте имя пользователя для учетной записи пользователя, например "jsmith" или "john.smith@example.com".

Имя
Имя пользователя.

Фамилия
Фамилия пользователя.

Полное Имя
Имя и фамилия в соответствии с учетной записью пользователя.

Пароль
Назначьте пароль учетной записи пользователя.

Повторный ввод пароля
Введите пароль еще раз, чтобы подтвердить его.

Доступ к странице "Информация о пользователе"

Следуйте описанной ниже процедуре:

  1. На странице пользователя iFolder с помощью средства поиска найдите пользователя с нужной учетной записью iFolder.

  2. Щелкните ссылку с именем пользователя, чтобы открыть на вкладке "Пользователи" страницу "Информация о пользователе".
