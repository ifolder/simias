Подробная информация об изменении доступа для пользователя

В этом интерфейсе можно просматривать подробную информацию о персонификации и изменении доступа пользователя и о папке iFolder, владельцем которой он является.

На странице информации об изменении доступа пользователя отображается следующая информация о пользователе для выбранной учетной записи iFolder.

Параметр

Описание

Имя пользователя

Имя пользователя, назначенное учетной записи пользователя, например "jsmith" или "john.smith@example.com".

Полное Имя

Имя и фамилия в соответствии с учетной записью пользователя.

Контекст LDAP

Контекст дерева LDAP используется для предоставления пользователям персонифицированного доступа к домену iFolder. Например, cn= test, o=novell.

Новый основной

Новый сервер для персонификации доступа пользователя.

Завершен

Состояние изменения доступа в процентах.

Состояние изменения доступа

Отображение одного из следующих состояний изменения доступа:

  • Инициализация
  • Инициализирован
  • Перемещение папки iFolder
  • Сброс основного сервера
  • Завершение

Подробная информация о iFolder

На странице информации об изменении доступа пользователя отображается следующая информация о папке iFolder для выбранной учетной записи iFolder.

Параметр

Описание

Тип

Обычная папка iFolder.

Зашифрованная папка iFolder.

Совместно используемая папка iFolder

Имя

Имя, назначенное папке iFolder.

Владелец

Имя пользователя владельца выбранной папки iFolder. Владельцем может быть индивидуальный пользователь или группа LDAPGroup. Если группа LDAPGroup является владельцем папки iFolder, у всех ее членов одинаковые права доступа. Владельцем папок iFolder с нарушенными ассоциациями назначается администратор iFolder до тех пор, пока они не будут переназначены или удалены.

Щелкните ссылку с именем пользователя владельца для просмотра подробной информации о его учетной записи.

Владелец папки iFolder обладает правом доступа "Полный контроль" для этой папки. Владелец управляет членством и правами доступа пользователей, а также может лишить любого из членов права "Полный контроль".

Размер

Размер папки iFolder в мегабайтах.

Состояние

Отображение одного из следующих состояний перемещения папки iFolder.

  • Инициализация
  • Передача
  • Завершен
