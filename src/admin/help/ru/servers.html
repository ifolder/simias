Управление серверами iFolder

Используйте данный интерфейс для управления серверами iFolder и настройки нескольких серверов. Для поиска нужного сервера можно воспользоваться средством поиска.

Поиск серверов iFolder

  1. Выберите критерий фильтрации ("Начинается с", "Заканчивается на", "Содержит", "Равно").

  2. Воспользуйтесь одним или несколькими из перечисленных методов поиска, а затем щелкните "Поиск":

    • Введите имя сервера в поле "Поиск серверов".

    • Введите одну или несколько букв в поле "Поиск серверов".

    • Введите символ звездочки (*) в поле "Поиск серверов" для возврата к списку всех серверов в системе.

    • Оставьте поле "Поиск серверов" пустым для возврата к списку всех серверов в системе.

    Не щелкайте какие-либо участки страницы до ее полного обновления; после этого Вы сможете просмотреть, отсортировать и использовать серверы, попавшие в результаты поиска.

    Выполняйте прокрутку вверх и вниз для просмотра результатов поиска и выбора нужного сервера.

Доступ к странице "Информация о сервере"

Следуйте описанной ниже процедуре:

  1. На странице "Сервер" выполните поиск сервера, задав критерий поиска в средстве поиска.

  2. Щелкните ссылку с именем сервера, чтобы открыть на странице "Серверы" страницу "Информация о сервере".

    Важно. На данной странице отображается информация только для сервера, на который вошел пользователь. Чтобы отобразить информацию о другом сервере, необходимо войти в консоль веб-администрирования соответствующего сервера.
