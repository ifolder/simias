Запуск приложения веб-администрирования

С помощью консоли веб-администрирования можно управлять серверами, параметрами политик, пользователями и папками iFolder в конкретном домене iFolder 3.7 и более поздней версии. Можно предоставлять доступ к домену iFolder пользователям LDAP и управлять серверами iFolder в домене iFolder. С помощью приложения веб-администрирования можно также включить принудительное использование шифрования. В консоли веб-администрирования появится возможность создания расширенных отчетов, содержащих информацию об общем состоянии сервера, отчеты об учетных записях iFolder и сведения об учетных записях пользователей.

ПРИМЕЧАНИЕ. Нельзя управлять серверами Novell iFolder 2.1x с подключаемым модулем Novell iFolder 3.7 и более поздней версии для iManager.

Чтобы получить доступ к странице "Управление iFolder", выполните следующие действия:

  1. Разверните в iManager функцию iFolder

  2. Щелкните "Запустить веб-администрирование", чтобы открыть страницу выбора сервера iFolder.

  3. Укажите DNS-имя или IP-адрес сервера iFolder Enterprise.

  4. Выполните одно из следующих действий.

    • Если Вы зарегистрировались в iManager с тем же именем пользователя, которое используется для регистрации в качестве администратора iFolder в приложении веб-администрирования, выберите "Аутентификация с использованием текущих учетных данных iManager".

    • Если имя пользователя, используемое при регистрации в iManager, отличается от используемого для регистрации в качестве администратора iFolder в приложении веб-администрирования, не устанавливайте флажок "Аутентификация с использованием текущих учетных данных iManager". На странице регистрации приложения веб-администрирования укажите пароль и имя администратора iFolder.

  5. Нажмите кнопку "ОК".

В зависимости от выбранных ранее значений выполняется переход к соответствующей странице приложения веб-администрирования.
