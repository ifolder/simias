Доступ к папкам и файлам в папках iFolder

Используйте данный интерфейс для доступа к файлам и папкам iFolder, доступным для учетной записи. Для каждой папки iFolder можно просматривать список файлов и папок, отправлять и загружать файлы, создавать папки, а также удалять файлы и папки. Чтобы создавать, удалять или совместно использовать папки iFolder, необходимо применять клиент iFolder для компьютера Linux*, Windows* или Macintosh*.

Вход и выход

Если во время работы с папками iFolder заканчивается срок действия системного соединения, веб-клиент возвращает пользователя на страницу входа. Можно повторно войти и продолжить работу, не закрывая веб-навигатор.

Для выхода щелкните "Выход" в верхней правой области заголовка и закройте веб-навигатор. Если выход не выполнить, сеанс остается открытым до тех пор, пока не наступит тайм-аут сеанса, что может представлять угрозу системы защиты.

Просмотр папок iFolder

Страница iFolder содержит список всех папок iFolder, доступных для текущей учетной записи. Доступные папки iFolder включают папки, владельцем которых является пользователь, а также папки, предоставленные ему другими пользователями для совместного использования. С помощью веб-клиента можно получить доступ к совместно используемым папкам iFolder даже в том случае, если клиент iFolder не установлен на локальном компьютере. Столбец "Владелец" определяет имя человека, которому принадлежит папка iFolder, чтобы облегчить поиск папки iFolder.

Щелкните имя папки iFolder для просмотра списка содержащихся в ней папок и файлов, отправки и загрузки файлов, создания папок, а также удаления файлов и папок.

Управление папками и файлами в папках iFolder

В папке iFolder можно выполнять следующие задачи:

Просмотр списка файлов и папок
На странице iFolder файлы и папки отображаются в алфавитном порядке. Перейдите к необходимой папке или к необходимому файлу.

Навигация по каталогу
Щелкните имя папки для открытия папки и просмотра списка ее содержимого. Для возврата к каталогу более высокого уровня щелкните значок "Вверх" на панели задач.

Открыть или сохранить файл
Щелкните имя файла , который необходимо открыть или сохранить.

  • Щелкните "Открыть" для просмотра содержимого файлов. Для открытия файла на компьютере должно быть установлено соответствующее приложение. Если Вы планируете работать с файлом, загрузите файл, измените его, а затем отправьте файл.

  • Щелкните "Сохранить" для сохранения файла на локальном жестком диске или назначенном сетевом диске. Перейдите к расположению, куда необходимо поместить файл, а затем щелкните "Сохранить". Файл загружается на компьютер.

Создание Новой Папки
В поле "Новая папка" укажите имя новой папки и щелкните "Создать". Имена папок зависят от регистра. После обновления страницы файл отобразится в алфавитном порядке.

Удаление папок и файлов
Установите флажки напротив файлов и папок, которые необходимо удалить, и щелкните "Удалить", чтобы подтвердить удаление.

Отправка файла
В поле "Отправка файла" щелкните "Просмотр", найдите локальную папку, которую необходимо отправить, а затем щелкните "Открыть" > "Отправить". После обновления страницы файл отобразится в алфавитном порядке.

 

Символ товарного знака (®, TM и т. д.) обозначает товарный знак Novell. Символ звездочки (*) обозначает товарный знак независимого производителя. Для получения дополнительной информации см. раздел Юридическая информация.
