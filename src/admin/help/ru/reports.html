Отчеты

Используйте данный интерфейс для включения функции создания отчетов и создания отчетов для iFolder и Каталогов. Создание отчетов выполняется с выбранной частотой.

  1. Выберите "Включить формирование отчетов" для включения создания отчетов.

  2. Выберите частоту ("Ежедневно", "Еженедельно", "Ежемесячно").

  3. Выберите время создания отчета.

  4. Выберите место для сохранения отчетов из предложенных вариантов ("Папка iFolder отчетов", "Каталог отчетов").

    Если выбрать параметр "Каталог отчета", то созданный отчет автоматически сохраняется в каталоге для отчетов, имеющем путь по умолчанию.

    Например, <путь>/simias/report.

    Если выбрать параметр "Создать отчет iFolder", то созданный отчет сохраняется в папке iFolder для отчетов.

    Папка iFolder для отчетов, имеющая имя "< имя домена>-<имя сервера>-Reports", создается автоматически для каждого сервера данного администратора.

  5. Выберите формат создаваемого отчета.

  6. Щелкните "Сохранить" для сохранения изменений.

    7. Щелкните "Отмена" для отмены изменений.
