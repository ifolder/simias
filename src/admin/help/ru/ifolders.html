Управление папками iFolder

Используйте данный интерфейс для управления владением, политиками и членством выбранной папки iFolder для учетной записи пользователя iFolder в данном домене iFolder. Для поиска нужной учетной записи можно воспользоваться средством поиска.

Примечание. Войдя как дополнительный администратор, вы сможете просматривать только те папки iFolder, владельцами которых являются пользователи, для которых вы назначены в качестве дополнительного администратора. Кроме того, управление папками iFolder осуществляется лишь в пределах прав, которыми вы наделены для такого управления.

Список Сервер

Воспользуйтесь одним или несколькими из перечисленных методов поиска, а затем щелкните "Поиск":

  • Выберите критерий фильтрации "Равно", а затем введите имя искомой папки iFolder в поле "Поиск папок iFolder".

  • Выберите критерии фильтрации имени папки iFolder ("Начинается с", "Заканчивается на", "Содержит", "Равно"), а затем введите одну или несколько букв в поле "Поиск папок iFolder".

  • Введите символ звездочки (*) в поле "Поиск папок iFolder" для возврата к списку всех папок iFolder в системе.

  • Оставьте поле "Поиск папок iFolder" пустым для возврата к списку всех папок iFolder в системе.

Не щелкайте какие-либо участки страницы до ее полного обновления; после этого Вы сможете просмотреть, отсортировать и использовать папки iFolder, попавшие в результаты поиска.

Просмотр папок iFolder в результатах поиска

Выполняйте прокрутку вверх и вниз для просмотра результатов поиска и выбора нужной папки iFolder. Определить нужную папку iFolder можно по имени и владельцу.

Включение, выключение и удаление папки iFolder


  1. Найдите нужную папку iFolder, а затем установите расположенный рядом с ней флажок.

  2. Выберите действие, которое необходимо совершить над папкой iFolder:

    • Щелкните "Удалить" для удаления папки iFolder.

    • Щелкните "Включить" для разрешения доступа к iFolder.

      Это позволяет пользователю входить в папки iFolder и выполнять их синхронизацию.

    • Щелкните "Запретить" для запрета доступа к папке iFolder.

      Сеанс пользователя, вошедшего в систему на момент установки данного значения, продолжается до тех пор, пока не будет выполнен выход. Данная политика вступает в силу при следующей попытке пользователя войти в систему с использованием своей учетной записи. Чтобы блокировка вступила в силу немедленно, необходимо перезапустить сервисы Apache для сервера iFolder. При этом выполняется отключение всех активных сеансов, включая сеансы пользователей.

ПРИМЕЧАНИЕ. По умолчанию доступ ко всем папкам iFolder разрешен.

Создание новой папки iFolder


  1. На странице "Пользователи iFolder" щелкните "Создать".

    На странице папок iFolder открывается страница создания iFolder.

  2. Укажите следующую информацию и щелкните "Создать".

Имя
Назначьте папке iFolder имя, например "ifolder 1".

Описание
Введите краткое описание папки iFolder.

Доступ к странице "Информация iFolder"


  1. Найдите необходимую папку iFolder с помощью средства поиска.

  2. Щелкните ссылку папки iFolder, чтобы открыть страницу "Информация iFolder" на вкладке iFolder.

    На странице "Информация iFolder" отображается подробная информация о папке iFolder, список членов, обладающих правами владения или совместного использования, и параметры политик текущей папки iFolder.
