Создание новой папки iFolder

Для создания новой папки iFolder воспользуйтесь описанной ниже процедурой:

  1. На странице "Пользователи iFolder" щелкните "Создать".

    На странице папок iFolder открывается страница создания iFolder.

  2. Укажите следующую информацию и щелкните "Создать".

    Имя
    Назначьте папке iFolder имя, например "ifolder 1".

    Описание
    Введите краткое описание папки iFolder.

  3. Щелкните "Далее" для выбора владельца папки iFolder.

Выбор владельца для новой папки iFolder


  1. На странице выбора владельца с помощью средства поиска найдите пользователя, которому необходимо назначить владение.

  2. Щелкните "Далее" для выбора дополнительных членов для новой папки iFolder.

    3. Щелкните "Назад", чтобы внести изменения.

    Щелкните "Отмена", чтобы отменить внесенные изменения.

  3. Щелкните "Далее" для выбора дополнительных членов для папки iFolder.

Выбор дополнительных членов для новой папки iFolder


  1. На странице выбора дополнительных членов с помощью средства поиска найдите необходимых пользователей.

  2. Выберите пользователей в результатах поиска.

    3. Щелкните "Назад", чтобы внести изменения.

  3. Щелкните "Создать" для завершения создания папки iFolder.
