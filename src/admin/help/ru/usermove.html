Состояние изменения доступа к системе iFolder

В этом интерфейсе можно просматривать состояние изменения доступа для текущего домена iFolder.

На этой странице отображается следующая подробная информация:

Параметр

Описание

Тип

Определение пользователей с персонифицированным доступом или измененным доступом .

Имя пользователя

Имя пользователя, назначенное каждой учетной записи пользователя, например "jsmith" или "john.smith@example.com".

Текущий основной

Основной сервер, назначенный пользователю с персонифицированным доступом.

Новый основной

Новый сервер для персонификации доступа пользователя.

Завершен

Состояние изменения доступа в процентах.

Состояние изменения доступа

Отображение одного из следующих состояний изменения доступа:

  • Инициализация
  • Ошибка при инициализации
  • Инициализирован
  • Перемещение папки iFolder
  • Сброс основного сервера
  • Завершение
Удалить Щелкните эту кнопку, чтобы удалить неполные запросы на удаление пользователя. Эта кнопка активна, в процессе перемещения пользователя на этапе инициализации возникает проблема.
