Managing Web Admin Console

Use this Web Admin interface to manage iFolder users, LdapGroups, the iFolder system, servers, iFolders, and the iFolder statistics report.

Users Tab

  1. Log in to the Web Admin using Web Admin credentials.

    NOTE: The term iFolder users refers to both individual users and LdapGroups.

    The Users page displays the user’s type (admin user or user), username , user's full name (if available), the server to which the user is provisioned, and the user status (Enabled or Disabled).

  2. Use the search functionality to locate the user whose iFolder account you want to manage.

  3. Click the user's name link to open the User Details page.

    The User Details page opens to the Users tab, which displays the user details, iFolders owned, and shared and policy settings for this particular user account.

iFolders Tab


  1. On the Web Admin console, click the iFolders tab.

    iFolders page displays the iFolder type (admin user or user), iFolder name, iFolder owner, members, the date when iFolder was last modified and the iFolder size.

  2. Use the search tool to locate the iFolder you want to manage using the search criterion.

  3. Click the iFolder's link to open the iFolder Details page to the iFolder tab.

    The iFolder Details page displays the iFolder details, list of members who own or share the iFolders and policy settings for the current iFolder.

Systems Tab


  1. On the Web Admin console, click the Systems tab.

    The Systems page displays the system settings and list of iFolder administrators. With System page, you can manage the policy settings for the whole system.

  2. Locate the iFolder Administrator you want to manage. You can add or delete iFolder Administrator.

    You can also manage the policy settings for the Admin user.

  3. Click the Admin user's name link to open the User Details page.

    The User Details page opens to the Users tab, which displays the user details, iFolders owned, and shared and policy settings for this particular user account.

Servers Tab


  1. On the Web Admin console, click the Servers tab.

    The Servers page displays the server type (Master or Slave), server name, public IP Address, private IP address, and status of the selected server.

  2. Use the search tool to locate the Server you want to manage.

  3. Click the Server name link to open the Servers Details page.

    The Server Details page displays server details, server type (Master or Slave), DNS name, server status, server logs, server reports, log level, LDAP details and iFolder Data Store.

Reports Tab


  1. On the Web Admin console, click Reports.

  2. Configure reporting according to the frequency and time schedule you want, then generate the output as desired.

    You can save the the generated report in the directory specified on Reports page.
