Reports

Use this interface to enable reporting and generate reports for iFolder and Directories. It generate reports based on the frequency you select.

  1. Select Enable reporting to enable reporting.

  2. Select the frequency from the given options (Daily, Weekly, Monthly).

  3. Select the time when you want to generate the report .

  4. Select the location where you want to save reports from the given options (Report iFolder, Report Directory)

    If you select Report Directory, the generated report is automatically saved to the report directory under the default data path.

    For example, <data path>/simias/report.

    If you select Report iFolder, the generated report is saved to the Report iFolder.

    The Report iFolder, named < domain name>-<server name>-Reports is automatically created for each server for the admin user.

  5. Select the format for generating the report.

  6. Click Save to save the settings.

    7. Click Cancel to cancel the settings.
