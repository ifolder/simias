Use this interface to create a new iFolder, select an owner and add additional members to it.

Create a New iFolder

To create a new iFolder, follow the procedure given below:

  1. On the iFolder Users page, click New.

    The iFolder creation page opens to the iFolders page.

  2. Specify the following information and then click Create.

    Name
    Assign a name to the iFolder such as ifolder 1.

    Description
    A short description about the iFolder.

  3. Click Next to select an owner for the iFolder.

Selecting Owner For the New iFolder


  1. On the Owner selection page, use the search tool to locate the desired user whom you want to assign the ownership.

  2. Click Next to select additional members to the new iFolder.

    3. Click Back to make any changes.

    Click Cancel to cancel the changes you made.

  3. Click Next to select additional members to the iFolder.

Selecting Additional Members to The New iFolder


  1. On the Additional member selection page, use the search tool to locate the desired users.

  2. Select the users from the search result.

    3. Click Back to make any changes.

  3. Click Create to complete the iFolder creation.
