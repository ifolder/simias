Launching Web Admin

Web Admin console will let you manage servers, policy settings, iFolder users, and ifolders in a given iFolder 3.7 or later domain.You can enable users to iFolder domain from LDAP users and manage iFolder servers participating in the iFolder domain. With Web admin, you can also enforce encryption on users.Web Admin console will have the capability to produce enhanced reports that include overall server status, iFolder account reports and user account information.
