Add Data Store

To add a new Data Store,

  1. Specify the following information:

    2. Name
    Assign a unique name to the Data Store, such as ifolder-store.

    Path
    Enter the path where the new volume is mounted.
    If it is a remote volume (CIFS, NFS, AFP), then ensure that the volume is mounted on every restart for proper functioning and load balancing.
    You need to check the permissions of the path specified, and change the ownership and group to apacheuser (wwwrun) and apachegroup (www) respectively. Unless you have set the permission for the directory on to which the volume is mounted, you cannot create or sync iFolders on this volume.
