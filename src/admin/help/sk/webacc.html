Spustenie služby Web Admin

Konzola Web Admin vám umožní spravovať servery, nastavenia politiky, používateľov priečinka iFolder a priečinky iFolder v doméne iFolder verzie 3.7 a novšej. Používateľom služby LDAP môžete povoliť doménu iFolder a spravovať servery iFolder, ktoré sú zapojené do domény iFolder. So konzolou Web Admin môžete tiež posilniť šifrovanie používateľov. Konzola Web Admin bude schopná vyrobiť vylepšené zostavy s celkovým stavom servera, zostavy pre konto priečinka iFolder and informácie o konte používateľa.

Poznámka: Nemôžete spravovať servery s priečinkami Novell iFolder 2.1x s aplikáciou Novell iFolder 3.7 a novším doplnkom pre správcu iManager.

Prístup k stránke Správa priečinka iFolder:

  1. V správcovi iManager, rozbaľte ponuku rola priečinka iFolder.

  2. Kliknutím na položku Spustiť konzolu Web Admin otvoríte stránku pre výber serverov iFolder.

  3. Zadajte názov DNS alebo adresu IP servera iFolder Enterprise.

  4. Urobte niektorý z nasledujúcich krokov:

    • Ak ste prihlásený k správcovi iManager rovnakým menom ako je meno používateľa správcu aplikácie iFolder v konzole Web Admin, vyberte možnosť Overenie pomocou aktuálnych poverení správcu iManager.

    • Ak ste prihlásený k správcovi iManager s odlišným menom ako je meno používateľa správcu aplikácie iFolder v konzole Web Admin, začiarkávacie políčko Overenie pomocou aktuálnych poverení správcu iManager nechajte voľné. Na prihlasovacej stránke konzole Web admin, zadajte meno a heslo používateľa správcu aplikácie iFolder.

  5. Kliknite na tlačidlo OK.

Na základe vášho výberu budete nasmerovaný na stránku používatelia konzole Web Admin.
