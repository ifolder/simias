访问 iFolder 中的文件夹和文件

请通过此界面访问当前帐户中可供您使用的 iFolder 文件夹和文件。对于每个 iFolder，您可以查看它的文件夹和文件列表、上载和下载文件、创建文件夹、删除文件夹和文件。要创建、删除或共享 iFolder，您必须借助用于 Linux*、Windows* 或 Macintosh* 计算机的 iFolder 客户程序。

此帮助讨论以下内容：

登录和注销
浏览 iFolder
管理 iFolder 中的文件夹和文件

登录和注销

当您在 iFolder 中工作时，如果系统连接超时，万维网访问会为您显示登录页。您可以再次登录以继续工作，而无需关闭浏览器。

您可以单击右上角标题区域的“注销”进行注销，然后关闭浏览器。如果不注销，您的会话将在超时前一直保持打开状态，这可能带来安全隐患。

浏览 iFolder

iFolder 页列出了当前帐户中可供您使用的所有 iFolder。可用 iFolder 包括您拥有的 iFolder 和其他用户与您共享的 iFolder。即使您未在本地计算机上设置 iFolder，也可以通过万维网访问来访问共享 iFolder。“拥有者”栏标识 iFolder 拥有者的名称，以帮助您找到需要的 iFolder。

单击 iFolder 的名称查看它的文件夹和文件列表、上载和下载文件、创建文件夹、删除文件夹和文件。

管理 iFolder 中的文件夹和文件

在 iFolder 中，您可以执行以下任务：

查看文件夹和文件列表
iFolder 页按字母顺序列出文件夹和文件。您可以滚动该列表以查找自己感兴趣的目录或文件。

浏览目录
单击文件夹的名称 以打开文件夹并查看其内容列表。要返回更高一级的目录，请单击任务栏中的“向上”文件夹图标 。

打开或保存文件
单击要打开或保存的文件的名称 。

  • 单击“打开”以查看文件的内容。为了打开文件，您的计算机上必须装有兼容的应用程序。如果要对文件进行处理，请下载该文件，进行修改，然后上载该文件。

  • 单击“保存”将该文件保存到本地硬盘驱动器或映射的网络驱动器上。导航到要存放该文件的位置，然后单击“保存”。此文件即会被下载到您的计算机。

创建新文件夹
在“新建文件夹”字段中，指定新文件夹的名称，然后单击“创建”。文件夹名称区分大小写。刷新该页之后，文件夹将按字母顺序显示在列表中。

删除文件夹或文件
选择要删除的文件夹或文件旁边的复选框，单击“删除”，然后确认删除。

上载文件
在“上载文件”字段中，单击“浏览”，定位待上载文件所在的本地文件夹，然后单击“打开”>“上载”。刷新该页之后，文件将按字母顺序显示在列表中。
