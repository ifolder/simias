Login à Web Access

Loguez-vous à Novell iFolder® 3.x Web Access pour obtenir l'accès aux dossiers et fichiers de tous iFolders disponibles dans votre compte iFolder 3.x. Cela inclut les iFolders auxquels vous participez, mais que vous n'avez pas configurés sur un ordinateur local. (Web Access ne permet pas d'accéder à des fichiers dans votre compte Novell iFolder 2.1x.) Votre nom d'utilisateur et votre mot de passe sont authentifiés auprès des services d'annuaire LDAP.

Vous pouvez également choisir la langue que vous voulez utiliser parmi celles qui sont prises en charge. Il s'agit généralement de la même langue que le code de langue du serveur d'entreprise iFolder et de votre ordinateur local. N'oubliez pas de configurer le paramètre de langues de votre navigateur pour qu'il prenne en charge la langue voulue.

  1. Spécifiez le nom d'utilisateur et le mot de passe pour le compte iFolder auquel vous voulez accéder.

    Les noms d'utilisateur et les mots de passe respectent la casse.

  2. Utilisez la liste déroulante Langues pour spécifier la langue à utiliser pour accéder à vos fichiers.

  3. Cliquez sur Se loguer.

 

Un symbole de marque commerciale (®, TM, etc.) indique qu'il s'agit d'une marque Novell. Un astérisque (*) indique qu'il s'agit d'une marque commerciale de fabricant tiers. Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
