Zugreifen auf Ordner und Dateien in Ihren iFolder-Ordnern

Über diese Schnittstelle können Sie auf Ordner und Dateien in den iFolder-Ordnern zugreifen, die in diesem Konto für Sie verfügbar sind. Für jeden iFolder-Ordner können Sie eine Liste der enthaltenen Ordner und Dateien anzeigen, Dateien hoch- oder herunterladen, Ordner erstellen oder Dateien und Ordner löschen. Um iFolder-Ordner zu erstellen, zu löschen oder freizugeben, müssen Sie den iFolder-Client für Ihren Linux*-, Windows*- oder Macintosh*-Computer verwenden.

In diesem Hilfethema wird Folgendes behandelt:

Anmelden und Abmelden
Durchsuchen von iFolder-Ordnern
Verwalten von Ordnern und Dateien in einem iFolder-Ordner

Anmeldung und Abmeldung

Falls für die Systemverbindung eine Zeitüberschreitung erfolgt, während Sie Ihre iFolder-Ordner bearbeiten, werden Sie durch Web Access auf die Anmeldeseite zurückgeleitet. Sie können sich erneut anmelden, um mit der Bearbeitung fortzufahren, ohne Ihren Browser zu schließen.

Sie können sich abmelden, indem Sie im Header-Bereich oben rechts auf "Abmelden" klicken und anschließend Ihren Browser schließen. Wenn Sie sich nicht abmelden, bleibt die Sitzung bis zur Zeitüberschreitung der Sitzung geöffnet. Dies kann ein Sicherheitsrisiko darstellen.

Durchsuchen von iFolder-Ordnern

Auf der Seite "iFolder-Ordner" werden alle für Sie im aktuellen Konto verfügbaren iFolder-Ordner angezeigt. Zu den verfügbaren iFolder-Ordnern zählen die Ordner, von denen Sie selbst Eigentümer sind, und die Ordner, die andere Benutzer für Sie freigegeben haben. Sie können durch Webzugriff auf freigegebene Ordner zugreifen, selbst wenn Sie den iFolder-Ordner nicht lokal auf Ihrem Computer eingerichtet haben. In der Spalte "Eigentümer" wird der Name der Person angegeben, die Eigentümer des iFolder-Ordners ist. Damit wird Ihnen das Auffinden des gesuchten iFolder-Ordners erleichtert.

Klicken Sie auf den iFolder-Ordnernamen, um eine Liste der enthaltenen Ordner und Dateien anzuzeigen, Dateien hoch- oder herunterzuladen, Ordner zu erstellen oder Dateien und Ordner zu löschen.

Verwalten von Ordnern und Dateien in einem iFolder-Ordner

In einem iFolder-Ordner können Sie die folgenden Aufgaben ausführen:

Liste von Ordnern und Dateien anzeigen
Auf der Seite "iFolder-Ordner" werden Ordner und Dateien alphabetisch aufgelistet. Blättern Sie in der Liste, um das gewünschte Verzeichnis bzw. die gewünschte Datei zu finden.

In einem Verzeichnis navigieren
Klicken Sie auf den Namen des Ordners , um den Ordner zu öffnen und den Ordnerinhalt aufzulisten. Um zu einer höheren Ebene in der Verzeichnisstruktur zu wechseln, klicken Sie in der Symbolleiste auf das Symbol .

Datei öffnen oder speichern
Klicken Sie auf den Namen der Datei , die Sie öffnen oder speichern möchten.

  • Klicken Sie auf "Öffnen", um den Inhalt der Datei anzuzeigen. Auf Ihrem Computer muss eine kompatible Anwendung installiert sein, um die Datei zu öffnen. Wenn Sie in der Datei arbeiten möchten, laden Sie sie herunter, ändern Sie die Datei und laden Sie sie anschließen erneut hoch.

  • Klicken Sie auf "Speichern", um die Datei auf Ihrem lokalen Festplattenlaufwerk oder auf einem zugeordneten Netzlaufwerk zu speichern. Navigieren Sie zu der Position, an der Sie die Datei speichern möchten, und klicken Sie auf "Speichern". Die Datei wird auf Ihren Computer geladen.

Neuen Ordner erstellen
Im Feld "Neuer Ordner" geben Sie den Namen des neuen Ordners an und klicken Sie anschließend auf "Erstellen". Bei Ordnernamen muss die Groß- und Kleinschreibung berücksichtigt werden. Wenn die Seite aktualisiert wird, werden die Ordner in alphabetischer Reihenfolge aufgelistet.

Ordner oder Dateien löschen
Aktivieren Sie das Kontrollkästchen neben den Ordnern bzw. den Dateien, die Sie löschen möchten, und klicken Sie anschließend auf "Löschen". Bestätigen Sie den Löschvorgang.

Datei hochladen
Klicken Sie im Feld "Datei heraufladen" auf "Durchsuchen", um einen lokalen Ordner auszuwählen, den Sie hochladen möchten. Klicken Sie dann auf "Öffnen" > "Heraufladen". Wenn die Seite aktualisiert wird, wird die Datei entsprechend ihrem Dateinamen in einer alphabetisch sortierten Liste angezeigt.
