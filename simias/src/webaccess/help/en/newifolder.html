Create a New iFolder

Perform the following steps to create a new iFolder:

  1. Click the New iFolder link listed under Actions at the left side of the Home page.

  2. In the new page that opens, specify the following information:

    3. Name
    Assign a name to the iFolder, such as myfolder.

    Description
    A short description of the iFolder.

    Type
    select the type of the iFolder that you want to create.
    Encryption Enabled
    Select this option to encrypt your iFolder and the files within it.

    IMPORTANT: This option is not enabled until your Admin set your encryption policy to On.

    Sharable
    Select Sharable to share your iFolder with other iFolder accounts.

    IMPORTANT: Shared iFolders cannot be encrypted and encrypted ones cannot be shared.

  3. Select the Recovery agent from the drop-down list.

    Note: This option will not be available if your iFolder Administrator does not select Enable Recovery agent option while configuring the iFolder server.

    The Recovery agent can be a system, organization, or a person independent of the iFolder system that where you can store the key to recover your secret passphrase.

    IMPORTANT: Unless you have set your Recovery agent, the encrypted data cannot be recovered in the unfortunate case that you lost the passphrase.

  4. Type the passphrase in the Enter Passphrase field.

  5. Click Create to complete the new iFolder creation.

    Click Cancel to cancel the operation.
