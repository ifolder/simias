存取 iFolder 中的資料夾和檔案

您可以使用此介面存取此帳戶下可用之 iFolder 中的資料夾和檔案。對於每個 iFolder，您都可以檢視其資料夾和檔案的清單、上載及下載檔案、建立資料夾，以及刪除資料夾和檔案。若要建立、刪除或共用 iFolder，您必須使用適用於 Linux*、Windows* 或 Macintosh* 電腦的「iFolder 用戶端」。

本項說明討論下列事項：

登入與登出
瀏覽 iFolder
管理 iFolder 中的資料夾和檔案

登入與登出

當您正在使用 iFolder 時，如果系統連線逾時，Web 存取會讓您返回登入頁面。您可以再次登入，以繼續使用 iFolder 而不需要關閉瀏覽器。

可以藉由在右上方標題區域中按一下「登出」來登出，然後關閉瀏覽器。如果您未登出，則工作階段會保持開啟狀態直到其逾時，這可能會有安全性風險。

瀏覽 iFolder

iFolder 頁面會列出目前帳戶下可使用的所有 iFolder。可用的 iFolder 包括那些您擁有的以及與其他人共用的 iFolder。即使本機電腦上沒有設定 iFolder，您也可以透過 Web 存取來存取共用的 iFolder。「擁有者」欄識別擁有 iFolder 的人員名稱，可協助您找到所需的 iFolder。

按一下 iFolder 的名稱，您即可檢視其資料夾和檔案的清單、上載及下載檔案、建立資料夾，以及刪除資料夾和檔案。

管理 iFolder 中的資料夾和檔案

在 iFolder 中，您可以執行下列任務：

檢視資料夾和檔案清單
iFolder 頁面會按字母順序列出資料夾和檔案。捲動以尋找所需目錄或檔案。

瀏覽目錄
按一下資料夾的名稱，以開啟該資料夾並檢視其內容清單。若要返回至上一層目錄，請在工作列中按一下「向上一層」圖示。

開啟或儲存檔案
按一下您要開啟或儲存之檔案的名稱。

  • 按一下「開啟」以檢視檔案的內容。您的電腦上必須有相容的應用程式，才能開啟檔案。如果您打算使用、下載和修改檔案，然後將其上載。

  • 按一下「儲存」，將檔案儲存至本地硬碟或映射網路磁碟機。瀏覽至您要放置檔案的位置，然後按一下「儲存」。該檔案便會下載到您的電腦。

建立新資料夾
在「新資料夾」欄位中，指定新資料夾的名稱，然後按一下「建立」。資料夾名稱是區分大小寫的。當頁面重新整理時，資料夾會按字母順序顯示在清單中。

刪除資料夾或檔案
選取您要刪除之一或多個資料夾或檔案旁邊的核取方塊，然後按一下「刪除」。確認刪除。

上載檔案
在「上載檔案」欄位中，按一下「瀏覽」，尋找您要上載的本地資料夾，然後按一下「開啟」>「上載」。當頁面重新整理時，檔案會按字母順序顯示在清單中。
