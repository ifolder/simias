Login di accesso Web

È possibile eseguire il login di accesso Web a Novell iFolder® 3.x per accedere a cartelle e file incluse in qualsiasi cartella iFolder disponibile per il conto iFolder 3.x in uso. Sono incluse le cartelle iFolder di cui si è membri per le quali tuttavia non si è configurata una cartella corrispondente nel computer locale. Non è possibile accedere ai file inclusi nel conto Novell iFolder 2.1x tramite accesso Web. Il nome utente e la parola d'ordine vengono autenticati in base ai servizi di directory LDAP.

È inoltre possibile specificare le lingue supportate da utilizzare. Si tratta in genere della stessa lingua del server iFolder aziendale e del computer locale dell'utente. Verificare che il browser sia configurato per supportare la lingua desiderata.

  1. Digitare il nome utente e la parola d'ordine del conto iFolder con cui si desidera accedere.

    Per nomi utenti e parole d'ordine viene fatta distinzione tra maiuscole e minuscole.

  2. Impostare la lingua che si desidera utilizzare per l'accesso ai file nell'apposito elenco a discesa.

  3. Fare clic su Login.
