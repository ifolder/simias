登入 Web 存取

以您的 iFolder 3.x 帳戶登入 Novell iFolder® 3.x Web 存取，以取得對可用之任何 iFolder 中資料夾和檔案的存取權限。這包括您參與但尚未在本機電腦上設定的 iFolder(無法使用「Web 存取」以 Novell iFolder 2.1x 帳戶存取檔案)。您的使用者名稱和密碼會根據 LDAP 目錄服務進行驗證。

您還可以指定要使用的支援語言。通常，此語言便是 iFolder 企業伺服器和您本機電腦的「語言代碼」。請確定設定您瀏覽器的「語言」設定以支援所需語言。

  1. 指定要存取之 iFolder 帳戶的使用者名稱和密碼。

    使用者名稱和密碼區分大小寫。

  2. 使用「語言」下拉式清單以指定要用來存取檔案的語言。

  3. 按一下「登入」。
